# Функции 

![Иллюстрация к уроку](/lessons/functions.png)

Функции — переиспользуемые блоки кода.

## Обычная функция

```javascript
function greet(name) {
  return `Привет, ${name}!`;
}

greet("Яша"); // "Привет, Яша!"
```

## Стрелочная функция

```javascript
const add = (a, b) => a + b;

add(2, 3); // 5
```

## Когда что использовать

- **Обычные** — когда нужен `this` или `arguments`
- **Стрелочные** — в остальных случаях (короче и чище)
