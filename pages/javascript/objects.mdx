# JavaScript: Мозги. Урок 3: Объекты

Объекты - это фундаментальный кирпичик JavaScript. Они позволяют хранить и организовывать данные, представляя собой коллекции свойств, состоящих из ключей и значений.

## Что такое объект?

Представьте себе объект как ящик с подписанными отделениями. Каждое отделение (ключ) содержит какую-то информацию (значение). В JavaScript объект – это структура данных, которая содержит свойства. Свойство состоит из имени (ключа) и значения.

```javascript
// Пример объекта "человек"
let person = {
  name: "Иван",
  age: 30,
  city: "Москва"
};

console.log(person.name); // Выведет: Иван
console.log(person.age);  // Выведет: 30
console.log(person["city"]); // Выведет: Москва (альтернативный способ доступа)
```

В этом примере `person` – это объект. `name`, `age` и `city` – это ключи (или имена свойств). `"Иван"`, `30` и `"Москва"` – это значения, связанные с этими ключами.

## Создание объектов

Есть несколько способов создания объектов в JavaScript:

1. **Литерал объекта:** (как в примере выше)
   ```javascript
   let myObject = {}; // Создаем пустой объект
   ```

2. **Конструктор `Object`:**
   ```javascript
   let anotherObject = new Object();
   anotherObject.property1 = "Значение 1";
   ```

3. **Функции-конструкторы (классы):** (будет рассмотрено позже, но для понимания)
   ```javascript
   function Car(make, model, year) {
     this.make = make;
     this.model = model;
     this.year = year;
   }

   let myCar = new Car("Toyota", "Camry", 2020);
   console.log(myCar.make); // Выведет: Toyota
   ```

## Добавление, изменение и удаление свойств

Объекты динамичны, то есть мы можем добавлять, изменять и удалять свойства после создания объекта.

```javascript
let book = {
  title: "JavaScript для начинающих",
  author: "Неизвестный"
};

// Добавляем свойство
book.pages = 300;
console.log(book.pages); // Выведет: 300

// Изменяем свойство
book.author = "Джон Смит";
console.log(book.author); // Выведет: Джон Смит

// Удаляем свойство
delete book.pages;
console.log(book.pages); // Выведет: undefined
```

## Методы объекта

Значения свойств могут быть функциями. Такие функции называются *методами* объекта.

```javascript
let calculator = {
  add: function(a, b) {
    return a + b;
  },
  subtract: function(a, b) {
    return a - b;
  }
};

console.log(calculator.add(5, 3));    // Выведет: 8
console.log(calculator.subtract(10, 4)); // Выведет: 6
```

## Жизненный пример

Объекты используются повсеместно в JavaScript. Например, при работе с DOM, каждый элемент HTML страницы представлен объектом.

```html
<!DOCTYPE html>
<html>
<head>
  <title>Пример DOM</title>
</head>
<body>
  <p id="myParagraph">Это параграф.</p>

  <script>
    let paragraph = document.getElementById("myParagraph");
    console.log(paragraph.textContent); // Выведет: Это параграф.
    paragraph.style.color = "blue"; // Изменяем цвет текста параграфа
  </script>
</body>
</html>
```

В этом примере `paragraph` – это объект, представляющий HTML-элемент `<p>`.  Мы можем обращаться к его свойствам (например, `textContent` или `style`) и методам, чтобы взаимодействовать с элементом на странице.  Фреймворки, такие как React, Vue и Angular, активно используют объекты для представления компонентов и управления состоянием приложения. JSON (JavaScript Object Notation) – это текстовый формат, основанный на объектах JavaScript, который широко используется для обмена данными между сервером и клиентом.

## Ключевые моменты

*   Объекты – это коллекции свойств (ключ-значение).
*   Ключи могут быть строками или символами.
*   Значения могут быть любыми типами данных (включая другие объекты и функции).
*   Объекты динамичны, можно добавлять, изменять и удалять свойства.
*   Функции, являющиеся свойствами объекта, называются методами.
*   Объекты являются основой для работы с DOM и используются во фреймворках.
