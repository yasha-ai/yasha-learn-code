# JavaScript: Мозги. Урок: События

![Иллюстрация к уроку](/lessons/javascript-events.png)


В этом уроке мы погрузимся в мир событий JavaScript. События позволяют вашему коду реагировать на действия пользователя и изменения в браузере, делая ваши веб-страницы интерактивными.

## Что такое события?

События – это сигналы, которые браузер генерирует, когда что-то происходит. Это может быть щелчок мыши, нажатие клавиши, загрузка страницы или изменение размера окна.  JavaScript позволяет "слушать" эти сигналы и выполнять определенный код в ответ на них.

## Типы событий

Существует множество типов событий. Вот некоторые из наиболее распространенных:

*   `click`: Щелчок мыши.
*   `mouseover`: Наведение курсора мыши.
*   `mouseout`: Уход курсора мыши.
*   `keydown`: Нажатие клавиши.
*   `keyup`: Отпускание клавиши.
*   `submit`: Отправка формы.
*   `load`: Загрузка страницы или ресурса (изображения, скрипта).
*   `change`: Изменение значения элемента формы (например, выбор опции в `<select>`).

## Как обрабатывать события

Чтобы обработать событие, нужно:

1.  Выбрать элемент DOM, на котором нужно "слушать" событие.
2.  Использовать метод `addEventListener()` для добавления обработчика события.

```javascript
// Находим кнопку по ее ID
const button = document.getElementById('myButton');

// Добавляем обработчик события 'click'
button.addEventListener('click', function() {
  alert('Кнопка была нажата!');
});
```

В этом примере:

*   `document.getElementById('myButton')` выбирает элемент с `id="myButton"`.
*   `addEventListener('click', function() { ... })` добавляет обработчик события `click`.  Функция, переданная в `addEventListener`, будет выполнена, когда произойдет событие `click` на кнопке.

## Практические примеры

### Изменение текста при наведении мыши

```html
<p id="myParagraph">Наведи на меня мышкой!</p>
```

```javascript
const paragraph = document.getElementById('myParagraph');

paragraph.addEventListener('mouseover', function() {
  paragraph.textContent = 'Курсор здесь!';
});

paragraph.addEventListener('mouseout', function() {
  paragraph.textContent = 'Наведи на меня мышкой!';
});
```

### Обработка отправки формы

```html
<form id="myForm">
  <input type="text" name="username">
  <button type="submit">Отправить</button>
</form>
```

```javascript
const form = document.getElementById('myForm');

form.addEventListener('submit', function(event) {
  // Предотвращаем отправку формы по умолчанию (перезагрузку страницы)
  event.preventDefault();

  // Получаем значение поля ввода
  const username = form.username.value;

  alert('Отправлено имя пользователя: ' + username);
});
```

Обратите внимание на `event.preventDefault()`. Он предотвращает стандартное поведение браузера при отправке формы (перезагрузку страницы). Это часто необходимо, когда вы хотите обработать данные формы с помощью JavaScript.

## Жизненный пример

События – это основа интерактивных веб-приложений.

*   **React, Vue, Angular:** Эти фреймворки активно используют события для обновления пользовательского интерфейса. Например, когда пользователь нажимает кнопку, фреймворк может обновить состояние компонента и перерисовать часть страницы.
*   **Игры:** События (нажатия клавиш, щелчки мыши) используются для управления персонажем, взаимодействия с игровым миром и т.д.
*   **Анимации:** События (например, окончание загрузки страницы) могут запускать анимации.
*   **Валидация форм:** События (например, изменение значения поля ввода) могут запускать валидацию данных.

## Ключевые моменты

*   События позволяют JavaScript реагировать на действия пользователя и изменения в браузере.
*   `addEventListener()` используется для добавления обработчиков событий.
*   `event.preventDefault()` предотвращает стандартное поведение браузера.
*   События широко используются во всех современных веб-приложениях и фреймворках.
