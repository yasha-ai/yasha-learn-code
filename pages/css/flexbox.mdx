#  CSS: Flexbox - Гибкая разметка

Flexbox (Flexible Box Layout) – это мощный инструмент CSS, предназначенный для создания сложных и адаптивных макетов веб-страниц. Он позволяет легко управлять выравниванием, направлением и порядком элементов внутри контейнера.

##  Концепция Flexbox

Представьте, что у вас есть коробка (контейнер), в которой лежат элементы (элементы flex). Flexbox позволяет вам управлять тем, как эти элементы располагаются внутри коробки: в строку, в столбец, как они выравниваются по горизонтали и вертикали, и даже как они распределяют доступное пространство.

Основная идея Flexbox заключается в том, чтобы элементы могли *гибко* подстраиваться под размеры контейнера, даже если размеры контейнера меняются. Это особенно полезно для создания адаптивных макетов, которые хорошо выглядят на разных устройствах.

##  Основные свойства Flexbox

Чтобы использовать Flexbox, нужно сначала сделать контейнер flex-контейнером. Для этого используется свойство `display: flex;` или `display: inline-flex;`.

```css
.container {
  display: flex; /* Или display: inline-flex; */
}
```

После того как контейнер стал flex-контейнером, можно управлять его поведением и поведением его элементов с помощью различных свойств.

Основные свойства контейнера:

*   `flex-direction`: Определяет направление главной оси (строка или столбец). Возможные значения: `row` (по умолчанию), `column`, `row-reverse`, `column-reverse`.
*   `justify-content`: Определяет выравнивание элементов вдоль главной оси. Возможные значения: `flex-start`, `flex-end`, `center`, `space-between`, `space-around`, `space-evenly`.
*   `align-items`: Определяет выравнивание элементов вдоль поперечной оси. Возможные значения: `flex-start`, `flex-end`, `center`, `baseline`, `stretch` (по умолчанию).
*   `flex-wrap`: Определяет, будут ли элементы переноситься на новую строку, если не помещаются в контейнере. Возможные значения: `nowrap` (по умолчанию), `wrap`, `wrap-reverse`.
*   `align-content`: Определяет выравнивание строк в flex-контейнере, когда используется `flex-wrap: wrap`. Возможные значения: `flex-start`, `flex-end`, `center`, `space-between`, `space-around`, `stretch` (по умолчанию).

Основные свойства элементов:

*   `flex-grow`: Определяет, насколько элемент должен увеличиться относительно других элементов, чтобы занять доступное пространство.
*   `flex-shrink`: Определяет, насколько элемент может сжаться, если места недостаточно.
*   `flex-basis`: Определяет начальный размер элемента перед распределением свободного пространства.
*   `flex`: Сокращенная запись для `flex-grow`, `flex-shrink` и `flex-basis`.
*   `align-self`: Переопределяет `align-items` для конкретного элемента.

##  Практические примеры

**Пример 1: Горизонтальное меню**

```html
<div class="menu">
  <a href="#">Главная</a>
  <a href="#">О нас</a>
  <a href="#">Услуги</a>
  <a href="#">Контакты</a>
</div>
```

```css
.menu {
  display: flex;
  justify-content: space-around; /* Равномерно распределяет элементы по горизонтали */
  background-color: #f0f0f0;
  padding: 10px;
}

.menu a {
  text-decoration: none;
  color: #333;
}
```

**Пример 2: Вертикальное выравнивание элемента по центру**

```html
<div class="container">
  <div class="item">Центрированный элемент</div>
</div>
```

```css
.container {
  display: flex;
  justify-content: center; /* Горизонтальное выравнивание */
  align-items: center; /* Вертикальное выравнивание */
  height: 200px; /* Задаем высоту контейнера для наглядности */
  background-color: #eee;
}

.item {
  background-color: #ddd;
  padding: 20px;
}
```

##  Жизненный пример

Flexbox широко используется в современных веб-проектах. Например:

*   **Панели навигации:** Большинство современных сайтов используют Flexbox для создания гибких и адаптивных панелей навигации.
*   **Макеты страниц:** Flexbox позволяет создавать сложные макеты страниц с несколькими колонками, которые легко адаптируются к разным размерам экрана.
*   **UI-фреймворки:** Многие UI-фреймворки, такие как Bootstrap и Materialize, используют Flexbox для создания адаптивных компонентов.
*   **Мобильные приложения:** Flexbox идеально подходит для создания макетов мобильных приложений, которые должны хорошо выглядеть на разных устройствах.

##  Ключевые моменты

*   Flexbox – это мощный инструмент для создания гибких макетов.
*   `display: flex;` или `display: inline-flex;` делает контейнер flex-контейнером.
*   `flex-direction`, `justify-content`, `align-items` – основные свойства контейнера.
*   `flex-grow`, `flex-shrink`, `flex-basis` – основные свойства элементов.
*   Flexbox широко используется в современных веб-проектах.
