##  CSS Grid: Создаем мощные макеты

![Иллюстрация к уроку](/lessons/css-grid.png)


CSS Grid Layout – это мощный инструмент для создания сложных и адаптивных макетов веб-страниц. Он позволяет разделить страницу на строки и столбцы, что дает полный контроль над расположением элементов.

###  Концепция Grid

Grid позволяет определить структуру страницы, как сетку, в которой элементы располагаются в определенных ячейках. Представьте, что у вас есть таблица, но гораздо более гибкая и мощная. Основные свойства:

*   `grid-container`: Родительский элемент, который становится grid-контейнером.
*   `grid-item`: Дочерние элементы внутри grid-контейнера.
*   `grid-template-rows`: Определяет высоту строк.
*   `grid-template-columns`: Определяет ширину столбцов.
*   `grid-gap`: Определяет расстояние между ячейками.
*   `grid-row`: Определяет, в какой строке начинается и заканчивается элемент.
*   `grid-column`: Определяет, в каком столбце начинается и заканчивается элемент.

###  Практические примеры

**Пример 1: Простой макет 2x2**

```html
<div class="grid-container">
  <div class="item item1">Item 1</div>
  <div class="item item2">Item 2</div>
  <div class="item item3">Item 3</div>
  <div class="item item4">Item 4</div>
</div>
```

```css
.grid-container {
  display: grid; /* Делаем контейнер grid-контейнером */
  grid-template-columns: auto auto; /* Определяем 2 столбца с автоматической шириной */
  grid-template-rows: auto auto; /* Определяем 2 строки с автоматической высотой */
  grid-gap: 10px; /* Добавляем отступы между ячейками */
  background-color: #2196F3;
  padding: 10px;
}

.grid-container > div {
  background-color: rgba(255, 255, 255, 0.8);
  text-align: center;
  padding: 20px 0;
  font-size: 30px;
}
```

**Пример 2: Более сложный макет с использованием `grid-column` и `grid-row`**

```html
<div class="grid-container-2">
  <div class="item item1">Header</div>
  <div class="item item2">Menu</div>
  <div class="item item3">Main</div>
  <div class="item item4">Right</div>
  <div class="item item5">Footer</div>
</div>
```

```css
.grid-container-2 {
  display: grid;
  grid-template-areas:
    'header header header'
    'menu main right'
    'menu footer footer';
  grid-gap: 10px;
  background-color: #2196F3;
  padding: 10px;
}

.grid-container-2 > div {
  background-color: rgba(255, 255, 255, 0.8);
  text-align: center;
  padding: 20px 0;
  font-size: 30px;
}

.item1 {
  grid-area: header; /* Занимает область header */
}
.item2 {
  grid-area: menu; /* Занимает область menu */
}
.item3 {
  grid-area: main; /* Занимает область main */
}
.item4 {
  grid-area: right; /* Занимает область right */
}
.item5 {
  grid-area: footer; /* Занимает область footer */
}

```

###  Жизненный пример

Многие современные веб-сайты используют CSS Grid для создания гибких и адаптивных макетов. Например:

*   **Новостные сайты:** Часто используют Grid для отображения новостей в виде сетки с разными размерами ячеек.
*   **Интернет-магазины:** Grid помогает организовать товары в каталоге, создавая удобный и визуально привлекательный интерфейс.
*   **Фреймворки:** Многие CSS-фреймворки, такие как Bootstrap и Foundation, используют Grid Layout в своей основе для создания гибких систем сеток.
*   **Pinterest:** Использует Grid для отображения изображений в динамической сетке.

###  Ключевые моменты

*   `display: grid;` - включает Grid Layout для контейнера.
*   `grid-template-columns` и `grid-template-rows` определяют структуру сетки.
*   `grid-gap` задает отступы между ячейками.
*   `grid-column` и `grid-row` позволяют размещать элементы по строкам и столбцам.
*   `grid-template-areas` и `grid-area` позволяют задавать именованные области в сетке для более читаемого кода.
