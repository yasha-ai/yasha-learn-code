### –ß—Ç–æ —Ç–∞–∫–æ–µ CSS Grid?

CSS Grid Layout (–∏–ª–∏ –ø—Ä–æ—Å—Ç–æ Grid) ‚Äî —ç—Ç–æ –¥–≤—É–º–µ—Ä–Ω–∞—è —Å–∏—Å—Ç–µ–º–∞ –º–∞–∫–µ—Ç–æ–≤ CSS, –∫–æ—Ç–æ—Ä–∞—è –ø–æ–∑–≤–æ–ª—è–µ—Ç —Ä–∞–∑—Ä–∞–±–æ—Ç—á–∏–∫–∞–º —Å–æ–∑–¥–∞–≤–∞—Ç—å —Å–ª–æ–∂–Ω—ã–µ, –æ—Ç–∑—ã–≤—á–∏–≤—ã–µ –¥–∏–∑–∞–π–Ω—ã –≤–µ–±-—Å—Ç—Ä–∞–Ω–∏—Ü, –æ–ø—Ä–µ–¥–µ–ª—è—è —Å—Ç—Ä–æ–∫–∏ –∏ —Å—Ç–æ–ª–±—Ü—ã. –í –æ—Ç–ª–∏—á–∏–µ –æ—Ç Flexbox, –∫–æ—Ç–æ—Ä—ã–π –æ–ø—Ç–∏–º–∏–∑–∏—Ä–æ–≤–∞–Ω –¥–ª—è –æ–¥–Ω–æ–º–µ—Ä–Ω—ã—Ö –º–∞–∫–µ—Ç–æ–≤ (—Å—Ç—Ä–æ–∫–∞ –∏–ª–∏ —Å—Ç–æ–ª–±–µ—Ü), Grid –∏–¥–µ–∞–ª—å–Ω–æ –ø–æ–¥—Ö–æ–¥–∏—Ç –¥–ª—è —Å–æ–∑–¥–∞–Ω–∏—è –º–∞–∫–µ—Ç–æ–≤ –≤—Å–µ–≥–æ —Å–∞–π—Ç–∞ –∏–ª–∏ –æ—Ç–¥–µ–ª—å–Ω—ã—Ö –µ–≥–æ —Å–µ–∫—Ü–∏–π, —Ç—Ä–µ–±—É—é—â–∏—Ö –æ–¥–Ω–æ–≤—Ä–µ–º–µ–Ω–Ω–æ–≥–æ –∫–æ–Ω—Ç—Ä–æ–ª—è –Ω–∞–¥ —Å—Ç—Ä–æ–∫–∞–º–∏ –∏ —Å—Ç–æ–ª–±—Ü–∞–º–∏.

### –û—Å–Ω–æ–≤–Ω—ã–µ –∫–æ–Ω—Ü–µ–ø—Ü–∏–∏ Grid

*   **Grid Container (–ö–æ–Ω—Ç–µ–π–Ω–µ—Ä —Å–µ—Ç–∫–∏):** –≠–ª–µ–º–µ–Ω—Ç, –∫ –∫–æ—Ç–æ—Ä–æ–º—É –ø—Ä–∏–º–µ–Ω–µ–Ω–æ `display: grid` –∏–ª–∏ `display: inline-grid`. –û–Ω —è–≤–ª—è–µ—Ç—Å—è —Ä–æ–¥–∏—Ç–µ–ª–µ–º –¥–ª—è —ç–ª–µ–º–µ–Ω—Ç–æ–≤ —Å–µ—Ç–∫–∏.
*   **Grid Items (–≠–ª–µ–º–µ–Ω—Ç—ã —Å–µ—Ç–∫–∏):** –ü—Ä—è–º—ã–µ –ø–æ—Ç–æ–º–∫–∏ Grid Container.
*   **Grid Line (–õ–∏–Ω–∏—è —Å–µ—Ç–∫–∏):** –†–∞–∑–¥–µ–ª–∏—Ç–µ–ª—å–Ω–∞—è –ª–∏–Ω–∏—è –º–µ–∂–¥—É —Å—Ç–æ–ª–±—Ü–∞–º–∏ –∏–ª–∏ —Å—Ç—Ä–æ–∫–∞–º–∏.
*   **Grid Track (–î–æ—Ä–æ–∂–∫–∞ —Å–µ—Ç–∫–∏):** –ü—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–æ –º–µ–∂–¥—É –¥–≤—É–º—è –ª–∏–Ω–∏—è–º–∏ —Å–µ—Ç–∫–∏ (—Å—Ç–æ–ª–±–µ—Ü –∏–ª–∏ —Å—Ç—Ä–æ–∫–∞).
*   **Grid Cell (–Ø—á–µ–π–∫–∞ —Å–µ—Ç–∫–∏):** –ü—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–æ, –æ–±—Ä–∞–∑–æ–≤–∞–Ω–Ω–æ–µ –ø–µ—Ä–µ—Å–µ—á–µ–Ω–∏–µ–º –¥–æ—Ä–æ–∂–∫–∏ —Å—Ç–æ–ª–±—Ü–∞ –∏ –¥–æ—Ä–æ–∂–∫–∏ —Å—Ç—Ä–æ–∫–∏.
*   **Grid Area (–û–±–ª–∞—Å—Ç—å —Å–µ—Ç–∫–∏):** –ü—Ä—è–º–æ—É–≥–æ–ª—å–Ω–∞—è –æ–±–ª–∞—Å—Ç—å, –∑–∞–Ω–∏–º–∞—é—â–∞—è –Ω–µ—Å–∫–æ–ª—å–∫–æ —è—á–µ–µ–∫ —Å–µ—Ç–∫–∏.

### Grid Container: –û–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ —Å–µ—Ç–∫–∏

–î–ª—è –Ω–∞—á–∞–ª–∞ —Ä–∞–±–æ—Ç—ã —Å Grid –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –æ–ø—Ä–µ–¥–µ–ª–∏—Ç—å —ç–ª–µ–º–µ–Ω—Ç –∫–∞–∫ –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä —Å–µ—Ç–∫–∏:

```css
.grid-container {
  display: grid; /* –í–∫–ª—é—á–∞–µ—Ç —Ä–µ–∂–∏–º —Å–µ—Ç–∫–∏ */
}
```

**–û–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ —Å—Ç—Ä–æ–∫ –∏ —Å—Ç–æ–ª–±—Ü–æ–≤:**

*   `grid-template-columns`: –û–ø—Ä–µ–¥–µ–ª—è–µ—Ç –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –∏ —Ä–∞–∑–º–µ—Ä—ã —Å—Ç–æ–ª–±—Ü–æ–≤.
*   `grid-template-rows`: –û–ø—Ä–µ–¥–µ–ª—è–µ—Ç –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –∏ —Ä–∞–∑–º–µ—Ä—ã —Å—Ç—Ä–æ–∫.

–ï–¥–∏–Ω–∏—Ü—ã –∏–∑–º–µ—Ä–µ–Ω–∏—è: `px`, `%`, `em`, `rem`, `auto`, `minmax()`, `fr` (fraction ‚Äî –¥–æ–ª—è –æ—Å—Ç–∞–≤—à–µ–≥–æ—Å—è –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–∞), `repeat()`.

**–ü—Ä–∏–º–µ—Ä 1: –ü—Ä–æ—Å—Ç–∞—è 3x2 —Å–µ—Ç–∫–∞**

–°–æ–∑–¥–∞–¥–∏–º —Å–µ—Ç–∫—É –∏–∑ —Ç—Ä–µ—Ö —Å—Ç–æ–ª–±—Ü–æ–≤ —Ä–∞–≤–Ω–æ–π —à–∏—Ä–∏–Ω—ã –∏ –¥–≤—É—Ö —Å—Ç—Ä–æ–∫.

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr; /* –¢—Ä–∏ —Å—Ç–æ–ª–±—Ü–∞, –∫–∞–∂–¥—ã–π –∑–∞–Ω–∏–º–∞–µ—Ç 1 –¥–æ–ª—é */
  grid-template-rows: 100px auto;    /* –ü–µ—Ä–≤–∞—è —Å—Ç—Ä–æ–∫–∞ 100px, –≤—Ç–æ—Ä–∞—è –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏ */
  gap: 10px;                         /* –û—Ç—Å—Ç—É–ø—ã –º–µ–∂–¥—É —è—á–µ–π–∫–∞–º–∏ */
  background-color: #eee;
  padding: 10px;
}

.grid-item {
  background-color: #64b5f6;
  border: 1px solid #2196f3;
  padding: 15px;
  text-align: center;
}
```

### Grid Items: –†–∞–∑–º–µ—â–µ–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç–æ–≤

–≠–ª–µ–º–µ–Ω—Ç—ã —Å–µ—Ç–∫–∏ –º–æ–≥—É—Ç –±—ã—Ç—å —Ä–∞–∑–º–µ—â–µ–Ω—ã —è–≤–Ω–æ —Å –ø–æ–º–æ—â—å—é —Å–ª–µ–¥—É—é—â–∏—Ö —Å–≤–æ–π—Å—Ç–≤:

*   `grid-column-start`, `grid-column-end`
*   `grid-row-start`, `grid-row-end`
*   –°–æ–∫—Ä–∞—â–µ–Ω–Ω—ã–µ —Å–≤–æ–π—Å—Ç–≤–∞: `grid-column`, `grid-row` (–Ω–∞–ø—Ä–∏–º–µ—Ä, `1 / span 2` –∏–ª–∏ `1 / 3`)

**–ü—Ä–∏–º–µ—Ä 2: –≠–ª–µ–º–µ–Ω—Ç –∑–∞–Ω–∏–º–∞–µ—Ç –Ω–µ—Å–∫–æ–ª—å–∫–æ —è—á–µ–µ–∫**

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr); /* –¢—Ä–∏ —Ä–∞–≤–Ω—ã—Ö —Å—Ç–æ–ª–±—Ü–∞ */
  grid-template-rows: repeat(2, 100px);  /* –î–≤–µ —Å—Ç—Ä–æ–∫–∏ –ø–æ 100px */
  gap: 10px;
}

.item-1 {
  grid-column: 1 / 3; /* –ù–∞—á–∏–Ω–∞–µ—Ç—Å—è —Å –ª–∏–Ω–∏–∏ 1, –∑–∞–∫–∞–Ω—á–∏–≤–∞–µ—Ç—Å—è –Ω–∞ –ª–∏–Ω–∏–∏ 3 (–∑–∞–Ω–∏–º–∞–µ—Ç 1 –∏ 2 —Å—Ç–æ–ª–±–µ—Ü) */
  background-color: #ffcc80;
}

.item-2 {
  grid-row: 2 / 3;    /* –ó–∞–Ω–∏–º–∞–µ—Ç 2-—é —Å—Ç—Ä–æ–∫—É */
  background-color: #a5d6a7;
}
```

### –ò–º–µ–Ω–æ–≤–∞–Ω–∏–µ –ª–∏–Ω–∏–π –∏ –æ–±–ª–∞—Å—Ç–µ–π (Grid Areas)

`grid-template-areas` –ø–æ–∑–≤–æ–ª—è–µ—Ç —Å–µ–º–∞–Ω—Ç–∏—á–µ—Å–∫–∏ –Ω–∞–∑—ã–≤–∞—Ç—å –æ–±–ª–∞—Å—Ç–∏ –º–∞–∫–µ—Ç–∞, –¥–µ–ª–∞—è –∫–æ–¥ –±–æ–ª–µ–µ —á–∏—Ç–∞–µ–º—ã–º.

**–ü—Ä–∏–º–µ—Ä 3: –ú–∞–∫–µ—Ç —Å Header, Nav, Main, Aside, Footer**

```css

  display: grid;
  grid-template-columns: 1fr 3fr 1fr; /* –ö–æ–ª–æ–Ω–∫–∞ –¥–ª—è –Ω–∞–≤–∏–≥–∞—Ü–∏–∏, –æ—Å–Ω–æ–≤–Ω–∞—è, —Å–∞–π–¥–±–∞—Ä */
  grid-template-rows: auto 1fr auto;   /* Header, –∫–æ–Ω—Ç–µ–Ω—Ç, Footer */
  grid-template-areas:
    "header header header"
    "nav    main   aside"
    "footer footer footer";
  height: 300px;
  gap: 10px;
}

.header { grid-area: header; background-color: #ef9a9a; }
.nav    { grid-area: nav;    background-color: #a7d08c; }
.main   { grid-area: main;   background-color: #e57373; }
.aside  { grid-area: aside;  background-color: #64b5f6; }
.footer { grid-area: footer; background-color: #90a4ae; }
```

### –ê–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–æ–µ —Ä–∞–∑–º–µ—â–µ–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç–æ–≤

–ö–æ–≥–¥–∞ —ç–ª–µ–º–µ–Ω—Ç—ã —Å–µ—Ç–∫–∏ –Ω–µ –∏–º–µ—é—Ç —è–≤–Ω–æ –∑–∞–¥–∞–Ω–Ω—ã—Ö –ø–æ–∑–∏—Ü–∏–π, –æ–Ω–∏ —Ä–∞–∑–º–µ—â–∞—é—Ç—Å—è –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏.

*   `grid-auto-flow`: –û–ø—Ä–µ–¥–µ–ª—è–µ—Ç, –∫–∞–∫ –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏ —Ä–∞–∑–º–µ—â–∞—é—Ç—Å—è —ç–ª–µ–º–µ–Ω—Ç—ã ‚Äî –ø–æ —Å—Ç—Ä–æ–∫–∞–º (`row`, –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é) –∏–ª–∏ –ø–æ —Å—Ç–æ–ª–±—Ü–∞–º (`column`). `dense` –ø—ã—Ç–∞–µ—Ç—Å—è –∑–∞–ø–æ–ª–Ω–∏—Ç—å –ø—É—Å—Ç—ã–µ –º–µ—Å—Ç–∞.
*   `grid-auto-columns`, `grid-auto-rows`: –û–ø—Ä–µ–¥–µ–ª—è–µ—Ç —Ä–∞–∑–º–µ—Ä –¥–ª—è –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏ —Å–æ–∑–¥–∞–≤–∞–µ–º—ã—Ö (–Ω–µ—è–≤–Ω—ã—Ö) —Å—Ç–æ–ª–±—Ü–æ–≤/—Å—Ç—Ä–æ–∫.

**–ü—Ä–∏–º–µ—Ä 4: –ê–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–æ–µ —Ä–∞–∑–º–µ—â–µ–Ω–∏–µ —Å `grid-auto-flow: column`**

  grid-auto-flow: column; /* –≠–ª–µ–º–µ–Ω—Ç—ã –∑–∞–ø–æ–ª–Ω—è—é—Ç—Å—è –ø–æ —Å—Ç–æ–ª–±—Ü–∞–º */

  gap: 10px;
  /* –î–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω—ã–µ —Å—Ç–∏–ª–∏ –¥–ª—è grid-item, –∫–∞–∫ –≤ –ü—Ä–∏–º–µ—Ä–µ 1 */
}

### –í—ã—Ä–∞–≤–Ω–∏–≤–∞–Ω–∏–µ –≤ Grid

Grid –ø—Ä–µ–¥–ª–∞–≥–∞–µ—Ç –º–æ—â–Ω—ã–µ —Å—Ä–µ–¥—Å—Ç–≤–∞ –≤—ã—Ä–∞–≤–Ω–∏–≤–∞–Ω–∏—è, –∞–Ω–∞–ª–æ–≥–∏—á–Ω—ã–µ Flexbox:

*   **–î–ª—è —Å–æ–¥–µ—Ä–∂–∏–º–æ–≥–æ —è—á–µ–π–∫–∏ (Grid Items):**
    *   `justify-items`: –≤—ã—Ä–∞–≤–Ω–∏–≤–∞–µ—Ç —ç–ª–µ–º–µ–Ω—Ç—ã –≤–¥–æ–ª—å –æ—Å–∏ —Å—Ç–æ–ª–±—Ü–æ–≤ (–≥–æ—Ä–∏–∑–æ–Ω—Ç–∞–ª—å–Ω–æ).
    *   `align-items`: –≤—ã—Ä–∞–≤–Ω–∏–≤–∞–µ—Ç —ç–ª–µ–º–µ–Ω—Ç—ã –≤–¥–æ–ª—å –æ—Å–∏ —Å—Ç—Ä–æ–∫ (–≤–µ—Ä—Ç–∏–∫–∞–ª—å–Ω–æ).
    *   `place-items`: —Å–æ–∫—Ä–∞—â–µ–Ω–∏–µ –¥–ª—è `align-items` –∏ `justify-items`.
*   **–î–ª—è —Å–µ—Ç–∫–∏ –≤–Ω—É—Ç—Ä–∏ –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä–∞ (Grid Tracks):**
    *   `justify-content`: –≤—ã—Ä–∞–≤–Ω–∏–≤–∞–µ—Ç –≤—Å–µ –¥–æ—Ä–æ–∂–∫–∏ —Å—Ç–æ–ª–±—Ü–æ–≤ –≤–Ω—É—Ç—Ä–∏ –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä–∞ (–≥–æ—Ä–∏–∑–æ–Ω—Ç–∞–ª—å–Ω–æ).
    *   `align-content`: –≤—ã—Ä–∞–≤–Ω–∏–≤–∞–µ—Ç –≤—Å–µ –¥–æ—Ä–æ–∂–∫–∏ —Å—Ç—Ä–æ–∫ –≤–Ω—É—Ç—Ä–∏ –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä–∞ (–≤–µ—Ä—Ç–∏–∫–∞–ª—å–Ω–æ).
    *   `place-content`: —Å–æ–∫—Ä–∞—â–µ–Ω–∏–µ –¥–ª—è `align-content` –∏ `justify-content`.

**–ü—Ä–∏–º–µ—Ä 5: –í—ã—Ä–∞–≤–Ω–∏–≤–∞–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç–æ–≤**

### –¢–∏–ø–∏—á–Ω—ã–µ –æ—à–∏–±–∫–∏

1.  **–ó–∞–±—ã—Ç—å `display: grid`:** –°–∞–º–∞—è —á–∞—Å—Ç–∞—è –æ—à–∏–±–∫–∞, –±–µ–∑ —ç—Ç–æ–≥–æ —Å–≤–æ–π—Å—Ç–≤–∞ –æ—Å—Ç–∞–ª—å–Ω—ã–µ —Å–≤–æ–π—Å—Ç–≤–∞ Grid –Ω–µ —Ä–∞–±–æ—Ç–∞—é—Ç.
2.  **–ü—É—Ç–∞—Ç—å Grid –∏ Flexbox:** Grid ‚Äî –¥–ª—è –¥–≤—É–º–µ—Ä–Ω—ã—Ö –º–∞–∫–µ—Ç–æ–≤, Flexbox ‚Äî –¥–ª—è –æ–¥–Ω–æ–º–µ—Ä–Ω—ã—Ö. –û–Ω–∏ –æ—Ç–ª–∏—á–Ω–æ —Ä–∞–±–æ—Ç–∞—é—Ç –≤–º–µ—Å—Ç–µ (–Ω–∞–ø—Ä–∏–º–µ—Ä, Flexbox –≤–Ω—É—Ç—Ä–∏ Grid —è—á–µ–π–∫–∏).
