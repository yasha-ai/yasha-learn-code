## CSS: Фильтры Blur, Brightness, Contrast


![Иллюстрация к уроку](/lessons/css-filter-basics.png)
В этом уроке мы познакомимся с мощными CSS фильтрами, которые позволяют изменять внешний вид элементов, добавляя размытие, регулируя яркость и контрастность. Эти фильтры открывают широкие возможности для создания интересных визуальных эффектов.

### Что такое CSS фильтры?

CSS фильтры применяются к элементам для изменения их визуального представления. Они позволяют добавлять эффекты, которые раньше требовали использования графических редакторов. Свойства `filter` позволяют применять различные предопределенные фильтры, такие как размытие (`blur`), изменение яркости (`brightness`) и контрастности (`contrast`).

### Свойство `filter: blur()`

Фильтр `blur()` добавляет размытие к элементу. Значение, указанное в пикселях (px), определяет степень размытия. Чем больше значение, тем сильнее размытие.

```css
.blurred-image {
  filter: blur(5px); /* Размытие в 5 пикселей */
}
```

```html
<img src="image.jpg" class="blurred-image" alt="Размытое изображение">
```

В этом примере изображение с классом `blurred-image` будет размыто.

### Свойство `filter: brightness()`

Фильтр `brightness()` регулирует яркость элемента. Значение `1` означает отсутствие изменений (100% яркости). Значения меньше `1` делают элемент темнее, а значения больше `1` - светлее.

```css
.brighter-image {
  filter: brightness(1.5); /* Увеличение яркости в 1.5 раза */
}

.darker-image {
  filter: brightness(0.5); /* Уменьшение яркости в 2 раза */
}
```

```html
<img src="image.jpg" class="brighter-image" alt="Более яркое изображение">
<img src="image.jpg" class="darker-image" alt="Более темное изображение">
```

Здесь одно изображение станет ярче, а другое – темнее.

### Свойство `filter: contrast()`

Фильтр `contrast()` регулирует контрастность элемента. Аналогично `brightness()`, значение `1` означает отсутствие изменений. Значения меньше `1` уменьшают контрастность, а значения больше `1` - увеличивают.

```css
.high-contrast-image {
  filter: contrast(1.8); /* Увеличение контрастности в 1.8 раза */
}

.low-contrast-image {
  filter: contrast(0.3); /* Уменьшение контрастности в 3.3 раза */
}
```

```html
<img src="image.jpg" class="high-contrast-image" alt="Изображение с высокой контрастностью">
<img src="image.jpg" class="low-contrast-image" alt="Изображение с низкой контрастностью">
```

В результате получим изображения с разной контрастностью.

### Комбинирование фильтров

Фильтры можно комбинировать, чтобы создавать более сложные эффекты. Просто перечислите их через пробел.

```css
.combined-image {
  filter: blur(3px) brightness(0.8) contrast(1.2);
}
```

```html
<img src="image.jpg" class="combined-image" alt="Изображение с комбинированным фильтром">
```

Этот пример применит размытие, уменьшит яркость и увеличит контрастность к изображению.

### Жизненный пример

Фильтры часто используются для создания эффекта размытия фона за модальными окнами или навигационными панелями, чтобы привлечь внимание пользователя к основному контенту.  Также фильтры применяют для создания эффекта "hover" на изображениях, плавно меняя яркость или контрастность. Многие современные UI-библиотеки и фреймворки (например, Material UI, Bootstrap) используют фильтры для стилизации элементов.

Например, часто можно встретить размытие фона под полупрозрачным навигационным баром:

```css
.navbar {
  background-color: rgba(255, 255, 255, 0.8); /* Полупрозрачный белый */
  backdrop-filter: blur(10px); /* Размытие фона */
}
```

### Ключевые моменты

*   `filter` - мощное CSS свойство для изменения внешнего вида элементов.
*   `blur()` - добавляет размытие.
*   `brightness()` - регулирует яркость.
*   `contrast()` - регулирует контрастность.
*   Фильтры можно комбинировать для создания сложных эффектов.
*   `backdrop-filter` - фильтр для фона элемента.
