# CSS: Медиа-запросы (адаптивность)


![Иллюстрация к уроку](/lessons/css-media-queries.png)
Добро пожаловать на урок, посвященный медиа-запросам! С их помощью мы научимся создавать адаптивные веб-страницы, которые корректно отображаются на разных устройствах. Это важный навык для любого веб-разработчика.

## Что такое медиа-запросы?

Медиа-запросы позволяют применять разные CSS-стили в зависимости от характеристик устройства, на котором просматривается сайт.  Проще говоря, мы можем указать, что определенные стили должны применяться только при определенных условиях, например, когда ширина экрана меньше определенного значения.

## Синтаксис медиа-запроса

Основной синтаксис медиа-запроса выглядит так:

```css
@media (условие) {
  /* CSS-правила, применяемые при выполнении условия */
}
```

*   `@media` - ключевое слово, обозначающее начало медиа-запроса.
*   `(условие)` - условие, которое должно быть выполнено для применения стилей. Это может быть ширина экрана, ориентация устройства и многое другое.
*   `{ ... }` - блок CSS-правил, которые будут применены, если условие выполнено.

## Примеры условий

Наиболее часто используемые условия:

*   `max-width`: Максимальная ширина экрана.
*   `min-width`: Минимальная ширина экрана.
*   `orientation`: Ориентация устройства (landscape или portrait).

Пример:

```css
/* Стили для экранов шириной до 768px */
@media (max-width: 768px) {
  body {
    font-size: 14px;
  }

  .container {
    width: 100%;
  }
}

/* Стили для экранов шириной от 768px */
@media (min-width: 768px) {
  body {
    font-size: 16px;
  }

  .container {
    width: 70%;
  }
}
```

В этом примере, если ширина экрана меньше или равна 768px, размер шрифта на странице будет 14px, а контейнер займет всю ширину экрана. Если ширина экрана больше или равна 768px, размер шрифта будет 16px, а ширина контейнера будет 70%.

## Практический пример кода

Предположим, у нас есть простой HTML-код:

```html
<!DOCTYPE html>
<html>
<head>
  <title>Медиа-запросы</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <div class="container">
    <h1>Привет, мир!</h1>
    <p>Этот текст будет отображаться по-разному на разных устройствах.</p>
  </div>
</body>
</html>
```

А вот CSS-код (style.css) с использованием медиа-запросов:

```css
/* Общие стили */
body {
  font-family: sans-serif;
  margin: 0;
}

.container {
  max-width: 960px;
  margin: 0 auto;
  padding: 20px;
}

h1 {
  text-align: center;
}

/* Медиа-запрос для экранов шириной до 600px (мобильные телефоны) */
@media (max-width: 600px) {
  h1 {
    font-size: 24px; /* Уменьшаем размер заголовка */
  }

  .container {
    padding: 10px; /* Уменьшаем отступы */
  }
}

/* Медиа-запрос для экранов шириной от 601px до 960px (планшеты) */
@media (min-width: 601px) and (max-width: 960px) {
  h1 {
    font-size: 32px;
  }
}
```

Этот код меняет размер заголовка и отступы контейнера в зависимости от ширины экрана.  Обратите внимание на использование `and` для комбинирования условий.

## Жизненный пример

Медиа-запросы используются практически на всех современных веб-сайтах и в веб-приложениях. Например:

*   **Bootstrap и другие CSS-фреймворки:**  Они активно используют медиа-запросы для создания адаптивных сеток и компонентов.
*   **Социальные сети (Facebook, Instagram и т.д.):** Интерфейс адаптируется под разные размеры экранов телефонов и планшетов.
*   **Новостные сайты:**  Отображение контента меняется в зависимости от устройства, чтобы обеспечить удобное чтение.

## Ключевые моменты

*   Медиа-запросы позволяют создавать адаптивные веб-страницы.
*   Используйте `max-width`, `min-width` и другие условия для определения характеристик устройства.
*   Медиа-запросы широко используются в CSS-фреймворках и реальных веб-проектах.
*   Не бойтесь экспериментировать и тестировать ваши стили на разных устройствах.
