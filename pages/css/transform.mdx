## CSS: Transform - Трансформации элементов

CSS `transform` позволяет изменять положение, размер, поворот и наклон элементов без изменения их исходного положения в потоке документа. Это мощный инструмент для создания динамичных интерфейсов и анимаций.

### Основные функции transform

*   **translate()** - Перемещение элемента
*   **rotate()** - Поворот элемента
*   **scale()** - Изменение размера элемента
*   **skew()** - Наклон элемента
*   **matrix()** - Комбинированная трансформация
