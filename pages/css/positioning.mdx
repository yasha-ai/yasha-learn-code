#  CSS: Позиционирование 

![Иллюстрация к уроку](/lessons/css-positioning.png)


Позиционирование – это один из важнейших аспектов CSS, позволяющий контролировать расположение элементов на веб-странице. Правильное позиционирование делает интерфейс удобным и визуально привлекательным. В этом уроке мы разберем основные типы позиционирования и научимся их применять.

##  Типы позиционирования

В CSS существует пять основных типов позиционирования:

*   **static** (статическое): По умолчанию, все элементы имеют статическое позиционирование. Элементы отображаются в том порядке, в котором они идут в HTML-коде. Свойства `top`, `right`, `bottom`, и `left` не влияют на элементы со статическим позиционированием.

*   **relative** (относительное): Элемент позиционируется относительно своего нормального положения в потоке документа. Свойства `top`, `right`, `bottom`, и `left` используются для смещения элемента от его исходного положения.

*   **absolute** (абсолютное): Элемент позиционируется относительно ближайшего *позиционированного* предка (то есть предка, у которого свойство `position` установлено в `relative`, `absolute`, `fixed`, или `sticky`). Если позиционированный предок отсутствует, элемент позиционируется относительно `<html>`. Свойства `top`, `right`, `bottom`, и `left` определяют смещение от границ позиционированного предка.

*   **fixed** (фиксированное): Элемент позиционируется относительно окна браузера (viewport) и остается на месте при прокрутке страницы. Свойства `top`, `right`, `bottom`, и `left` определяют смещение от границ окна браузера.

*   **sticky** (липкое): Элемент ведет себя как `relative` до тех пор, пока он не достигнет определенной позиции прокрутки, после чего он становится `fixed`.  Свойства `top`, `right`, `bottom`, и `left` используются для определения этой позиции.

##  Примеры кода

```html
<!DOCTYPE html>
<html>
<head>
  <title>Позиционирование в CSS</title>
  <style>
    .container {
      position: relative; /* Создаем позиционированный контекст */
      width: 300px;
      height: 200px;
      border: 1px solid black;
    }

    .box {
      width: 50px;
      height: 50px;
      background-color: lightblue;
    }

    .relative {
      position: relative;
      left: 20px; /* Сдвигаем на 20px вправо от исходного положения */
      top: 10px;  /* Сдвигаем на 10px вниз от исходного положения */
    }

    .absolute {
      position: absolute;
      top: 0;     /* Прижимаем к верхнему краю контейнера */
      right: 0;   /* Прижимаем к правому краю контейнера */
      background-color: lightcoral;
    }

    .fixed {
      position: fixed;
      bottom: 0;   /* Прижимаем к нижнему краю окна браузера */
      left: 0;     /* Прижимаем к левому краю окна браузера */
      width: 100%;
      background-color: lightgreen;
      text-align: center;
    }

    .sticky {
      position: sticky;
      top: 0;       /* Фиксируется, когда верхний край элемента достигает верхнего края окна браузера */
      background-color: lightyellow;
      padding: 10px;
    }
  </style>
</head>
<body>

  <div class="container">
    <div class="box relative">Relative</div>
    <div class="box absolute">Absolute</div>
  </div>

  <div class="sticky">Sticky element</div>

  <p>Прокрутите страницу, чтобы увидеть эффект sticky.</p>
  <p style="height: 500px;">Прокрутка...</p>

  <div class="fixed">Fixed element</div>

</body>
</html>
```

##  Жизненный пример

*   **Absolute:** Часто используется для позиционирования модальных окон (pop-up) поверх основного контента.
*   **Fixed:**  Применяется для создания фиксированных навигационных панелей (header) или кнопок "наверх", которые остаются видимыми при прокрутке страницы.
*   **Relative:** Используется для создания позиционированного контекста для `absolute` позиционированных дочерних элементов. Это позволяет точно контролировать положение дочерних элементов внутри родительского.
*   **Sticky:** Используется для создания меню, которые "прилипают" к верху экрана при прокрутке. Многие современные сайты и фреймворки (например, Bootstrap) активно используют этот тип позиционирования для улучшения пользовательского опыта.

##  Ключевые моменты

*   `position: static` – поведение по умолчанию.
*   `position: relative` – смещение относительно исходного положения.
*   `position: absolute` – позиционирование относительно ближайшего позиционированного предка.
*   `position: fixed` – позиционирование относительно окна браузера.
*   `position: sticky` – гибридное поведение relative и fixed.
*   Важно понимать, что `absolute` позиционирование выводит элемент из потока документа.
*   Для корректной работы `absolute` часто требуется `relative` позиционирование родительского элемента.
*   `z-index` управляет порядком наложения элементов с позиционированием, отличным от `static`.
