# HTML: Data-атрибуты

Data-атрибуты — это мощный инструмент, позволяющий хранить дополнительную информацию прямо в HTML-тегах. Эту информацию удобно использовать в JavaScript для изменения поведения страницы.

### Что это такое?

Вы можете добавить любой свой атрибут к любому тегу, если он начинается с префикса `data-`. Это не нарушает правила HTML и позволяет "приклеить" данные к элементу.

Пример в коде:
```html
<div data-user-id="123" data-role="admin">Пользователь</div>
```

### Доступ через JavaScript

В JS доступ к этим данным осуществляется через свойство `dataset`. Названия преобразуются в camelCase (например, `data-user-id` станет `userId`).

<Playground
  template="static"
  files={{
    "index.html": `<!DOCTYPE html>
<html>
<body>
  <div id="user" data-user-id="777" data-user-name="Yasha">
    Информация о пользователе (скрыта в атрибутах)
  </div>
  <button onclick="showData()">Показать данные в консоли</button>

  <script>
    function showData() {
      const el = document.getElementById('user');
      const id = el.dataset.userId;
      const name = el.dataset.userName;
      alert('ID: ' + id + ', Имя: ' + name);
    }
  </script>
</body>
</html>`
  }}
/>

### Практический пример: Фильтрация

Data-атрибуты часто используют для фильтрации списков или хранения категорий.

<Playground
  template="static"
  files={{
    "index.html": `<!DOCTYPE html>
<html>
<body>
  <button onclick="filter('fruit')">Фрукты</button>
  <button onclick="filter('veg')">Овощи</button>
  <button onclick="filter('all')">Все</button>

  <ul id="list">
    <li data-type="fruit">Яблоко</li>
    <li data-type="veg">Морковь</li>
    <li data-type="fruit">Банан</li>
    <li data-type="veg">Огурец</li>
  </ul>

  <script>
    function filter(type) {
      const items = document.querySelectorAll('#list li');
      items.forEach(item => {
        if (type === 'all' || item.dataset.type === type) {
          item.style.display = 'block';
        } else {
          item.style.display = 'none';
        }
      });
    }
  </script>
</body>
</html>`
  }}
/>

### Ключевые моменты

* Data-атрибуты всегда начинаются с `data-`.
* В JavaScript они доступны через `element.dataset`.
* Это отличный способ передать настройки из HTML в ваш скрипт.
* Не используйте их для хранения чувствительной информации (паролей), так как любой пользователь может увидеть их в коде страницы.
