# HTML: Рисование на Canvas


![Иллюстрация к уроку](/lessons/html-canvas-drawing.png)
В этом уроке мы познакомимся с элементом `<canvas>`, который позволяет рисовать графику с помощью JavaScript. Canvas — это мощный инструмент для создания интерактивных визуализаций и игр.

### Что такое Canvas?

Canvas — это область на странице, которая служит холстом. Сам по себе тег `<canvas>` ничего не отображает. Вся работа по рисованию ложится на JavaScript. Представьте себе пустой лист бумаги, на котором вы можете рисовать программно.

### Пример 1: Рисование прямоугольника

Посмотрите, как создать холст и нарисовать на нем простую фигуру.

<Playground
  template="static"
  files={{
    "index.html": `<!DOCTYPE html>
<html>
<body>
  <canvas id="myCanvas" width="300" height="150" style="border:1px solid #ddd;"></canvas>

  <script>
    const canvas = document.getElementById("myCanvas");
    const ctx = canvas.getContext("2d");

    ctx.fillStyle = "#2ecc71"; // Устанавливаем цвет
    ctx.fillRect(50, 25, 200, 100); // Рисуем прямоугольник (x, y, ширина, высота)
  </script>
</body>
</html>`
  }}
/>

### Пример 2: Линии и круги

С помощью путей можно рисовать более сложные формы.

<Playground
  template="static"
  files={{
    "index.html": `<!DOCTYPE html>
<html>
<body>
  <canvas id="shapesCanvas" width="300" height="150" style="border:1px solid #ddd;"></canvas>

  <script>
    const ctx = document.getElementById("shapesCanvas").getContext("2d");

    // Рисуем линию
    ctx.beginPath();
    ctx.moveTo(20, 20);
    ctx.lineTo(100, 130);
    ctx.strokeStyle = "blue";
    ctx.stroke();

    // Рисуем круг
    ctx.beginPath();
    ctx.arc(200, 75, 40, 0, 2 * Math.PI);
    ctx.fillStyle = "red";
    ctx.fill();
  </script>
</body>
</html>`
  }}
/>

### Где это применяется?

* **Графики:** Библиотеки вроде Chart.js используют Canvas.
* **Игры:** Это основа для большинства 2D-игр в браузере.
* **Редакторы:** Онлайн-редакторы фото используют Canvas для обработки пикселей.

### Ключевые моменты

* `getContext("2d")` — обязательный шаг для начала работы.
* `fillRect()` рисует закрашенную фигуру.
* `stroke()` обводит путь линией.
* Координаты (0,0) — это левый верхний угол холста.
