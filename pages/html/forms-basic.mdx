##  HTML: Скелет - Формы (базовые)

![Иллюстрация к уроку](/lessons/html-forms-basic.png)


Формы - это ключевой элемент веб-страниц, позволяющий пользователям взаимодействовать с сайтом, отправлять данные и получать обратную связь. В этом уроке мы рассмотрим основы создания форм в HTML, изучим основные теги и атрибуты, необходимые для создания базовых форм.

### Что такое формы и зачем они нужны?

Формы позволяют собирать информацию от пользователей.  Представьте себе поле для ввода вашего имени, выбора пола или написания сообщения - все это элементы формы.  Данные, введенные пользователем, отправляются на сервер для дальнейшей обработки.

### Основные теги и атрибуты форм

Главный тег для создания формы - `<form>`.  Внутри этого тега размещаются элементы управления формы, такие как текстовые поля, кнопки, выпадающие списки и т.д.

```html
<form action="/submit" method="post">
  <!-- Элементы управления формы -->
</form>
```

*   **`action`**:  Указывает URL, на который будут отправлены данные формы.
*   **`method`**:  Определяет HTTP-метод отправки данных (обычно `get` или `post`). `post` используется для отправки больших объемов данных или конфиденциальной информации, а `get` – для небольших объемов и когда данные видны в URL.

Внутри `<form>` используются различные теги для создания полей ввода:

*   `<input>`:  Самый распространенный тег, используется для создания текстовых полей, паролей, чекбоксов, радиокнопок и кнопок.  Атрибут `type` определяет тип поля.
*   `<textarea>`:  Создает многострочное текстовое поле для ввода больших объемов текста.
*   `<select>`:  Создает выпадающий список.
*   `<button>`:  Создает кнопку.
*   `<label>`: Подписывает поля формы, улучшая доступность.

Пример текстового поля:

```html
<label for="name">Имя:</label><br>
<input type="text" id="name" name="name"><br>
```

*   **`type="text"`**: Определяет поле как текстовое для ввода имени.
*   **`id="name"`**: Уникальный идентификатор поля (используется для связывания с `<label>`).
*   **`name="name"`**: Имя поля, которое будет использоваться при отправке данных на сервер.
*   **`<label for="name">`**: Связывает подпись "Имя:" с полем ввода, при клике на подпись, фокус переходит на поле ввода.

Пример радиокнопок:

```html
<p>Выберите пол:</p>
<input type="radio" id="male" name="gender" value="male">
<label for="male">Мужской</label><br>
<input type="radio" id="female" name="gender" value="female">
<label for="female">Женский</label><br>
```

*   **`type="radio"`**: Определяет поле как радиокнопку.
*   **`name="gender"`**:  Все радиокнопки в группе должны иметь одинаковое имя, чтобы только одна могла быть выбрана.
*   **`value="male"`**:  Значение, которое будет отправлено на сервер, если эта радиокнопка выбрана.

Пример кнопки отправки формы:

```html
<button type="submit">Отправить</button>
```

###  Жизненный пример

Формы используются повсеместно. Например:

*   **Регистрация и авторизация:**  Любой сайт, требующий регистрации (социальные сети, интернет-магазины), использует формы для сбора информации о пользователе (имя, email, пароль).
*   **Поиск:**  Поле поиска на Google или любом другом сайте - это форма.
*   **Комментарии:**  Формы позволяют оставлять комментарии на блогах и форумах.
*   **Оформление заказа:**  Интернет-магазины используют формы для сбора адреса доставки, способа оплаты и другой информации, необходимой для оформления заказа.
*   **Фреймворки:** Практически любой современный веб-фреймворк (React, Angular, Vue.js) предоставляет удобные инструменты для работы с формами, упрощая валидацию данных и отправку на сервер.

###  Ключевые моменты

*   `<form>` - основной тег для создания формы.
*   `action` и `method` атрибуты определяют, куда и как отправляются данные формы.
*   `<input>` - универсальный тег для создания различных типов полей ввода.
*   `type` атрибут определяет тип поля ввода (`text`, `password`, `radio`, `checkbox`, `submit` и т.д.).
*   `name` атрибут важен для идентификации поля при обработке данных на сервере.
*   `<label>` улучшает доступность и связывает текст с полем ввода.
