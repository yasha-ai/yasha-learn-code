# HTML: Формы (базовые)

![Иллюстрация к уроку](/lessons/html-forms-basic.png)

Формы - это ключевой элемент веб-страниц, позволяющий пользователям взаимодействовать с сайтом.



## Интерактивный пример

<Playground
  template="vanilla"
  files={{
    "/index.html": `
<!DOCTYPE html>
<html>
<head>
  <title>HTML Forms</title>
  <style>
    body { background-color: #282c34; color: white; font-family: sans-serif; }
    form { margin: 20px; padding: 20px; border: 1px solid white; width: 300px; }
    input[type="text"], input[type="email"], button { margin-top: 10px; padding: 5px; width: 100%; box-sizing: border-box; }
  </style>
</head>
<body>

  <form id="myForm">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name"><br><br>

    <label for="email">Email:</label>
    <input type="email" id="email" name="email"><br><br>

    <button type="submit">Submit</button>
  </form>

  <div id="output"></div>

  <script>
    const form = document.getElementById('myForm');
    form.addEventListener('submit', function(event) {
      event.preventDefault();
      const name = document.getElementById('name').value;
      const email = document.getElementById('email').value;
      document.getElementById('output').textContent = \`Name: \${name}, Email: \${email}\`;
    });
  <\/script>

</body>
</html>
`
  }}
/>
