# HTML: Атрибут contenteditable

`contenteditable` — это глобальный HTML-атрибут, который позволяет пользователю редактировать содержимое элемента прямо в браузере. Это простой способ добавить интерактивности без использования сложных форм.

### Как это работает?

Атрибут `contenteditable` может принимать три значения:
* `true`: Элемент можно редактировать.
* `false`: Редактирование запрещено.
* `inherit`: Наследует поведение от родителя.

### Где это применяется?

* **WYSIWYG-редакторы:** Основа для визуальных редакторов текста (как в WordPress или Notion).
* **Быстрое редактирование:** Позволяет менять данные "на лету" в админ-панелях.
* **Заметки:** Простые приложения для записей прямо в браузере.

### Ключевые моменты

* `contenteditable` делает обычные теги (`p`, `div`, `h1`) похожими на поля ввода.
* Чтобы сохранить изменения, понадобится JavaScript, который считает `innerHTML` или `innerText` элемента.
* Для обычных форм лучше использовать `<input>` или `<textarea>`, а `contenteditable` — для богатого текстового контента.
