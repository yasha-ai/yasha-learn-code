# HTML: ARIA States and Properties

ARIA (Accessible Rich Internet Applications) - это набор атрибутов, которые улучшают доступность веб-контента. ARIA states (состояния) и properties (свойства) описывают текущее состояние и характеристики элементов для вспомогательных технологий.

### Что такое ARIA States и Properties?

* **ARIA states (состояния)** описывают *динамические* характеристики, которые меняются при взаимодействии (например, `aria-selected`, `aria-expanded`).
* **ARIA properties (свойства)** описывают *статические* характеристики, которые обычно не меняются (например, `aria-label`, `aria-describedby`).

### Практические примеры

**Пример 1: Кнопка-переключатель**
С помощью `aria-expanded` мы сообщаем скринридеру, развернут ли связанный блок.

<Playground
  template="static"
  files={{
    "index.html": `<!DOCTYPE html>
<html>
<body>
  <button id="toggleBtn" aria-expanded="false" aria-controls="contentBlock">
    Показать/Скрыть контент
  </button>
  
  <div id="contentBlock" style="display: none; margin-top: 10px; padding: 10px; border: 1px solid #ccc;">
    Этот контент теперь виден!
  </div>

  <script>
    const btn = document.getElementById('toggleBtn');
    const content = document.getElementById('contentBlock');

    btn.addEventListener('click', () => {
      const isExpanded = btn.getAttribute('aria-expanded') === 'true';
      btn.setAttribute('aria-expanded', !isExpanded);
      content.style.display = isExpanded ? 'none' : 'block';
    });
  </script>
</body>
</html>`
  }}
/>

**Пример 2: Вкладки (Tabs)**
Атрибут `aria-selected` указывает на активную вкладку.

<Playground
  template="static"
  files={{
    "index.html": `<div role="tablist" aria-label="Пример вкладок">
  <button role="tab" aria-selected="true" id="tab-1" style="background: #eee;">Вкладка 1</button>
  <button role="tab" aria-selected="false" id="tab-2">Вкладка 2</button>

  <div role="tabpanel" aria-labelledby="tab-1" style="padding: 10px; border: 1px solid #eee;">

/>

### Ключевые моменты

* ARIA states описывают динамические изменения (открыто/закрыто, выбрано/не выбрано).
* ARIA properties описывают статические связи и метки.
* Всегда обновляйте ARIA-состояния через JavaScript при изменении интерфейса.
* Не заменяйте нативный HTML там, где он справляется сам.
