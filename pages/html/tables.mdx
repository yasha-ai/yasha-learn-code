#  HTML: Таблицы - Скелет данных

Таблицы в HTML - это способ организации данных в структурированном виде, как в электронной таблице. Они используются для представления информации в строках и столбцах, делая её более читаемой и понятной.

##  Основы таблиц

Для создания таблицы в HTML используются следующие теги:

*   `<table>`: Контейнер для всей таблицы.
*   `<tr>`: Определяет строку таблицы (table row).
*   `<th>`: Определяет заголовочную ячейку (table header). Обычно используется для заголовков столбцов.
*   `<td>`: Определяет ячейку данных (table data).

Простейший пример таблицы:

```html
<table>
  <tr>
    <th>Имя</th>
    <th>Возраст</th>
  </tr>
  <tr>
    <td>Иван</td>
    <td>30</td>
  </tr>
  <tr>
    <td>Мария</td>
    <td>25</td>
  </tr>
</table>
```

В этом примере создается таблица с двумя столбцами: "Имя" и "Возраст".  Первая строка содержит заголовки, а последующие строки содержат данные.

##  Атрибуты таблиц

Таблицы также могут иметь атрибуты для управления внешним видом и поведением. Некоторые из наиболее часто используемых атрибутов:

*   `border`: Определяет толщину границы таблицы. (Устаревший, лучше использовать CSS)
*   `colspan`: Указывает, сколько столбцов должна занимать ячейка.
*   `rowspan`: Указывает, сколько строк должна занимать ячейка.

Пример использования `colspan`:

```html
<table>
  <tr>
    <th colspan="2">Информация о сотрудниках</th>
  </tr>
  <tr>
    <td>Имя</td>
    <td>Возраст</td>
  </tr>
  <tr>
    <td>Иван</td>
    <td>30</td>
  </tr>
</table>
```

В этом примере заголовочная ячейка "Информация о сотрудниках" занимает два столбца.

Пример использования `rowspan`:

```html
<table>
  <tr>
    <th>Имя</th>
    <td>Иван</td>
  </tr>
  <tr>
    <th rowspan="2">Контактная информация</th>
    <td>Email: ivan@example.com</td>
  </tr>
  <tr>
    <td>Телефон: 123-456-7890</td>
  </tr>
</table>
```

В этом примере заголовочная ячейка "Контактная информация" занимает две строки.

##  Жизненный пример

Таблицы широко используются для отображения данных в различных веб-приложениях. Например:

*   **Интернет-магазины:** Отображение списка товаров с ценами, характеристиками и изображениями.
*   **Административные панели:** Отображение списка пользователей, заказов, статей и других данных.
*   **Финансовые сайты:** Отображение котировок акций, курсов валют и других финансовых данных.
*   **Календари:** Отображение событий по дням.

Современные веб-разработчики часто используют CSS для стилизации таблиц, делая их более привлекательными и удобными для чтения.  Фреймворки, такие как Bootstrap и Materialize, предоставляют готовые классы для стилизации таблиц, упрощая процесс разработки.  Однако, семантически корректное использование тегов `<table>`, `<tr>`, `<th>` и `<td>`  остается важным для доступности и SEO.

##  Ключевые моменты

*   `<table>`, `<tr>`, `<th>`, `<td>` - основные теги для создания таблиц.
*   `colspan` и `rowspan` позволяют объединять ячейки.
*   CSS используется для стилизации таблиц.
*   Семантически корректные таблицы важны для доступности и SEO.
*   Таблицы подходят для представления структурированных данных.
