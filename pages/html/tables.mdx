# HTML: Таблицы

![Иллюстрация к уроку](/lessons/html-tables.png)

Таблицы в HTML - это способ организации данных в структурированном виде. Они используются для представления информации в строках и столбцах, делая её более читаемой.

## Основы таблиц

Для создания таблицы в HTML используются следующие теги:

* `<table>`: Контейнер для всей таблицы.
* `<tr>`: Строка таблицы (table row).
* `<th>`: Заголовочная ячейка (table header).
* `<td>`: Ячейка данных (table data).

Простейший пример таблицы:

<Playground
  template="static"
  files={{
    "index.html": `<table border="1">
  <tr>
    <th>Имя</th>
    <th>Возраст</th>
  </tr>
  <tr>
    <td>Иван</td>
    <td>30</td>
  </tr>
  <tr>
    <td>Мария</td>
    <td>25</td>
  </tr>
</table>`
  }}
/>

## Объединение ячеек

Вы можете объединять ячейки с помощью атрибутов:

* `colspan`: Объединение столбцов.
* `rowspan`: Объединение строк.

Пример использования `colspan`:

<Playground
  template="static"
  files={{
    "index.html": `<table border="1">
  <tr>
    <th colspan="2">Сотрудники</th>
  </tr>
  <tr>
    <td>Имя</td>
    <td>Возраст</td>
  </tr>
  <tr>
    <td>Иван</td>
    <td>30</td>
  </tr>
</table>`
  }}
/>

Пример использования `rowspan`:

<Playground
  template="static"
  files={{
    "index.html": `<table border="1">
  <tr>
    <th>Имя</th>
    <td>Иван</td>
  </tr>
  <tr>
    <th rowspan="2">Контакты</th>
    <td>Email: ivan@example.com</td>
  </tr>
  <tr>
    <td>Телефон: 123-456</td>
  </tr>
</table>`
  }}
/>

## Жизненный пример

Таблицы используются для отображения:
* Списков товаров и цен.
* Списков пользователей в админ-панелях.
* Расписаний и календарей.
* Финансовых данных.

## Ключевые моменты

* `<table>`, `<tr>`, `<th>`, `<td>` - основные теги.
* `colspan` и `rowspan` позволяют объединять ячейки.
* Современные таблицы стилизуются через CSS.
* Используйте таблицы только для табличных данных, а не для верстки макета страницы.
