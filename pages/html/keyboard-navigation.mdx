# HTML: Навигация с клавиатуры

Привет! В этом уроке мы рассмотрим, как сделать ваш веб-сайт доступным для навигации с помощью клавиатуры. Это критически важно для пользователей, которые не используют мышь.

### Что такое Keyboard Navigation?

Keyboard Navigation — это возможность перемещаться и взаимодействовать с элементами страницы, используя только клавиатуру (клавиши `Tab`, `Shift+Tab`, `Enter`, `Space` и стрелки).

### Как это работает в HTML?

По умолчанию ссылки (`<a>`), кнопки (`<button>`) и поля форм (`<input>`) уже поддерживают навигацию через `Tab`.

<Playground
  template="static"
  files={{
    "index.html": `<div>
  <a href="#">Первая ссылка</a><br>
  <button onclick="alert('Нажато!')">Кнопка</button><br>
  <a href="#">Вторая ссылка</a>
  <p>Попробуйте нажимать Tab, чтобы перемещаться между элементами выше.</p>
</div>`
  }}
/>

### Атрибут tabindex

Атрибут `tabindex` позволяет включать в навигацию элементы, которые обычно в неё не попадают.

* `tabindex="0"`: Элемент попадает в очередь табуляции в обычном порядке.
* `tabindex="-1"`: Элемент нельзя выбрать через Tab, но можно сфокусировать через JavaScript.

<Playground
  template="static"
  files={{
    "index.html": `<div tabindex="0" style="padding: 10px; border: 1px solid blue;">
  Этот блок div теперь можно выбрать клавишей Tab!
</div>
<p>Обычно div не фокутируется, но tabindex="0" это исправляет.</p>`
  }}
/>

### Практический пример

<Playground
  template="static"
  files={{
    "index.html": `<!DOCTYPE html>
<html>
<body>
  <h3>Список с фокусом:</h3>
  <ul>
    <li tabindex="0">Пункт 1</li>
    <li tabindex="0">Пункт 2</li>
    <li tabindex="0">Пункт 3</li>
  </ul>

  <button onclick="alert('Кнопка нажата!')">Нажми меня</button>
</body>
</html>`
  }}
/>

### Ключевые моменты

* Навигация с клавиатуры — основа доступности.
* Нативные элементы (`button`, `a`) уже поддерживают `Tab`.
* Используйте `tabindex="0"` для кастомных интерактивных элементов.
* Избегайте `tabindex` со значениями больше 0 (это ломает привычный порядок).
* Всегда визуально выделяйте элемент в фокусе (через CSS `:focus`).
