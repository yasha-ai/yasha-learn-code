# HTML: Загрузка файлов

В этом уроке мы рассмотрим, как добавить на страницу возможность загрузки файлов. Это позволяет пользователям отправлять изображения, документы и другие данные на ваш сервер.

### Как это работает?

Для выбора файлов используется знакомый нам тег `<input>`, но с типом `file`. Чтобы файлы действительно отправились на сервер, форма должна иметь специальный атрибут `enctype`.

### Базовый пример

Посмотрите, как выглядит стандартное поле выбора файла.

<Playground
  template="static"
  files={{
    "index.html": `<!DOCTYPE html>
<html>
<body>
  <form action="/upload" method="post" enctype="multipart/form-data">
    <label for="avatar">Выберите фото профиля:</label><br><br>
    
    <input type="file" id="avatar" name="avatar" accept="image/*">
    
    <br><br>
    <button type="submit">Загрузить на сервер</button>
  </form>
</body>
</html>`
  }}
/>

### Важные атрибуты

* **`type="file"`**: Превращает обычное поле ввода в кнопку выбора файла.
* **`enctype="multipart/form-data"`**: **Обязательный** атрибут для тега `<form>`. Без него браузер отправит только имя файла, но не его содержимое.
* **`accept`**: Позволяет ограничить типы файлов. Например, `image/*` разрешит только картинки, а `.pdf` — только PDF-документы.
* **`multiple`**: Позволяет пользователю выбрать сразу несколько файлов (удерживая Ctrl или Shift).

<Playground
  template="static"
  files={{
    "index.html": `<label for="docs">Загрузить несколько документов:</label><br>
<input type="file" id="docs" name="docs" multiple accept=".pdf,.doc,.docx">`
  }}
/>

### Где это применяется?

* **Социальные сети:** Загрузка аватарок и постов.
* **Почта:** Прикрепление файлов к письму.
* **Облачные диски:** Загрузка документов для хранения.

### Ключевые моменты

* Сама загрузка (сохранение на диске) происходит на сервере (на Node.js, Python или PHP). HTML только предоставляет интерфейс.
* Не забывайте про `label` для доступности.
* Атрибут `name` критически важен — по нему сервер "узнает" присланный файл.
