# HTML: Изображения и Медиа

![Иллюстрация к уроку](/lessons/images-and-media.png)

В этом уроке мы научимся добавлять изображения и медиафайлы на наши веб-страницы. Изображения делают сайты более привлекательными, а медиафайлы позволяют доносить информацию в различных форматах.

### Изображения: тег `<img>`

Для вставки изображений в HTML используется тег `<img>`. Это одиночный тег (не имеет закрывающего тега) и требует обязательный атрибут `src`, указывающий путь к файлу изображения.

<Playground
  template="static"
  files={{
    "index.html": `<img 
  src="https://images.unsplash.com/photo-1517336714731-489689fd1ca8?w=400" 
  alt="Ноутбук на столе" 
  width="400"
>`
  }}
/>

* `src`: Атрибут, указывающий адрес изображения.
* `alt`: Альтернативный текст. Он отображается, если изображение не загрузилось, и используется программами чтения с экрана.
* `width` и `height`: Ширина и высота изображения.

### Аудио: тег `<audio>`

Тег `<audio>` позволяет встраивать аудиофайлы. Атрибут `controls` добавляет кнопки воспроизведения.

<Playground
  template="static"
  files={{
    "index.html": `<audio controls>
  <source src="https://www.soundhelix.com/examples/mp3/SoundHelix-Song-1.mp3" type="audio/mpeg">
  Ваш браузер не поддерживает аудио.
</audio>`
  }}
/>

### Видео: тег `<video>`

Тег `<video>` предназначен для встраивания видеофайлов.

<Playground
  template="static"
  files={{
    "index.html": `<video width="100%" controls poster="https://images.unsplash.com/photo-1485846234645-a62644f84728?w=800">
  <source src="https://www.w3schools.com/html/mov_bbb.mp4" type="video/mp4">
  Ваш браузер не поддерживает видео.
</video>`
  }}
/>

### Жизненный пример

* **Интернет-магазины:** Используют изображения товаров.
* **Новостные сайты:** Фотографии и видео для иллюстрации новостей.
* **Онлайн-курсы:** Видеоуроки для обучения.

### Ключевые моменты

* Используйте тег `<img>` для вставки изображений. Обязательно указывайте `alt`.
* Используйте тег `<audio>` для вставки аудио.
* Используйте тег `<video>` для вставки видео.
* Предоставляйте несколько форматов для лучшей совместимости.
* Оптимизируйте размер файлов для быстрой загрузки.
