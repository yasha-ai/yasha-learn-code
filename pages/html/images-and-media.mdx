##  HTML: Изображения и Медиа 

![Иллюстрация к уроку](/lessons/images-and-media.png)

В этом уроке мы научимся добавлять изображения и медиафайлы на наши веб-страницы. Изображения делают сайты более привлекательными, а медиафайлы позволяют доносить информацию в различных форматах.

###  Изображения: тег `<img>`

Для вставки изображений в HTML используется тег `<img>`. Это одиночный тег (не имеет закрывающего тега) и требует обязательный атрибут `src`, указывающий путь к файлу изображения.  Также рекомендуется использовать атрибут `alt` для описания изображения, что важно для доступности и SEO.

```html
<img src="images/my_image.jpg" alt="Фотография красивого пейзажа">
```

*   `src`: Атрибут, указывающий URL-адрес (путь) к изображению. Может быть как относительным (как в примере выше, `images/my_image.jpg`), так и абсолютным (например, `https://example.com/images/my_image.jpg`).
*   `alt`: Атрибут, предоставляющий альтернативный текст для изображения.  Он отображается, если изображение не может быть загружено или если пользователь использует программу чтения с экрана. Важно писать информативные описания.
*   `width` и `height`: Атрибуты, определяющие ширину и высоту изображения в пикселях. Лучше задавать их через CSS, но можно и в HTML.

```html
<img src="images/logo.png" alt="Логотип компании" width="200" height="100">
```

###  Аудио: тег `<audio>`

Тег `<audio>` позволяет встраивать аудиофайлы на веб-страницу. Он поддерживает различные атрибуты, такие как `src`, `controls`, `autoplay`, `loop` и `muted`.  Для лучшей совместимости рекомендуется предоставлять несколько источников в разных форматах (например, MP3 и OGG).

```html
<audio controls>
  <source src="audio/my_song.mp3" type="audio/mpeg">
  <source src="audio/my_song.ogg" type="audio/ogg">
  Ваш браузер не поддерживает аудио.
</audio>
```

*   `controls`: Отображает элементы управления аудио (кнопки воспроизведения, паузы, громкости и т.д.).
*   `autoplay`: Автоматически начинает воспроизведение аудио при загрузке страницы (не рекомендуется, так как может раздражать пользователей).
*   `loop`: Зацикливает воспроизведение аудио.
*   `muted`: Отключает звук по умолчанию.

###  Видео: тег `<video>`

Тег `<video>` аналогичен тегу `<audio>`, но предназначен для встраивания видеофайлов. Он также поддерживает атрибуты `src`, `controls`, `autoplay`, `loop` и `muted`.  Как и с аудио, рекомендуется предоставлять несколько источников в разных форматах для лучшей совместимости.

```html
<video width="640" height="360" controls>
  <source src="videos/my_video.mp4" type="video/mp4">
  <source src="videos/my_video.webm" type="video/webm">
  Ваш браузер не поддерживает видео.
</video>
```

*   `width` и `height`: Атрибуты, определяющие ширину и высоту видеоплеера.
*   `poster`: Атрибут, указывающий URL-адрес изображения, которое будет отображаться до начала воспроизведения видео.

###  Жизненный пример

*   **E-commerce сайты:** Используют изображения товаров для визуального представления.
*   **Новостные порталы:** Вставляют фотографии и видео для иллюстрации статей.
*   **Онлайн-курсы:** Используют видеоуроки для обучения.
*   **Социальные сети:** Позволяют пользователям загружать и делиться изображениями и видео.
*   **Фреймворки (React, Angular, Vue):**  Используют теги `<img>`, `<audio>`, `<video>` для отображения медиа контента, часто динамически подгружая `src` атрибуты из данных.

###  Ключевые моменты

*   Используйте тег `<img>` для вставки изображений. Обязательно указывайте атрибут `src` и `alt`.
*   Используйте тег `<audio>` для вставки аудиофайлов.
*   Используйте тег `<video>` для вставки видеофайлов.
*   Предоставляйте несколько источников медиафайлов в разных форматах для лучшей совместимости.
*   Не забывайте об атрибуте `alt` для изображений – это важно для доступности и SEO.
*   Оптимизируйте изображения и видео для уменьшения размера файлов и ускорения загрузки страницы.
