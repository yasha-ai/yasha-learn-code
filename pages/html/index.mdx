## Что освоишь?

После прохождения курса ты сможешь:
- Создавать семантичную и доступную разметку
- Работать с формами, медиа и интерактивными элементами
- Рисовать и анимировать с Canvas/SVG
- Создавать Web Components
- Оптимизировать SEO и доступность
- Использовать современные HTML5 API

4. **Применяй** — используй в своих проектах
