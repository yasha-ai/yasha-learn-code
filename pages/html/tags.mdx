# Теги — кирпичики HTML 

![Иллюстрация к уроку](/lessons/tags.png)

Каждый тег имеет своё назначение.

## Основные теги

| Тег | Назначение |
|-----|------------|
| `<h1>`-`<h6>` | Заголовки |
| `<p>` | Параграф |
| `<a>` | Ссылка |
| `<img>` | Изображение |
| `<div>` | Контейнер |
| `<span>` | Инлайн-контейнер |

## Пример

```html
<h1>Заголовок</h1>
<p>Это параграф с <a href="https://google.com">ссылкой</a>.</p>
<img src="cat.jpg" alt="Котик">
```
