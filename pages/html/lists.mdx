# HTML: Списки

![Иллюстрация к уроку](/lessons/lists.png)

Списки - это важный элемент HTML, позволяющий структурировать информацию на веб-странице.



## Интерактивный пример

<Playground
  template="vanilla"
  files={{
    "/index.html": `
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>HTML Lists</title>
    <style>
        body {
            background-color: #282c34;
            color: white;
            font-family: sans-serif;
            padding: 20px;
        }
        ul, ol {
            margin-bottom: 20px;
        }
    </style>
</head>
<body>
    <h2>Список покупок:</h2>
    <ul id="shopping-list">
        <li>Молоко</li>
        <li>Хлеб</li>
        <li>Яйца</li>
    </ul>

    <h2>Порядок действий:</h2>
    <ol>
        <li>Открыть сайт</li>
        <li>Зарегистрироваться</li>
        <li>Войти в аккаунт</li>
    </ol>

    <button onclick="addItem()">Добавить элемент в список покупок</button>

    <script>
        function addItem() {
            const list = document.getElementById('shopping-list');
            const newItem = document.createElement('li');
            newItem.textContent = 'Новый элемент';
            list.appendChild(newItem);
        }
    <\/script>
</body>
</html>
`
  }}
