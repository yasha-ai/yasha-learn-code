##  HTML: Скелет - Урок "Списки"

Списки - это важный элемент HTML, позволяющий структурировать информацию на веб-странице. Они помогают представить данные в организованном и легко читаемом формате. В этом уроке мы рассмотрим два основных типа списков: упорядоченные и неупорядоченные.

###  Упорядоченные списки (`<ol>`)

Упорядоченные списки используются для представления элементов в определенном порядке. Каждый элемент списка нумеруется.

```html
<ol>
  <li>Первый элемент</li>
  <li>Второй элемент</li>
  <li>Третий элемент</li>
</ol>
```

В этом примере `ol` означает "ordered list" (упорядоченный список), а `li` - "list item" (элемент списка). Браузер отобразит этот код следующим образом:

1.  Первый элемент
2.  Второй элемент
3.  Третий элемент

Можно изменить тип нумерации, используя атрибут `type` у тега `<ol>`. Например:

```html
<ol type="A">
  <li>Первый элемент</li>
  <li>Второй элемент</li>
  <li>Третий элемент</li>
</ol>
```

Это отобразит список с буквами:

A. Первый элемент
B. Второй элемент
C. Третий элемент

Другие возможные значения `type`: `1` (по умолчанию), `a` (маленькие буквы), `I` (римские цифры).

###  Неупорядоченные списки (`<ul>`)

Неупорядоченные списки используются, когда порядок элементов не важен. Каждый элемент списка обычно отмечается маркером (например, точкой).

```html
<ul>
  <li>Первый элемент</li>
  <li>Второй элемент</li>
  <li>Третий элемент</li>
</ul>
```

Здесь `ul` означает "unordered list" (неупорядоченный список). Браузер отобразит это как:

*   Первый элемент
*   Второй элемент
*   Третий элемент

Атрибут `type` у тега `<ul>` позволяет изменить тип маркера (хотя сейчас рекомендуется использовать CSS для стилизации списков). Возможные значения `type`: `disc` (по умолчанию, закрашенный круг), `circle` (пустой круг), `square` (квадрат).

```html
<ul type="square">
  <li>Первый элемент</li>
  <li>Второй элемент</li>
  <li>Третий элемент</li>
</ul>
```

###  Жизненный пример

Списки широко используются на веб-сайтах. Например:

*   **Меню навигации:** Часто реализуется с помощью неупорядоченных списков.
*   **Список дел:** Может быть представлен упорядоченным или неупорядоченным списком.
*   **Шаги в инструкции:** Идеально подходит для упорядоченного списка.
*   **Перечень характеристик товара:** Часто используют неупорядоченные списки.

Фреймворки, такие как React, Vue и Angular, используют списки для рендеринга динамических данных. Например, список пользователей, полученный с сервера, может быть отображен с помощью списков HTML.

```javascript
// Пример React компонента, отображающего список пользователей
function UserList({ users }) {
  return (
    <ul>
      {users.map(user => (
        <li key={user.id}>{user.name}</li>
      ))}
    </ul>
  );
}
```

###  Ключевые моменты

*   `<ol>` - упорядоченный список, элементы нумеруются.
*   `<ul>` - неупорядоченный список, элементы отмечаются маркерами.
*   `<li>` - элемент списка, используется внутри `<ol>` и `<ul>`.
*   Атрибут `type` позволяет изменить стиль нумерации/маркера (лучше использовать CSS).
*   Списки используются для структурирования информации и навигации.
