# HTML: Основы Canvas

![Иллюстрация к уроку](/lessons/html-canvas-basics.png)


Привет! Сегодня мы научимся рисовать прямо в браузере. Мы будем использовать элемент `<canvas>`, который превращает веб-страницу в настоящий холст.

### Что такое Canvas?

`<canvas>` — это элемент для рисования 2D-графики с помощью JavaScript. Представьте его как пустую доску, на которой можно рисовать линии, фигуры, текст и даже создавать анимации.

### Основы работы

Чтобы начать рисовать, нужно:
1. Создать элемент `<canvas>` в HTML.
2. Получить доступ к нему через JavaScript.
3. Получить "контекст" (`2d`) — это ваш набор инструментов (кисти, краски).

### Пример 1: Рисование прямоугольников

<Playground
  template="static"
  files={{
    "index.html": `<!DOCTYPE html>
<html>
<body>
  <canvas id="rectCanvas" width="400" height="200" style="border:1px solid #333; background: #eee;"></canvas>

  <script>
    const canvas = document.getElementById('rectCanvas');
    const ctx = canvas.getContext('2d');

    // Синий закрашенный прямоугольник
    ctx.fillStyle = 'blue';
    ctx.fillRect(20, 20, 150, 100);

    // Красная рамка (прямоугольник без заливки)
    ctx.strokeStyle = 'red';
    ctx.lineWidth = 5;
    ctx.strokeRect(200, 20, 150, 100);
  </script>
</body>
</html>`
  }}
/>

### Пример 2: Линии и круги (Пути)

Для сложных фигур используются "пути" (`beginPath`, `moveTo`, `lineTo`, `arc`).

<Playground
  template="static"
  files={{
    "index.html": `<!DOCTYPE html>
<html>
<body>
  <canvas id="pathCanvas" width="400" height="200" style="border:1px solid #333;"></canvas>

  <script>
    const ctx = document.getElementById('pathCanvas').getContext('2d');

    // Рисуем треугольник
    ctx.beginPath();
    ctx.moveTo(50, 50);
    ctx.lineTo(150, 50);
    ctx.lineTo(100, 150);
    ctx.closePath();
    ctx.fillStyle = 'purple';
    ctx.fill();

    // Рисуем круг
    ctx.beginPath();
    ctx.arc(300, 100, 50, 0, Math.PI * 2);
    ctx.fillStyle = 'orange';
    ctx.fill();
    ctx.stroke();
  </script>
</body>
</html>`
  }}
/>

### Пример 3: Текст на холсте

<Playground

<body>
  <canvas id="textCanvas" width="400" height="100" style="border:1px solid #333;"></canvas>

    const ctx = document.getElementById('textCanvas').getContext('2d');

    ctx.fillText('Привет, Canvas!', 50, 60);

  }}
/>

### Ключевые моменты

* Координаты начинаются с (0,0) в верхнем левом углу.
* `fillRect` — закрасить, `strokeRect` — обвести.
* Всегда используйте `beginPath()` перед рисованием новой сложной фигуры.
* `requestAnimationFrame` — лучший способ создания анимаций на Canvas.
