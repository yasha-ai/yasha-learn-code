# HTML: Основы WCAG

В этом уроке мы познакомимся с основами WCAG (Web Content Accessibility Guidelines) – международным стандартом доступности веб-контента. Применение этих принципов сделает ваши страницы удобными для всех пользователей.

### Что такое WCAG?

WCAG - это набор рекомендаций по созданию веб-контента, доступного для людей с инвалидностью (зрительной, слуховой, моторной или когнитивной).

### Принципы POUR

WCAG строится на четырех принципах:
1. **Perceivable (Воспринимаемость):** Контент должен быть доступен для органов чувств (зрение, слух).
2. **Operable (Управляемость):** Интерфейс должен работать с разными устройствами ввода (клавиатура, голос).
3. **Understandable (Понятность):** Информация и работа интерфейса должны быть ясными.
4. **Robust (Надежность):** Контент должен корректно работать в разных браузерах и со вспомогательными технологиями.

### Практические примеры

**1. Альтернативный текст для изображений**
Атрибут `alt` обязателен для информативных картинок.

<Playground
  template="static"
  files={{
    "index.html": `<img src="https://images.unsplash.com/photo-1517336714731-489689fd1ca8?w=200" alt="Ноутбук на столе в светлой комнате">`
  }}
/>

**2. Структура заголовков**
Соблюдайте иерархию (h1 -> h2 -> h3).

<Playground
  template="static"
  files={{
    "index.html": `<h1>Главный заголовок</h1>
<section>
  <h2>Основы дизайна</h2>
  <h3>Цветовая теория</h3>
  <p>Текст подраздела.</p>
</section>`
  }}
/>

**3. Семантические элементы**
Используйте смысловые теги вместо универсальных `div`.

<Playground
  template="static"
  files={{
    "index.html": `<nav>
  <a href="/">Домой</a>
</nav>
<main>
  <article>
    <h2>Моя статья</h2>
    <p>Содержание...</p>
  </article>
</main>
<footer>
  <p>&copy; 2026 Компания</p>
</footer>`
  }}
/>

**4. Контрастность**
Текст должен хорошо читаться на фоне.

<Playground
  template="static"
  files={{
    "index.html": `<!-- Плохой контраст -->
<div style="background: #eee; color: #ccc; padding: 10px;">
  Этот текст трудно прочитать.
</div>

<!-- Хороший контраст -->
<div style="background: #000; color: #fff; padding: 10px; margin-top: 10px;">
  Этот текст читается отлично.
</div>`
  }}
/>

### Ключевые моменты

* WCAG – это стандарт, помогающий сделать интернет доступным для всех.
* Принципы POUR: Воспринимаемость, Управляемость, Понятность, Надежность.
* Семантика, правильные заголовки и альтернативный текст — база доступности.
* Всегда проверяйте контрастность цветов.
