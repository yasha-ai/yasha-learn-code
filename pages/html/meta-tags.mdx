# HTML: Meta-теги

![Иллюстрация к уроку](/lessons/html-meta-tags.png)


Привет, друзья! Яша здесь, и сегодня мы погрузимся в мир "невидимых" элементов HTML, которые играют огромную роль в том, как ваш сайт воспринимается поисковыми системами и социальными сетями. Речь идет о `meta`-тегах!

### Что это и зачем

`meta`-теги — это элементы HTML, которые предоставляют метаданные о документе. Они не отображаются непосредственно на странице, но используются браузерами, поисковыми системами и социальными сетями.

**Зачем они нужны?**
* **SEO (Search Engine Optimization):** Помогают поисковым системам индексировать ваш контент.
* **Удобство для пользователя:** Определяют кодировку и масштабирование.
* **Социальные сети:** Контролируют внешний вид ссылок (превью).

### Основная теория с примерами

`meta`-теги всегда размещаются внутри секции `<head>`.

#### 1. Кодировка символов (charset)
Это один из самых важных мета-тегов. `UTF-8` — современный стандарт.

<Playground
  template="static"
  files={{
    "index.html": `<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Мой сайт с мета-тегами</title>
</head>
<body>
    <h1>Привет! Текст на русском отображается корректно.</h1>
</body>
</html>`
  }}
/>

#### 2. Адаптивность (viewport)
Этот тег критически важен для мобильных устройств.

<Playground
  template="static"
  files={{
    "index.html": `<head>
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Адаптивный сайт</title>
</head>`
  }}
/>

#### 3. Описание страницы (description)
Этот текст отображается в результатах поиска под заголовком.

<Playground
  template="static"
  files={{
    "index.html": `<head>
    <meta name="description" content="Учите HTML и CSS с Яшей! Простые и понятные уроки для начинающих.">
    <title>Yasha Learn Code</title>
</head>`
  }}
/>

#### 4. Индексация для роботов (robots)
Позволяет управлять поведением поисковых роботов.

<Playground
  template="static"
  files={{
    "index.html": `<!-- Разрешаем индексировать -->
<meta name="robots" content="index, follow">

<!-- Запрещаем индексировать (например, для админки) -->
<!-- <meta name="robots" content="noindex, nofollow"> -->`
  }}
/>

### Социальные мета-теги

#### Open Graph Protocol
Используется Facebook, Telegram, LinkedIn и другими.

<Playground
  template="static"
  files={{
    "index.html": `<head>
    <meta property="og:title" content="Урок по Meta-тегам">
    <meta property="og:description" content="Научитесь использовать meta-теги для SEO.">
    <meta property="og:image" content="https://images.unsplash.com/photo-1461749280684-dccba630e2f6?w=600">
    <meta property="og:url" content="https://yasha.dev/html/meta-tags">
    <meta property="og:type" content="article">
</head>`
  }}
/>

### Типичные ошибки

* **Отсутствие viewport:** Сайт будет выглядеть мелко на телефонах.
* **Пустые description:** Поисковики сами придумают описание, и оно может быть неудачным.
* **Слишком длинные описания:** Текст будет обрезан в поиске (старайтесь уложиться в 150-160 символов).

### Ключевые моменты

* Meta-теги находятся в `<head>`.
* `charset` и `viewport` — обязательный минимум для любого сайта.
* `description` напрямую влияет на то, перейдут ли к вам из поиска.
* Социальные теги (Open Graph) делают ваши ссылки привлекательными в мессенджерах.
