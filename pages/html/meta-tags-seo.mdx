#  HTML: Скелет - Meta-теги и SEO

![Иллюстрация к уроку](/lessons/html-meta-tags-seo.png)


Привет! Сегодня мы поговорим о meta-тегах, незаметных, но очень важных элементах HTML, которые помогают поисковым системам понимать содержание ваших веб-страниц и правильно их отображать. Они играют ключевую роль в SEO (Search Engine Optimization), то есть в оптимизации вашего сайта для поисковых систем.

## Что такое Meta-теги? 

Meta-теги - это HTML-теги, которые предоставляют метаданные о HTML-документе. Они не отображаются на самой странице, но содержат информацию о странице, такую как описание, ключевые слова, автор и другие важные данные.  Эта информация используется поисковыми системами, браузерами и другими веб-сервисами.  Meta-теги всегда располагаются внутри тега `<head>`.

### Основные Meta-теги:

*   **`description`:** Краткое описание страницы. Используется поисковыми системами для отображения в результатах поиска.
*   **`keywords`:** Ключевые слова, связанные с содержанием страницы.  (Хотя их важность снизилась, все еще полезно их указывать).
*   **`author`:** Информация об авторе страницы.
*   **`viewport`:**  Контролирует отображение страницы на разных устройствах (особенно важно для мобильных).
*   **`robots`:** Указывает поисковым роботам, как индексировать страницу.
*   **`charset`:** Определяет кодировку символов для страницы.

### Примеры кода:

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">  <!-- Указываем кодировку UTF-8 для поддержки русских символов -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0"> <!-- Адаптация под разные экраны -->
    <meta name="description" content="Курс по основам HTML для начинающих. Урок про meta-теги и их роль в SEO."> <!-- Описание страницы -->
    <meta name="keywords" content="HTML, meta-теги, SEO, веб-разработка, курс"> <!-- Ключевые слова -->
    <meta name="author" content="Ваш Имя"> <!-- Автор страницы -->
    <title>Meta-теги и SEO</title>
</head>
<body>
    <h1>Привет, мир!</h1>
    <p>Это пример страницы с meta-тегами.</p>
</body>
</html>
```

**Разберем некоторые важные meta-теги:**

*   **`charset="UTF-8"`**:  Обеспечивает правильное отображение текста на разных языках, включая русский.  Рекомендуется использовать UTF-8.

*   **`viewport`**:  `width=device-width` устанавливает ширину страницы равной ширине экрана устройства.  `initial-scale=1.0` устанавливает начальный масштаб страницы.

*   **`robots`**:
    ```html
    <meta name="robots" content="index, follow"> <!-- Разрешаем индексировать и переходить по ссылкам -->
    <meta name="robots" content="noindex, nofollow"> <!-- Запрещаем индексировать и переходить по ссылкам -->
    <meta name="robots" content="index, nofollow"> <!-- Разрешаем индексировать, но запрещаем переходить по ссылкам -->
    <meta name="robots" content="noindex, follow"> <!-- Запрещаем индексировать, но разрешаем переходить по ссылкам -->
    ```

##  Жизненный пример

Почти каждый веб-сайт использует meta-теги для улучшения SEO и удобства пользователей.

*   **Интернет-магазины:** Используют `description` с ключевыми словами, чтобы их товары отображались в результатах поиска при запросе конкретных продуктов.
*   **Блоги:** Используют `description` для краткого описания каждой статьи, чтобы привлечь читателей из поисковых систем.
*   **Новостные сайты:**  Используют meta-теги для указания авторства и категорий новостей.
*   **Фреймворки (React, Angular, Vue):**  При создании веб-приложений meta-теги настраиваются динамически в зависимости от текущей страницы или компонента. Например, заголовок и описание страницы могут меняться при переходе между разными разделами приложения.

##  Ключевые моменты

*   Meta-теги располагаются внутри тега `<head>`.
*   `description` важен для SEO, пишите краткие и информативные описания.
*   `viewport` необходим для адаптивного отображения на мобильных устройствах.
*   `charset="UTF-8"` обеспечивает правильное отображение текста.
*   Meta-теги помогают поисковым системам понимать содержание ваших страниц.
*   Правильное использование meta-тегов улучшает видимость вашего сайта в поисковых системах.
