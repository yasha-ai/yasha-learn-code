# HTML: Meta-теги и SEO

![Иллюстрация к уроку](/lessons/html-meta-tags-seo.png)

Привет! Сегодня мы поговорим о meta-тегах, незаметных, но очень важных элементах HTML, которые помогают поисковым системам понимать содержание ваших страниц и правильно их отображать. Они играют ключевую роль в SEO (Search Engine Optimization).

## Что такое Meta-теги?

Meta-теги - это элементы, которые предоставляют метаданные о документе. Они не отображаются на самой странице, но содержат информацию для поисковых систем и браузеров. Meta-теги всегда располагаются внутри тега `<head>`.

### Основные Meta-теги:

* **`description`**: Краткое описание страницы. Используется поисковиками в результатах поиска.
* **`viewport`**: Контролирует отображение на разных устройствах.
* **`robots`**: Указывает роботам, как индексировать страницу.
* **`charset`**: Определяет кодировку символов.

### Примеры кода:

### Важные теги подробнее:

* **`charset="UTF-8"`**: Обеспечивает правильное отображение русского текста.
* **`viewport`**: `width=device-width` делает страницу адаптивной.

Примеры для `robots`:

## Жизненный пример

Почти каждый сайт использует эти теги:
* **Интернет-магазины**: Добавляют описания товаров в `description`.
* **Блоги**: Указывают краткий анонс статьи.
* **Социальные сети**: Используют специальные meta-теги (Open Graph) для красивого отображения ссылок.

## Ключевые моменты

* Meta-теги находятся внутри `<head>`.
* `description` важен для кликабельности в поиске.
* `viewport` обязателен для мобильных устройств.
* `charset="UTF-8"` - стандарт индустрии.
