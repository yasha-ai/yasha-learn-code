# HTML: Datalist и автозаполнение

В этом уроке мы рассмотрим тег `<datalist>`, который позволяет добавлять подсказки (автозаполнение) к полям ввода. Это делает заполнение форм быстрее и удобнее для пользователя.

### Что такое `<datalist>`?

Тег `<datalist>` содержит список вариантов, которые браузер предложит пользователю при вводе текста в поле `<input>`. Он связывается с полем через атрибут `list`.

### Пример: Выбор браузера

Попробуйте начать вводить название браузера (например, "Ch" или "Fi") в поле ниже.

<Playground
  template="static"
  files={{
    "index.html": `<!DOCTYPE html>
<html>
<body>
  <label for="browser-choice">Выберите ваш любимый браузер:</label><br><br>
  
  <input list="browsers" id="browser-choice" name="browser" placeholder="Начните вводить...">

  <datalist id="browsers">
    <option value="Chrome">
    <option value="Firefox">
    <option value="Safari">
    <option value="Opera">
    <option value="Edge">
  </datalist>
</body>
</html>`
  }}
/>

### Как это работает?

1. Вы создаете обычный `<input>`.
2. Добавляете ему атрибут `list="имя_списка"`.
3. Создаете `<datalist>` с `id="имя_списка"`.
4. Внутри `datalist` перечисляете варианты через теги `<option>`.

### Отличие от `<select>`

В отличие от обычного выпадающего списка (`<select>`), `<datalist>` позволяет пользователю вводить свой собственный вариант, которого нет в списке. Список служит лишь набором подсказок.

### Где это применяется?

* **Поиск:** Предложение популярных запросов.
* **Города/Страны:** Подсказки при вводе адреса.
* **Теги:** Выбор категории или тега из существующих.

### Ключевые моменты

* `<datalist>` не отображается на странице сам по себе.
* Связь происходит через `input[list]` и `datalist[id]`.
* Это нативный способ сделать автозаполнение без использования тяжелых JavaScript-библиотек.
