# HTML: Семантика и доступность


![Иллюстрация к уроку](/lessons/html-semantic-html-a11y.png)
В этом уроке мы рассмотрим, как использовать семантический HTML для создания доступных и понятных веб-страниц. Это помогает поисковым системам и скринридерам лучше понимать структуру вашего сайта.

### Что такое Semantic HTML?

Semantic HTML — это использование тегов, которые несут смысл. Вместо того чтобы делать всё на `<div>`, мы используем элементы, которые говорят сами за себя: `<article>`, `<nav>`, `<header>`, `<footer>`.

Посмотрите, как семантические теги организуют страницу.

### Ключевые моменты

* Семантический HTML делает код читаемым и для людей, и для машин.
* Правильные теги улучшают доступность (a11y) "из коробки".
* Поисковики (SEO) лучше ранжируют сайты с четкой семантической структурой.
* Используйте `<div>` только для стилизации, когда ни один смысловой тег не подходит.
