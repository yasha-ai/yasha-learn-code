# HTML: Структура документа

![Иллюстрация к уроку](/lessons/html-document-structure.png)

Привет, будущие веб-разработчики! Сегодня мы изучим основу любой веб-страницы – её структуру. Понимание структуры HTML документа – это первый и важнейший шаг к созданию красивых и функциональных веб-сайтов.

### Основные элементы структуры

HTML документ имеет четкую иерархическую структуру. Это как скелет, который держит все вместе. Основные элементы, которые всегда присутствуют в HTML-документе:

* `<!DOCTYPE html>`: Объявление типа документа. Говорит браузеру, что это HTML5 документ.
* `<html>`: Корневой элемент, который содержит все остальные элементы страницы.
* `<head>`: Содержит метаданные о странице, такие как заголовок, кодировка, подключенные стили и скрипты.
* `<body>`: Содержит все содержимое страницы, которое видит пользователь.

Вот пример базовой структуры HTML документа:

<Playground
  template="static"
  files={{
    "index.html": `<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Моя первая страница</title>
</head>
<body>
    <h1>Привет, мир!</h1>
    <p>Это моя первая веб-страница.</p>
</body>
</html>`
  }}
/>

### Практический пример

Давайте создадим простую веб-страницу с семантической структурой:

<Playground
  template="static"
  files={{
    "index.html": `<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Мой блог</title>
</head>
<body>
    <header>
        <h1>Добро пожаловать в мой блог!</h1>
        <nav>
            <a href="#">Главная</a> |
            <a href="#">Обо мне</a>
        </nav>
    </header>

    <main>
        <article>
            <h2>Первая статья</h2>
            <p>Это первая статья в моем блоге.</p>
        </article>
    </main>

    <footer>
        <p>© 2026 Мой блог</p>
    </footer>
</body>
</html>`
  }}
/>

### Ключевые моменты

* `<!DOCTYPE html>`: Всегда указывайте в начале документа.
* `<html>` является корневым элементом.
* `<head>` содержит метаданные (не отображаются на странице).
* `<body>` содержит видимое содержимое страницы.
* Используйте семантические элементы (`<header>`, `<nav>`, `<main>`, `<article>`, `<footer>`) для улучшения структуры.
