# Git: Машина времени. Урок 3: Branches (ветки)


![Иллюстрация к уроку](/lessons/git-branches.png)
Ветки в Git - это мощный инструмент для параллельной работы над разными фичами вашего проекта, не затрагивая основную кодовую базу. Они позволяют вам экспериментировать, исправлять ошибки и разрабатывать новые функции изолированно, прежде чем интегрировать их в главную ветку.

## Что такое ветка?

Представьте, что у вас есть прямая дорога (главная ветка, обычно `main` или `master`). Ветка - это как ответвление от этой дороги. Вы сворачиваете на него, строите там что-то (вносите изменения в код), и потом, если все хорошо, возвращаетесь обратно на главную дорогу и сливаете свой участок с основной.

В Git ветка – это просто указатель на конкретный коммит.  Когда вы создаете ветку, Git создает новый указатель на тот же коммит, на котором вы находитесь.

## Основные команды для работы с ветками

*   `git branch <имя_ветки>` - создает новую ветку.
*   `git checkout <имя_ветки>` - переключает вас на указанную ветку.  Это как "сесть за руль" в другом ответвлении дороги.
*   `git checkout -b <имя_ветки>` - создает новую ветку и сразу же переключает вас на нее (сокращение для двух предыдущих команд).
*   `git branch` - показывает список всех существующих веток. Текущая ветка будет отмечена звездочкой (*).
*   `git merge <имя_ветки>` - сливает указанную ветку с текущей веткой. Это как "соединить" ответвление дороги с основной.
*   `git branch -d <имя_ветки>` - удаляет ветку (только если она была слита с другой веткой).
*   `git branch -D <имя_ветки>` - принудительно удаляет ветку (даже если она не была слита).

## Практические примеры

```bash
# Создаем новую ветку под названием "feature/new-button"
git branch feature/new-button

# Переключаемся на эту ветку
git checkout feature/new-button

# Или, используя сокращенную команду:
git checkout -b feature/new-button

# Вносим изменения в код, например, добавляем кнопку в HTML:
# (Предположим, у нас есть index.html)
# ```html
# <!DOCTYPE html>
# <html>
# <head>
#   <title>Мой сайт</title>
# </head>
# <body>
#   <h1>Привет, мир!</h1>
#   <!-- Здесь будет новая кнопка -->
#   <button>Кнопка</button>
# </body>
# </html>
# ```

# Фиксируем изменения
git add index.html
git commit -m "Добавлена новая кнопка на страницу"

# Возвращаемся в главную ветку (например, main)
git checkout main

# Сливаем изменения из ветки "feature/new-button" в главную ветку
git merge feature/new-button

# Удаляем ветку "feature/new-button" (после успешного слияния)
git branch -d feature/new-button
```

## Жизненный пример

Представьте, что вы работаете над большим веб-приложением.  Один разработчик работает над новой системой авторизации (ветка `feature/auth`), другой - исправляет ошибку в отображении профиля пользователя (ветка `bugfix/profile-display`), а третий - добавляет новую функциональность для администраторов (ветка `feature/admin-panel`).

Каждый разработчик работает в своей ветке, не мешая друг другу. Когда работа завершена и протестирована, ветка разработчика сливается с главной веткой (`main`), и изменения становятся частью основного приложения.

Фреймворки, такие как React, Angular, Vue.js, активно используют ветки при разработке новых версий или исправлении ошибок. Разработчики фреймворков работают в отдельных ветках, а затем сливают их в основную ветку репозитория.

## Ключевые моменты

*   Ветки позволяют работать над разными задачами параллельно.
*   Они изолируют изменения, предотвращая конфликты в основной кодовой базе.
*   `git branch`, `git checkout`, `git merge` - основные команды для работы с ветками.
*   Используйте ветки для разработки новых функций, исправления ошибок и экспериментов.
*   Всегда проверяйте и тестируйте изменения в ветке перед слиянием с главной веткой.


## Визуализация веток

Интерактивная схема показывает, как ветки расходятся от `main` и сливаются обратно. Это именно то, что происходит внутри Git:

  template="static"
  files={{
    "/index.html": {
      code: `<!DOCTYPE html>
<html lang="ru">
<head>
  <meta charset="UTF-8">
  <title>Git Branches Visualization</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <div class="controls">
    <button id="btn-feature" class="btn btn-purple">+ feature ветка</button>
    <button id="btn-commit-feat" class="btn btn-purple" disabled>Коммит в feature</button>
    <button id="btn-bugfix" class="btn btn-orange">+ bugfix ветка</button>
    <button id="btn-commit-main" class="btn btn-blue">Коммит в main</button>
    <button id="btn-merge" class="btn btn-green" disabled>Merge feature → main</button>
    <button id="btn-reset" class="btn btn-gray">Сброс</button>
  </div>

  <div class="legend">
    <span class="dot main"></span>main
    <span class="dot feature"></span>feature
    <span class="dot bugfix"></span>bugfix
    <span class="dot merge"></span>merge commit
  </div>

  <canvas id="canvas"></canvas>
  <div id="log" class="log"></div>

  <script>
    const canvas = document.getElementById('canvas');
    const ctx = canvas.getContext('2d');
    const log = document.getElementById('log');

    const COLORS = {
      main: '#6366f1',
      feature: '#a855f7',
      bugfix: '#f59e0b',
      merge: '#22c55e',
    };

    const ROW = { main: 80, feature: 160, bugfix: 240 };
    const COL_START = 40;
    const COL_STEP = 80;

    let state = {
      commits: [
        { id: 'A', branch: 'main', col: 0, label: 'Initial' },
        { id: 'B', branch: 'main', col: 1, label: 'Add README' },
      ],
      featureStart: null,
      bugfixStart: null,
      featureCreated: false,
      bugfixCreated: false,
      featureMerged: false,
      mainCol: 1,
      featureCol: null,
      bugfixCol: null,
    };

    function resize() {
      canvas.width = canvas.offsetWidth;
      canvas.height = 300;
      draw();
    }

    function addLog(msg, color) {
      const line = document.createElement('div');
      line.innerHTML = \`<span style="color:\${color}">▶</span> \${msg}\`;
      log.prepend(line);
    }

    function getX(col) { return COL_START + col * COL_STEP; }
    function getY(branch) { return ROW[branch]; }

    function drawCommit(commit, highlight) {
      const x = getX(commit.col);
      const y = getY(commit.branch);
      const color = COLORS[commit.mergeCommit ? 'merge' : commit.branch];

      ctx.beginPath();
      ctx.arc(x, y, 14, 0, Math.PI * 2);
      ctx.fillStyle = highlight ? '#fff' : color;
      ctx.fill();
      ctx.strokeStyle = color;
      ctx.lineWidth = 3;
      ctx.stroke();

      ctx.fillStyle = highlight ? color : '#0f172a';
      ctx.font = 'bold 11px monospace';
      ctx.textAlign = 'center';
      ctx.textBaseline = 'middle';
      ctx.fillText(commit.id, x, y);

      ctx.fillStyle = '#94a3b8';
      ctx.font = '10px system-ui';
      ctx.fillText(commit.label, x, y + 26);
    }

    function drawLine(x1, y1, x2, y2, color, dashed) {
      ctx.beginPath();
      if (dashed) ctx.setLineDash([5, 4]);
      ctx.moveTo(x1, y1);
      ctx.lineTo(x2, y2);
      ctx.strokeStyle = color;
      ctx.lineWidth = 3;
      ctx.stroke();
      ctx.setLineDash([]);
    }

    function draw() {
      ctx.clearRect(0, 0, canvas.width, canvas.height);

      // Соединяем коммиты линиями
      const main = state.commits.filter(c => c.branch === 'main' || c.mergeCommit);
      for (let i = 1; i < main.length; i++) {
        const prev = main[i-1], cur = main[i];
        drawLine(getX(prev.col), getY('main'), getX(cur.col), getY('main'),
          cur.mergeCommit ? COLORS.merge : COLORS.main);
      }

      // Feature линия
      const feat = state.commits.filter(c => c.branch === 'feature');
      if (feat.length) {
        // Ветвление от main
        const origin = state.commits.find(c => c.id === state.featureStart);
        drawLine(getX(origin.col), getY('main'), getX(feat[0].col), getY('feature'),
          COLORS.feature, true);
        for (let i = 1; i < feat.length; i++) {
          drawLine(getX(feat[i-1].col), getY('feature'), getX(feat[i].col), getY('feature'),
            COLORS.feature);
        }
        // Линия к merge-коммиту
        const merge = state.commits.find(c => c.mergeCommit);
        if (merge) {
          drawLine(getX(feat[feat.length-1].col), getY('feature'), getX(merge.col), getY('main'),
            COLORS.merge, true);
        }
      }

      // Bugfix линия
      const bug = state.commits.filter(c => c.branch === 'bugfix');
      if (bug.length) {
        const origin = state.commits.find(c => c.id === state.bugfixStart);
        drawLine(getX(origin.col), getY('main'), getX(bug[0].col), getY('bugfix'),
          COLORS.bugfix, true);
        for (let i = 1; i < bug.length; i++) {
          drawLine(getX(bug[i-1].col), getY('bugfix'), getX(bug[i].col), getY('bugfix'),
            COLORS.bugfix);
        }
      }

      // Рисуем все коммиты
      state.commits.forEach(c => drawCommit(c, c.mergeCommit));

      // Метки веток
      const labels = [
        { branch: 'main',    col: state.mainCol, label: 'main', color: COLORS.main },
      ];
      if (state.featureCol !== null && !state.featureMerged) {
        labels.push({ branch: 'feature', col: state.featureCol, label: 'feature', color: COLORS.feature });
      }
      if (state.bugfixCol !== null) {
        labels.push({ branch: 'bugfix', col: state.bugfixCol, label: 'bugfix', color: COLORS.bugfix });
      }

      labels.forEach(({ branch, col, label, color }) => {
        const x = getX(col) + 20, y = getY(branch);
        ctx.fillStyle = color;
        ctx.font = 'bold 12px monospace';
        ctx.textAlign = 'left';
        ctx.textBaseline = 'middle';
        ctx.fillText('← ' + label, x, y);
      });
    }

    let commitCounter = 0;
    function nextId() {
      const letters = 'ABCDEFGHIJKLMNOPQRSTUVWXYZ';
      commitCounter++;
      return letters[state.commits.length % 26] + (state.commits.length > 25 ? Math.floor(state.commits.length/26) : '');
    }

    document.getElementById('btn-feature').onclick = () => {
      if (state.featureCreated) return;
      state.featureCreated = true;
      state.featureStart = state.commits.filter(c => c.branch === 'main').slice(-1)[0].id;
      state.mainCol++;
      state.featureCol = state.mainCol;
      state.commits.push({ id: nextId(), branch: 'feature', col: state.featureCol, label: 'feature start' });
      document.getElementById('btn-commit-feat').disabled = false;
      document.getElementById('btn-merge').disabled = false;
      document.getElementById('btn-feature').disabled = true;
      addLog('git checkout -b feature', COLORS.feature);
      draw();
    };

    document.getElementById('btn-commit-feat').onclick = () => {
      state.featureCol++;
      state.commits.push({ id: nextId(), branch: 'feature', col: state.featureCol, label: 'feat commit' });
      addLog('git commit -m "feat: ..."', COLORS.feature);
      draw();
    };

    document.getElementById('btn-bugfix').onclick = () => {
      if (state.bugfixCreated) return;
      state.bugfixCreated = true;
      state.bugfixStart = state.commits.filter(c => c.branch === 'main').slice(-1)[0].id;
      state.mainCol++;
      state.bugfixCol = state.mainCol;
      state.commits.push({ id: nextId(), branch: 'bugfix', col: state.bugfixCol, label: 'bugfix start' });
      document.getElementById('btn-bugfix').disabled = true;
      addLog('git checkout -b bugfix', COLORS.bugfix);
      draw();
    };

    document.getElementById('btn-commit-main').onclick = () => {
      state.mainCol++;
      state.commits.push({ id: nextId(), branch: 'main', col: state.mainCol, label: 'main commit' });
      addLog('git commit -m "..."  (main)', COLORS.main);
      draw();
    };

    document.getElementById('btn-merge').onclick = () => {
      if (!state.featureCreated || state.featureMerged) return;
      state.featureMerged = true;
      state.mainCol = Math.max(state.mainCol, state.featureCol) + 1;
      state.commits.push({ id: nextId(), branch: 'main', col: state.mainCol, label: 'merge', mergeCommit: true });
      document.getElementById('btn-merge').disabled = true;
      document.getElementById('btn-commit-feat').disabled = true;
      addLog('git checkout main && git merge feature', COLORS.merge);
      draw();
    };

    document.getElementById('btn-reset').onclick = () => {
      state = {
        commits: [
          { id: 'A', branch: 'main', col: 0, label: 'Initial' },
          { id: 'B', branch: 'main', col: 1, label: 'Add README' },
        ],
        featureStart: null, bugfixStart: null,
        featureCreated: false, bugfixCreated: false, featureMerged: false,
        mainCol: 1, featureCol: null, bugfixCol: null,
      };
      commitCounter = 1;
      ['btn-feature','btn-bugfix','btn-commit-main'].forEach(id => document.getElementById(id).disabled = false);
      ['btn-commit-feat','btn-merge'].forEach(id => document.getElementById(id).disabled = true);
      log.innerHTML = '';
      addLog('Сброс. Начинаем заново.', '#64748b');
      draw();
    };

    window.addEventListener('resize', resize);
    resize();
    addLog('Нажми кнопки выше, чтобы создать ветки и коммиты', '#64748b');
  <\/script>
</body>
