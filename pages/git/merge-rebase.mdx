## Git: Merge vs Rebase ‚Äì –ò–Ω—Ç–µ–≥—Ä–∞—Ü–∏—è –∏–∑–º–µ–Ω–µ–Ω–∏–π —á–∏—Å—Ç–æ –∏ —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ

**–ó–∞–¥–∞—á–∞**: –í —ç—Ç–æ–º —É—Ä–æ–∫–µ –º—ã —Ä–∞–∑–±–µ—Ä–µ–º –¥–≤–∞ –æ—Å–Ω–æ–≤–Ω—ã—Ö —Å–ø–æ—Å–æ–±–∞ –∏–Ω—Ç–µ–≥—Ä–∞—Ü–∏–∏ –∏–∑–º–µ–Ω–µ–Ω–∏–π –∏–∑ –æ–¥–Ω–æ–π –≤–µ—Ç–∫–∏ –≤ –¥—Ä—É–≥—É—é: `git merge` –∏ `git rebase`. –í—ã —É–∑–Ω–∞–µ—Ç–µ, –∫–æ–≥–¥–∞ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –∫–∞–∂–¥—ã–π –∏–∑ –Ω–∏—Ö, –∏—Ö –ø—Ä–µ–∏–º—É—â–µ—Å—Ç–≤–∞, –Ω–µ–¥–æ—Å—Ç–∞—Ç–∫–∏ –∏ –∫–∞–∫ –∏–∑–±–µ–∂–∞—Ç—å —Ç–∏–ø–∏—á–Ω—ã—Ö –ø—Ä–æ–±–ª–µ–º. –ü–æ–Ω–∏–º–∞–Ω–∏–µ —ç—Ç–∏—Ö –∏–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç–æ–≤ –∫—Ä–∏—Ç–∏—á–µ—Å–∫–∏ –≤–∞–∂–Ω–æ –¥–ª—è —Ä–∞–±–æ—Ç—ã –≤ –∫–æ–º–∞–Ω–¥–µ –∏ –ø–æ–¥–¥–µ—Ä–∂–∞–Ω–∏—è —á–∏—Å—Ç–æ–≥–æ —Ä–µ–ø–æ–∑–∏—Ç–æ—Ä–∏—è.

### Git Merge: –°–æ—Ö—Ä–∞–Ω—è–µ–º –∏—Å—Ç–æ—Ä–∏—é –∫–∞–∫ –æ–Ω–∞ –µ—Å—Ç—å

**–ö–æ–Ω—Ü–µ–ø—Ü–∏—è**: `git merge` –æ–±—ä–µ–¥–∏–Ω—è–µ—Ç –¥–≤–µ –∏—Å—Ç–æ—Ä–∏–∏, —Å–æ–∑–¥–∞–≤–∞—è –Ω–æ–≤—ã–π "[merge commit](/git/init-commit)". –≠—Ç–æ—Ç –∫–æ–º–º–∏—Ç –∏–º–µ–µ—Ç –¥–≤—É—Ö —Ä–æ–¥–∏—Ç–µ–ª–µ–π, —É–∫–∞–∑—ã–≤–∞—è –Ω–∞ –ø–æ—Å–ª–µ–¥–Ω–∏–π –∫–æ–º–º–∏—Ç –æ–±–µ–∏—Ö –æ–±—ä–µ–¥–∏–Ω—è–µ–º—ã—Ö –≤–µ—Ç–æ–∫. –≠—Ç–æ –Ω–µ–∏–Ω–≤–∞–∑–∏–≤–Ω–∞—è –æ–ø–µ—Ä–∞—Ü–∏—è: –æ–Ω–∞ –Ω–µ –∏–∑–º–µ–Ω—è–µ—Ç —Å—É—â–µ—Å—Ç–≤—É—é—â–∏–µ –∫–æ–º–º–∏—Ç—ã, –∞ –ª–∏—à—å –¥–æ–±–∞–≤–ª—è–µ—Ç –Ω–æ–≤—ã–π.

**–í–∏–∑—É–∞–ª—å–Ω–∞—è —Å—Ö–µ–º–∞ (3-way merge):**
```
      A -- B -- C (main)
           \      /
            D -- E (feature)
```
–ü–æ—Å–ª–µ `git merge feature` –≤ `main`:
```
      A -- B -- C -- M (main)
           \      /
            D ---- E
```
*(–ì–¥–µ `M` ‚Äì —ç—Ç–æ merge commit, –æ–±—ä–µ–¥–∏–Ω—è—é—â–∏–π `C` –∏ `E`)*

**–ö–æ–º–∞–Ω–¥—ã**:
–°–Ω–∞—á–∞–ª–∞ –ø–µ—Ä–µ–∫–ª—é—á–∏—Ç–µ—Å—å –Ω–∞ [–≤–µ—Ç–∫—É](/git/branches), –≤ –∫–æ—Ç–æ—Ä—É—é —Ö–æ—Ç–∏—Ç–µ –∏–Ω—Ç–µ–≥—Ä–∏—Ä–æ–≤–∞—Ç—å –∏–∑–º–µ–Ω–µ–Ω–∏—è (—Ü–µ–ª–µ–≤—É—é –≤–µ—Ç–∫—É):
```bash
git switch main
git merge feature # –û–±—ä–µ–¥–∏–Ω–∏—Ç—å –≤–µ—Ç–∫—É 'feature' –≤ —Ç–µ–∫—É—â—É—é 'main'
```

**–ü–æ—Å–ª–µ–¥—Å—Ç–≤–∏—è**:
*   –°–æ–∑–¥–∞–µ—Ç—Å—è –Ω–æ–≤—ã–π –∫–æ–º–º–∏—Ç (`M`).
*   –ò—Å—Ç–æ—Ä–∏—è —Å–æ—Ö—Ä–∞–Ω—è–µ—Ç—Å—è —Ç–æ—á–Ω–æ —Ç–∞–∫, –∫–∞–∫ –æ–Ω–∞ —Ä–∞–∑–≤–∏–≤–∞–ª–∞—Å—å, –Ω–æ –º–æ–∂–µ—Ç –≤—ã–≥–ª—è–¥–µ—Ç—å –≤–µ—Ç–≤–∏—Å—Ç–æ–π.
*   –û—Ç–ª–∏—á–Ω–æ –ø–æ–¥—Ö–æ–¥–∏—Ç –¥–ª—è –∏–Ω—Ç–µ–≥—Ä–∞—Ü–∏–∏ –≤ –æ–±—â–µ–¥–æ—Å—Ç—É–ø–Ω—ã–µ, —Å—Ç–∞–±–∏–ª—å–Ω—ã–µ –≤–µ—Ç–∫–∏.

**–ü—Ä–∏–º–µ—Ä 1: Fast-forward merge (–µ—Å–ª–∏ —Ü–µ–ª–µ–≤–∞—è –≤–µ—Ç–∫–∞ –Ω–µ –∏–∑–º–µ–Ω–∏–ª–∞—Å—å)**
```bash
git switch main
git checkout -b feature-ff
echo "Initial feature content" > feature.txt
git add feature.txt
git commit -m "feat: Add initial feature content"

git switch main # main –Ω–µ –∏–∑–º–µ–Ω–∏–ª–∞—Å—å —Å –º–æ–º–µ–Ω—Ç–∞ —Å–æ–∑–¥–∞–Ω–∏—è feature-ff
git merge feature-ff # –ü—Ä–æ–∏–∑–æ–π–¥–µ—Ç fast-forward: —É–∫–∞–∑–∞—Ç–µ–ª—å main –ø—Ä–æ—Å—Ç–æ –ø–µ—Ä–µ–º–µ—Å—Ç–∏—Ç—Å—è –≤–ø–µ—Ä–µ–¥
```
_**–†–µ–∑—É–ª—å—Ç–∞—Ç:** –ö–æ–º–º–∏—Ç—ã `feature-ff` —Ç–µ–ø–µ—Ä—å –Ω–∞–ø—Ä—è–º—É—é –Ω–∞ `main` –±–µ–∑ merge-–∫–æ–º–º–∏—Ç–∞._

**–ü—Ä–∏–º–µ—Ä 2: Merge —Å [–∫–æ–Ω—Ñ–ª–∏–∫—Ç–æ–º](/git/conflicts)**
```bash
git switch main
git checkout -b conflict-branch
echo "Line 1 from conflict-branch" > conflict.txt
git add conflict.txt
git commit -m "feat: Add line from conflict-branch"

git switch main
echo "Line 1 from main" > conflict.txt # –ò–∑–º–µ–Ω—è–µ–º —Ç—É –∂–µ —Å—Ç—Ä–æ–∫—É
git add conflict.txt
git commit -m "feat: Add line from main"

git merge conflict-branch # –í—ã–∑–æ–≤–µ—Ç –∫–æ–Ω—Ñ–ª–∏–∫—Ç, —Ç–∞–∫ –∫–∞–∫ –æ–¥–Ω–∞ –∏ —Ç–∞ –∂–µ —Å—Ç—Ä–æ–∫–∞ –∏–∑–º–µ–Ω–µ–Ω–∞
# !!! –û—Ç–∫—Ä–æ–π—Ç–µ conflict.txt, —Ä–∞–∑—Ä–µ—à–∏—Ç–µ –∫–æ–Ω—Ñ–ª–∏–∫—Ç –≤—Ä—É—á–Ω—É—é !!!
# –ó–∞—Ç–µ–º:
git add conflict.txt
git commit -m "Merge branch 'conflict-branch' into main" # –°–æ–∑–¥–∞–µ—Ç—Å—è merge commit
```
_**–†–µ–∑—É–ª—å—Ç–∞—Ç:** –ò—Å—Ç–æ—Ä–∏—è —Å merge-–∫–æ–º–º–∏—Ç–æ–º, —Ä–∞–∑—Ä–µ—à–∏–≤—à–∏–º –∫–æ–Ω—Ñ–ª–∏–∫—Ç._

### Git Rebase: –ü–µ—Ä–µ–ø–∏—Å—ã–≤–∞–µ–º –∏—Å—Ç–æ—Ä–∏—é –¥–ª—è —á–∏—Å—Ç–æ—Ç—ã

**–ö–æ–Ω—Ü–µ–ø—Ü–∏—è**: `git rebase` –±–µ—Ä–µ—Ç –∫–æ–º–º–∏—Ç—ã –∏–∑ –≤–∞—à–µ–π –≤–µ—Ç–∫–∏ –∏ "–ø–µ—Ä–µ–º–µ—â–∞–µ—Ç" –∏—Ö –ø–æ–≤–µ—Ä—Ö –¥—Ä—É–≥–æ–π –≤–µ—Ç–∫–∏. –≠—Ç–æ **–ø–µ—Ä–µ–ø–∏—Å—ã–≤–∞–µ—Ç –∏—Å—Ç–æ—Ä–∏—é**, —Å–æ–∑–¥–∞–≤–∞—è –Ω–æ–≤—ã–µ –∫–æ–º–º–∏—Ç—ã –¥–ª—è –∫–∞–∂–¥–æ–≥–æ –ø–µ—Ä–µ–Ω–µ—Å–µ–Ω–Ω–æ–≥–æ, –≤–º–µ—Å—Ç–æ —Å—Ç–∞—Ä—ã—Ö. –¶–µ–ª—å ‚Äì –ø–æ–ª—É—á–∏—Ç—å —á–∏—Å—Ç—É—é, –ª–∏–Ω–µ–π–Ω—É—é –∏—Å—Ç–æ—Ä–∏—é.

**–í–∏–∑—É–∞–ª—å–Ω–∞—è —Å—Ö–µ–º–∞**:
```
      A -- B -- C (main)
           \
            D -- E (feature)
```
–ü–æ—Å–ª–µ `git rebase main` –≤ `feature`:
```
      A -- B -- C -- D' -- E' (feature)
```
*(–ì–¥–µ `D'`, `E'` ‚Äì —ç—Ç–æ –Ω–æ–≤—ã–µ –∫–æ–º–º–∏—Ç—ã, –∏–¥–µ–Ω—Ç–∏—á–Ω—ã–µ `D`, `E`, –Ω–æ —Å –Ω–æ–≤—ã–º —Ä–æ–¥–∏—Ç–µ–ª–µ–º `C`)*

**–ö–æ–º–∞–Ω–¥—ã**:
–ü–µ—Ä–µ–∫–ª—é—á–∏—Ç–µ—Å—å –Ω–∞ –≤–µ—Ç–∫—É, –∫–æ—Ç–æ—Ä—É—é –≤—ã —Ö–æ—Ç–∏—Ç–µ –ø–µ—Ä–µ–±–∞–∑–∏—Ä–æ–≤–∞—Ç—å (–æ–±—ã—á–Ω–æ –≤–∞—à–∞ —Ñ–∏—á–µ-–≤–µ—Ç–∫–∞):
```bash
git switch feature
git rebase main # –ü–µ—Ä–µ–Ω–µ—Å—Ç–∏ –∫–æ–º–º–∏—Ç—ã feature –ø–æ–≤–µ—Ä—Ö –∞–∫—Ç—É–∞–ª—å–Ω–æ–π main
# (–ï—Å–ª–∏ –µ—Å—Ç—å –∫–æ–Ω—Ñ–ª–∏–∫—Ç—ã, —Ä–∞–∑—Ä–µ—à–∏—Ç–µ –∏—Ö, –∑–∞—Ç–µ–º 'git rebase --continue')
```
–ü–æ—Å–ª–µ —É—Å–ø–µ—à–Ω–æ–≥–æ —Ä–µ–±–µ–π–∑–∞, —á—Ç–æ–±—ã `main` –æ—Ç—Ä–∞–∑–∏–ª–∞ —ç—Ç–∏ –∏–∑–º–µ–Ω–µ–Ω–∏—è:
```bash
git switch main
git merge feature # –û–±—ã—á–Ω–æ —ç—Ç–æ –±—É–¥–µ—Ç fast-forward, —Ç–∞–∫ –∫–∞–∫ feature —Ç–µ–ø–µ—Ä—å –ø–æ—Ç–æ–º–æ–∫ main
```

**–ü–æ—Å–ª–µ–¥—Å—Ç–≤–∏—è**:
*   –°–æ–∑–¥–∞–µ—Ç—Å—è –Ω–æ–≤–∞—è –ª–∏–Ω–µ–π–Ω–∞—è –∏—Å—Ç–æ—Ä–∏—è, —á—Ç–æ –æ–±–ª–µ–≥—á–∞–µ—Ç –µ–µ —á—Ç–µ–Ω–∏–µ.
*   **–û—Å—Ç–æ—Ä–æ–∂–Ω–æ**: –ø–µ—Ä–µ–ø–∏—Å—ã–≤–∞–µ—Ç –∏—Å—Ç–æ—Ä–∏—é (–∏–∑–º–µ–Ω—è–µ—Ç SHA1-—Ö–µ—à–∏ –∫–æ–º–º–∏—Ç–æ–≤), —á—Ç–æ –æ–ø–∞—Å–Ω–æ –¥–ª—è –æ–±—â–∏—Ö –≤–µ—Ç–æ–∫.

**–ü—Ä–∏–º–µ—Ä Rebase**:
```bash
git switch main
git checkout -b feature-rebase
echo "Rebase feature A" > rebase.txt
git add rebase.txt
git commit -m "feat: rebase feature A"
echo "Rebase feature B" >> rebase.txt
git add rebase.txt
git commit -m "feat: rebase feature B"

git switch main
echo "Main update for rebase" > main.txt
git add main.txt
git commit -m "feat: main updated"

git switch feature-rebase
git rebase main # –ü–µ—Ä–µ–º–µ—Å—Ç–∏—Ç –∫–æ–º–º–∏—Ç—ã "rebase feature A" –∏ "rebase feature B" –ø–æ–≤–µ—Ä—Ö "main updated"
# (–ï—Å–ª–∏ –≤–æ–∑–Ω–∏–∫–Ω—É—Ç –∫–æ–Ω—Ñ–ª–∏–∫—Ç—ã, —Ä–∞–∑—Ä–µ—à–∏—Ç–µ –∏—Ö –∏ –≤—ã–ø–æ–ª–Ω–∏—Ç–µ 'git rebase --continue')

git switch main
git merge feature-rebase # Fast-forward merge: main —Ç–µ–ø–µ—Ä—å —É–∫–∞–∑—ã–≤–∞–µ—Ç –Ω–∞ –ø–æ—Å–ª–µ–¥–Ω–∏–π –∫–æ–º–º–∏—Ç feature-rebase
```
_**–†–µ–∑—É–ª—å—Ç–∞—Ç:** –ß–∏—Å—Ç–∞—è, –ª–∏–Ω–µ–π–Ω–∞—è –∏—Å—Ç–æ—Ä–∏—è –≤ `main` –±–µ–∑ merge-–∫–æ–º–º–∏—Ç–æ–≤._

### –ö–æ–≥–¥–∞ —á—Ç–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å (–õ—É—á—à–∏–µ –ø—Ä–∞–∫—Ç–∏–∫–∏ Git)

*   **–ò—Å–ø–æ–ª—å–∑—É–π—Ç–µ `git merge` –∫–æ–≥–¥–∞:**
    *   –ò–Ω—Ç–µ–≥—Ä–∏—Ä—É–µ—Ç–µ –≤ **–æ–±—â–∏–µ, –æ–ø—É–±–ª–∏–∫–æ–≤–∞–Ω–Ω—ã–µ –≤–µ—Ç–∫–∏** (`main`, `develop`). Merge-–∫–æ–º–º–∏—Ç—ã –ø–æ–∫–∞–∑—ã–≤–∞—é—Ç, –∫–æ–≥–¥–∞ –±—ã–ª–∏ –∏–Ω—Ç–µ–≥—Ä–∏—Ä–æ–≤–∞–Ω—ã —Ñ–∏—á–∏, —Å–æ—Ö—Ä–∞–Ω—è—è –∏—Å—Ç–æ—Ä–∏—á–µ—Å–∫–∏–π –∫–æ–Ω—Ç–µ–∫—Å—Ç.
    *   –í—ã —Ö–æ—Ç–∏—Ç–µ —Å–æ—Ö—Ä–∞–Ω–∏—Ç—å **—Ç–æ—á–Ω—É—é, –Ω–µ–∏–∑–º–µ–Ω–Ω—É—é –∏—Å—Ç–æ—Ä–∏—é** —Å merge-–∫–æ–º–º–∏—Ç–∞–º–∏, –¥–∞–∂–µ –µ—Å–ª–∏ –æ–Ω–∞ –≤—ã–≥–ª—è–¥–∏—Ç –º–µ–Ω–µ–µ –ª–∏–Ω–µ–π–Ω–æ–π.
    *   –í—ã —Ä–∞–±–æ—Ç–∞–µ—Ç–µ –≤ –∫–æ–º–∞–Ω–¥–µ –∏ –Ω–µ —Ö–æ—Ç–∏—Ç–µ –∑–∞—Å—Ç–∞–≤–ª—è—Ç—å –∫–æ–ª–ª–µ–≥ –¥–µ–ª–∞—Ç—å `git pull --rebase` –∏–ª–∏ `git push --force`.
*   **–ò—Å–ø–æ–ª—å–∑—É–π—Ç–µ `git rebase` –∫–æ–≥–¥–∞:**
    *   –í—ã —Ö–æ—Ç–∏—Ç–µ –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞—Ç—å **—á–∏—Å—Ç—É—é, –ª–∏–Ω–µ–π–Ω—É—é –∏—Å—Ç–æ—Ä–∏—é** –≤ —Å–≤–æ–µ–π **–ª–æ–∫–∞–ª—å–Ω–æ–π, –Ω–µ–æ–ø—É–±–ª–∏–∫–æ–≤–∞–Ω–Ω–æ–π –≤–µ—Ç–∫–µ** –ø–µ—Ä–µ–¥ —Å–ª–∏—è–Ω–∏–µ–º —Å `main`.
    *   –í–∞–º –Ω—É–∂–Ω–æ "–ø–æ–¥—Ç—è–Ω—É—Ç—å" –∏–∑–º–µ–Ω–µ–Ω–∏—è –∏–∑ `main` –≤ –≤–∞—à—É —Ñ–∏—á–µ-–≤–µ—Ç–∫—É, —á—Ç–æ–±—ã –∏–∑–±–µ–∂–∞—Ç—å –±–æ–ª—å—à–æ–≥–æ merge-–∫–æ–º–º–∏—Ç–∞ –∏–ª–∏ –∫–æ–Ω—Ñ–ª–∏–∫—Ç–æ–≤ –ø—Ä–∏ —Ñ–∏–Ω–∞–ª—å–Ω–æ–º —Å–ª–∏—è–Ω–∏–∏. –†–µ–≥—É–ª—è—Ä–Ω—ã–π —Ä–µ–±–µ–π–∑ –ø–æ–º–æ–≥–∞–µ—Ç —Ä–∞–Ω—å—à–µ –Ω–∞–π—Ç–∏ –∫–æ–Ω—Ñ–ª–∏–∫—Ç—ã.
    *   **–ó–æ–ª–æ—Ç–æ–µ –ø—Ä–∞–≤–∏–ª–æ**: **–ù–ò–ö–û–ì–î–ê** –Ω–µ –¥–µ–ª–∞–π—Ç–µ `git rebase` –Ω–∞ –≤–µ—Ç–∫–µ, –∫–æ—Ç–æ—Ä—É—é –≤—ã —É–∂–µ –æ–ø—É–±–ª–∏–∫–æ–≤–∞–ª–∏ (—Ç.–µ., [`git push`](/git/remote-push-pull) –µ–µ), –∏ —Å –∫–æ—Ç–æ—Ä–æ–π —Ä–∞–±–æ—Ç–∞—é—Ç –¥—Ä—É–≥–∏–µ –ª—é–¥–∏! –≠—Ç–æ –∏–∑–º–µ–Ω–∏—Ç –∏—Å—Ç–æ—Ä–∏—é, –∫–æ—Ç–æ—Ä—É—é –æ–Ω–∏ —É–∂–µ –ø–æ–ª—É—á–∏–ª–∏, –∏ –ø—Ä–∏–≤–µ–¥–µ—Ç –∫ —Å–µ—Ä—å–µ–∑–Ω—ã–º –ø—Ä–æ–±–ª–µ–º–∞–º —Å —Å–∏–Ω—Ö—Ä–æ–Ω–∏–∑–∞—Ü–∏–µ–π.

### –¢–∏–ø–∏—á–Ω—ã–µ –ø—Ä–æ–±–ª–µ–º—ã –∏ –∏—Ö —Ä–µ—à–µ–Ω–∏—è

*   **[–ö–æ–Ω—Ñ–ª–∏–∫—Ç—ã](/git/conflicts)**: –ú–æ–≥—É—Ç –≤–æ–∑–Ω–∏–∫–Ω—É—Ç—å –ø—Ä–∏ –æ–±–æ–∏—Ö –æ–ø–µ—Ä–∞—Ü–∏—è—Ö. –ü—Ä–∏ rebase –∫–æ–Ω—Ñ–ª–∏–∫—Ç –º–æ–∂–µ—Ç –ø–æ—è–≤–∏—Ç—å—Å—è –¥–ª—è –∫–∞–∂–¥–æ–≥–æ –ø–µ—Ä–µ–Ω–æ—Å–∏–º–æ–≥–æ –∫–æ–º–º–∏—Ç–∞, —á—Ç–æ –º–æ–∂–µ—Ç –±—ã—Ç—å —É—Ç–æ–º–∏—Ç–µ–ª—å–Ω–æ.
    *   **–†–µ—à–µ–Ω–∏–µ**: –†–∞–∑—Ä–µ—à–∏—Ç–µ –∫–æ–Ω—Ñ–ª–∏–∫—Ç–Ω—ã–µ —Ñ–∞–π–ª—ã –≤—Ä—É—á–Ω—É—é, `git add .`, –∑–∞—Ç–µ–º `git commit` (–¥–ª—è merge) –∏–ª–∏ `git rebase --continue` (–¥–ª—è rebase). –î–ª—è –æ—Ç–º–µ–Ω—ã rebase: `git rebase --abort`.
*   **–ü–µ—Ä–µ–ø–∏—Å—ã–≤–∞–Ω–∏–µ –æ–±—â–µ–π –∏—Å—Ç–æ—Ä–∏–∏ (`rebase`)**: –ì–ª–∞–≤–Ω–∞—è –æ–ø–∞—Å–Ω–æ—Å—Ç—å rebase. –ï—Å–ª–∏ –≤—ã –ø–µ—Ä–µ–±–∞–∑–∏—Ä—É–µ—Ç–µ –æ–ø—É–±–ª–∏–∫–æ–≤–∞–Ω–Ω—É—é –≤–µ—Ç–∫—É, –¥—Ä—É–≥–∏–º —É—á–∞—Å—Ç–Ω–∏–∫–∞–º –∫–æ–º–∞–Ω–¥—ã –±—É–¥–µ—Ç —Å–ª–æ–∂–Ω–æ —Å–∏–Ω—Ö—Ä–æ–Ω–∏–∑–∏—Ä–æ–≤–∞—Ç—å—Å—è.
    *   **–†–µ—à–µ–Ω–∏–µ**: –°–ª–µ–¥—É–π—Ç–µ –∑–æ–ª–æ—Ç–æ–º—É –ø—Ä–∞–≤–∏–ª—É. –ï—Å–ª–∏ –≤—ã *–¥–µ–π—Å—Ç–≤–∏—Ç–µ–ª—å–Ω–æ* –¥–æ–ª–∂–Ω—ã (–Ω–∞–ø—Ä–∏–º–µ—Ä, –ø–æ—Å–ª–µ squashing –ø—Ä–∏–≤–∞—Ç–Ω—ã—Ö –∫–æ–º–º–∏—Ç–æ–≤) –∏ –≤–µ—Ç–∫–∞ —Ç–æ–ª—å–∫–æ –≤–∞—à–∞, –∏—Å–ø–æ–ª—å–∑—É–π—Ç–µ `git push --force-with-lease` –¥–ª—è –±–µ–∑–æ–ø–∞—Å–Ω–æ–≥–æ –ø—Ä–∏–Ω—É–¥–∏—Ç–µ–ª—å–Ω–æ–≥–æ push.

### –ü—Ä–∞–∫—Ç–∏–∫–∞

1.  –°–æ–∑–¥–∞–π—Ç–µ –Ω–æ–≤—ã–π —Ä–µ–ø–æ–∑–∏—Ç–æ—Ä–∏–π –∏ –∏–Ω–∏—Ü–∏–∞–ª–∏–∑–∏—Ä—É–π—Ç–µ Git.
    ```bash

    cd git-rebase-merge-practice
    git init
    ```
2.  –°–æ–∑–¥–∞–π—Ç–µ –Ω–∞—á–∞–ª—å–Ω—ã–π [–∫–æ–º–º–∏—Ç](/git/init-commit) –Ω–∞ `main`.

    ```bash
    git add .

    ```
3.  **–ü—Ä–∞–∫—Ç–∏–∫–∞ `git merge`**:
    *   –°–æ–∑–¥–∞–π—Ç–µ –≤–µ—Ç–∫—É `my-feature-merge` –∏ —Å–¥–µ–ª–∞–π—Ç–µ 2 –∫–æ–º–º–∏—Ç–∞.

    *   –ü–µ—Ä–µ–∫–ª—é—á–∏—Ç–µ—Å—å –Ω–∞ `main` –∏ —Å–¥–µ–ª–∞–π—Ç–µ 1 –∫–æ–º–º–∏—Ç (—á—Ç–æ–±—ã –≤—ã–∑–≤–∞—Ç—å merge commit).

4.  **–ü—Ä–∞–∫—Ç–∏–∫–∞ `git rebase`**:
    *   –°–æ–∑–¥–∞–π—Ç–µ –≤–µ—Ç–∫—É `my-feature-rebase` –∏–∑ –∞–∫—Ç—É–∞–ª—å–Ω–æ–≥–æ `main`.

    *   –ü–µ—Ä–µ–∫–ª—é—á–∏—Ç–µ—Å—å –Ω–∞ `main` –∏ —Å–¥–µ–ª–∞–π—Ç–µ 1 –∫–æ–º–º–∏—Ç (—á—Ç–æ–±—ã `main` "—É—à–µ–ª" –≤–ø–µ—Ä–µ–¥).

    *   –ü–µ—Ä–µ–∫–ª—é—á–∏—Ç–µ—Å—å –Ω–∞ `my-feature-rebase` –∏ —Å–¥–µ–ª–∞–π—Ç–µ `git rebase main`. –†–∞–∑—Ä–µ—à–∏—Ç–µ –∫–æ–Ω—Ñ–ª–∏–∫—Ç—ã, –µ—Å–ª–∏ –µ—Å—Ç—å.

    *   –ü–µ—Ä–µ–∫–ª—é—á–∏—Ç–µ—Å—å –Ω–∞ `main` –∏ —Å–¥–µ–ª–∞–π—Ç–µ `git merge my-feature-rebase`. –ü—Ä–æ–∞–Ω–∞–ª–∏–∑–∏—Ä—É–π—Ç–µ `git log --graph --oneline`.
