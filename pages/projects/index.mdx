# Проекты

Раздел **Проекты** — это пошаговые гайды по созданию реальных приложений от начала до конца.

## Что внутри

Каждый проект включает:
- **Установка инструментов** — ссылки и инструкции для подготовки окружения
- **Пошаговое создание** — код с подробными объяснениями каждого шага
- **Деплой/запуск** — как опубликовать готовое приложение
- **Playground** — экспериментируй с кодом прямо в браузере

## Список проектов

### 1. HTML Landing Page
Создай свою первую посадочную страницу с нуля. Научишься структурировать контент и деплоить на Netlify.

**Технологии:** HTML5, Netlify  
**Сложность:** Начинающий  
**Время:** 1-2 часа

---

### 2. CSS Portfolio
Построй стильное портфолио с адаптивным дизайном, анимациями и темной темой.

**Технологии:** CSS3, Grid, Flexbox  
**Сложность:** Средний  
**Время:** 2-3 часа

---

### 3. JS Todo App
Создай полноценное Todo приложение с LocalStorage, фильтрами и drag-and-drop.

**Технологии:** JavaScript, DOM API, LocalStorage  
**Сложность:** Средний  
**Время:** 2-3 часа

---

### 4. TS Weather App
Построй погодное приложение с TypeScript и внешним API. Научишься типизации и работе с данными.

**Технологии:** TypeScript, Fetch API, OpenWeatherMap  
**Сложность:** Продвинутый  
**Время:** 3-4 часа

---

### 5. React Blog
Создай современный блог на React с роутингом, state management и деплоем на Vercel.

**Технологии:** React, React Router, Vite, Vercel  
**Сложность:** Продвинутый  
**Время:** 4-5 часов

---

### 6. WordPress Theme
Разработай кастомную тему WordPress с нуля: PHP шаблоны, ACF, Custom Post Types.

**Технологии:** PHP, WordPress, ACF, MySQL  
**Сложность:** Эксперт  
**Время:** 5-6 часов

---

## Как использовать

1. **Выбери проект** по своему уровню
2. **Следуй инструкциям** шаг за шагом
3. **Экспериментируй** с кодом в Playground
4. **Деплой** готовое приложение
5. **Делись** результатом!

## Требования

- Базовое знание HTML/CSS (для первых двух проектов)
- Опыт с JavaScript (для JS/TS/React проектов)
- Локальный сервер (для WordPress проекта)
