# React Компоненты 

![Иллюстрация к уроку](/lessons/components.png)

Компонент = функция, возвращающая JSX.

## Простой компонент

```tsx
function Welcome({ name }: { name: string }) {
  return <h1>Привет, {name}!</h1>;
}

// Использование
<Welcome name="Яша" />
```

## Props

Props — это входные данные компонента.

```tsx
interface CardProps {
  title: string;
  children: React.ReactNode;
}

function Card({ title, children }: CardProps) {
  return (
    <div className="card">
      <h2>{title}</h2>
      {children}
    </div>
  );
}
```
