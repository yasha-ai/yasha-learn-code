## CSS: clip-path и shape-outside. Формируем нестандартные фигуры.

В этом уроке мы познакомимся с двумя мощными CSS-свойствами: `clip-path` и `shape-outside`. Они позволяют создавать нестандартные формы для элементов, открывая новые возможности для дизайна.

### clip-path: Обрезаем элементы

`clip-path` позволяет обрезать элемент, показывая только его часть, заключенную внутри заданной формы. Это как вырезать фигуру из бумаги ножницами.

**Синтаксис:**

```css
clip-path: <clip-source> | <geometry-box> | none | initial | inherit;
```

Наиболее часто используются `<geometry-box>` и `<clip-source>`.

**Примеры:**

1.  **Обрезаем элемент в круг:**

    ```css
    .circle {
      width: 200px;
      height: 200px;
      background-color: lightblue;
      clip-path: circle(50%); /* Круг с радиусом 50% от размера элемента */
    }
    ```

    Этот код обрежет квадратный элемент в круг.

2.  **Обрезаем элемент в многоугольник:**

    ```css
    .polygon {
      width: 200px;
      height: 200px;
      background-color: lightgreen;
      clip-path: polygon(50% 0%, 100% 50%, 50% 100%, 0% 50%); /* Ромб */
    }
    ```

    Здесь мы определяем многоугольник, используя координаты его вершин.

3.  **Обрезаем изображение с помощью SVG:**

    ```html
    <svg width="0" height="0">
      <defs>
        <clipPath id="myClip">
          <ellipse cx="100" cy="50" rx="100" ry="50"/>
        </clipPath>
      </defs>
    </svg>

    <img src="image.jpg" style="clip-path: url(#myClip);" width="200" height="100">
    ```

    В этом примере мы используем SVG для определения фигуры обрезки и применяем ее к изображению.

### shape-outside: Определяем форму для обтекания текста

`shape-outside` позволяет определить форму вокруг элемента, вокруг которой будет обтекать текст. В отличие от `clip-path`, который обрезает сам элемент, `shape-outside` влияет на то, как другие элементы (например, текст) располагаются вокруг него.

**Синтаксис:**

```css
shape-outside: <shape-value> | <box-value> | none | margin-box | border-box | padding-box | content-box;
```

Наиболее часто используется `<shape-value>`.

**Примеры:**

```html
<div style="width: 200px; height: 200px; float: left; shape-outside: circle(50%); margin-right: 20px; background-color: red;"></div>
<p>Этот текст будет обтекать круг, созданный с помощью shape-outside. Этот текст будет обтекать круг, созданный с помощью shape-outside. Этот текст будет обтекать круг, созданный с помощью shape-outside.</p>
```

Этот код создаст круг, вокруг которого будет обтекать текст.

### Жизненный пример

*   **Портфолио дизайнеров:** `clip-path` часто используется для создания интересных форм аватарок или элементов дизайна в портфолио.
*   **Журналы и блоги:** `shape-outside` активно применяется для обтекания текста вокруг изображений, добавляя динамичности и визуальной привлекательности.
*   **Лендинг страницы:** Нестандартные формы для секций и элементов интерфейса, привлекающие внимание пользователей.
*   **Фреймворки и библиотеки:** Некоторые CSS-фреймворки предлагают готовые классы для работы с `clip-path` и `shape-outside`, упрощая их использование.

### Ключевые моменты

*   `clip-path` обрезает элемент, делая видимой только его часть.
*   `shape-outside` определяет форму, вокруг которой обтекает контент.
*   Оба свойства позволяют создавать сложные и интересные дизайны.
*   Можно использовать геометрические фигуры, SVG или изображения для определения формы.
*   `shape-outside` требует, чтобы элемент был плавающим (`float: left` или `float: right`).
