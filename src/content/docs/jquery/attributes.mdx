Работа с HTML атрибутами и DOM свойствами.

## Атрибуты

.attr(), .removeAttr()

```javascript
// Пример кода для Атрибуты
$('.example').text('Hello from Атрибуты');
```

## Свойства

.prop(), .removeProp()

```javascript
// Пример кода для Свойства
$('.example').text('Hello from Свойства');
```

## Значения

.val() для форм

```javascript
// Пример кода для Значения
$('.example').text('Hello from Значения');
```

## Данные

.data() для хранения произвольных данных

```javascript
// Пример кода для Данные
$('.example').text('Hello from Данные');
```

## HTML и текст

.html(), .text()

```javascript
// Пример кода для HTML и текст
$('.example').text('Hello from HTML и текст');
```
