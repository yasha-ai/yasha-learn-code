AJAX позволяет загружать данные без перезагрузки страницы.

## Что такое AJAX

Асинхронные запросы

```javascript
// Пример кода для Что такое AJAX
$('.example').text('Hello from Что такое AJAX');
```

## Зачем нужен AJAX

Динамические веб-приложения

```javascript
// Пример кода для Зачем нужен AJAX
$('.example').text('Hello from Зачем нужен AJAX');
```

## JSON формат

Обмен данными

```javascript
// Пример кода для JSON формат
$('.example').text('Hello from JSON формат');
```

    $('#doPost').click(function() {
      mockRequest('POST', {text: 'Отличная статья!'}, function(sent) {
        var mockResponse = JSON.stringify({
          success: true,
          saved: sent,
          id: Math.floor(Math.random() * 1000)
        }, null, 2);
