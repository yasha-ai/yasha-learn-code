Как отказаться от jQuery и перейти на чистый JavaScript.

## Зачем мигрировать

Производительность, размер

```javascript
// Пример кода для Зачем мигрировать
$('.example').text('Hello from Зачем мигрировать');
```

## Таблица соответствий

jQuery → Vanilla JS

```javascript
// Пример кода для Таблица соответствий
$('.example').text('Hello from Таблица соответствий');
```

## Постепенная миграция

Стратегия перехода

```javascript
// Пример кода для Постепенная миграция
$('.example').text('Hello from Постепенная миграция');
```

## Альтернативы

Micro-библиотеки

```javascript
// Пример кода для Альтернативы
$('.example').text('Hello from Альтернативы');
```
