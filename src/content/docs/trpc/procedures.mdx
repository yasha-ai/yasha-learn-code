Процедуры — это основные строительные блоки tRPC API. Каждая процедура — это функция, которую клиент может вызвать на сервере. Они делятся на два типа: `query` для чтения данных и `mutation` для их изменения.

## publicProcedure

`publicProcedure` — базовая процедура, доступная всем без аутентификации. Её можно создать через `t.procedure`, где `t` — экземпляр инициализированного tRPC. Все остальные процедуры (например, `protectedProcedure`) создаются путём расширения базовой через `.use(middleware)`.

## Процедура query

Query-процедуры используются для получения данных. Они вызываются через GET-запросы (при использовании httpBatchLink) и кешируются React Query. Определяются через `.query(async ({ input, ctx }) => { ... })`.

## Процедура mutation

Mutation-процедуры изменяют данные на сервере. Они вызываются через POST-запросы и не кешируются. Определяются через `.mutation(async ({ input, ctx }) => { ... })`. На клиенте используются с `useMutation()`.

## Вывод типов

Одна из главных возможностей tRPC — автоматический вывод типов. TypeScript анализирует возвращаемое значение процедуры и автоматически типизирует клиентский код. Вам не нужно явно указывать типы ответов.

## Цепочка методов

Процедура строится цепочкой методов: `publicProcedure.input(schema).query(handler)`. Каждый вызов возвращает новую процедуру. Порядок важен: сначала `.input()`, затем `.use()` (middleware), потом `.query()` или `.mutation()`.

const BG = '#0f172a';
const TRPC = '#398ccb';
const CARD = '#1e293b';
const TEXT = '#e2e8f0';
const GREEN = '#22c55e';
const YELLOW = '#eab308';
const ORANGE = '#f97316';

type ProcType = 'query' | 'mutation';

interface Field {
  name: string;
  type: 'string' | 'number' | 'boolean';
}

export default function App() {
  const [procType, setProcType] = useState<ProcType>('query');
  const [procName, setProcName] = useState('getUser');
  const [fields, setFields] = useState<Field[]>([
    { name: 'id', type: 'string' },
  ]);
  const [returnType, setReturnType] = useState('User');
  const [hasMiddleware, setHasMiddleware] = useState(false);

  const addField = () => {
    setFields(f => [...f, { name: 'field' + (f.length + 1), type: 'string' }]);
  };

  const removeField = (i: number) => {
    setFields(f => f.filter((_, idx) => idx !== i));
  };

  const updateField = (i: number, key: keyof Field, value: string) => {
    setFields(f => f.map((field, idx) => idx === i ? { ...field, [key]: value } : field));
  };

  const zodType: Record<string, string> = {
    string: 'z.string()',
    number: 'z.number()',
    boolean: 'z.boolean()',
  };

  const inputLines = fields.map(f => '    ' + f.name + ': ' + zodType[f.type] + ',').join('\n');
  const inputBlock = fields.length > 0
    ? '.input(z.object({\n' + inputLines + '\n  }))\n  '
    : '';

  const middlewareLine = hasMiddleware ? '.use(isAuthed)\n  ' : '';

  const generated =
    'export const ' + procName + ' = publicProcedure\n' +
    '  ' + middlewareLine +
    inputBlock +
    '.' + procType + '(async ({ input, ctx }) => {\n' +
    (fields.length > 0 ? '    // input.' + fields[0].name + ' типизирован!\n' : '') +
    '    return {} as ' + returnType + ';\n' +
    '  });';

  const clientUsage = procType === 'query'
    ? (
      'const { data, isLoading } = trpc\n' +
      '  .' + procName + '.useQuery(' +
      (fields.length > 0 ? '{\n    ' + fields.map(f => f.name + ': /* ' + f.type + ' */').join(',\n    ') + '\n  }' : '{}') +
      ');\n\n' +
      '// data автоматически типизирован как:\n' +
      '// ' + returnType
    )
    : (
      'const mutation = trpc.' + procName + '.useMutation();\n\n' +
      'mutation.mutate(' +
      (fields.length > 0 ? '{\n  ' + fields.map(f => f.name + ': /* ' + f.type + ' */').join(',\n  ') + '\n}' : '{}') +
      ');\n\n' +
      '// returns: ' + returnType
    );

  return (
    <div style={{ background: BG, minHeight: '100vh', padding: '20px', fontFamily: 'monospace', color: TEXT }}>
      <h2 style={{ color: TRPC, textAlign: 'center', marginTop: 0, fontSize: '20px' }}>
        Конструктор процедур tRPC
      </h2>

      <div style={{ display: 'flex', gap: '14px', flexWrap: 'wrap' }}>
        <div style={{ background: CARD, border: '1px solid #334155', borderRadius: '10px', padding: '16px', minWidth: '260px', flex: '0 0 auto' }}>
          <div style={{ color: TRPC, fontWeight: 'bold', fontSize: '13px', marginBottom: '14px' }}>⚙️ Настройки</div>

          <div style={{ marginBottom: '12px' }}>
            <label style={{ color: TEXT, opacity: 0.7, fontSize: '12px', display: 'block', marginBottom: '4px' }}>Тип процедуры</label>
            <div style={{ display: 'flex', gap: '6px' }}>
              {(['query', 'mutation'] as ProcType[]).map(t => (
                <button key={t} onClick={() => setProcType(t)} style={{
                  flex: 1, background: procType === t ? (t === 'query' ? TRPC : YELLOW) : BG,
                  color: procType === t ? '#fff' : TEXT,
                  border: '1px solid ' + (t === 'query' ? TRPC : YELLOW),
                  borderRadius: '6px', padding: '6px', cursor: 'pointer', fontSize: '12px',
                }}>
                  {t === 'query' ? '📖 query' : '✏️ mutation'}
                </button>
              ))}
            </div>
          </div>

          <div style={{ marginBottom: '12px' }}>
            <label style={{ color: TEXT, opacity: 0.7, fontSize: '12px', display: 'block', marginBottom: '4px' }}>Имя процедуры</label>
            <input
              value={procName}
              onChange={e => setProcName(e.target.value)}
              style={{
                width: '100%', background: BG, border: '1px solid #334155', borderRadius: '6px',
                padding: '6px 10px', color: TEXT, fontSize: '12px', boxSizing: 'border-box',
              }}
            />
          </div>

          <div style={{ marginBottom: '12px' }}>
            <label style={{ color: TEXT, opacity: 0.7, fontSize: '12px', display: 'block', marginBottom: '4px' }}>Тип возврата</label>
            <input
              value={returnType}
              onChange={e => setReturnType(e.target.value)}
              style={{
                width: '100%', background: BG, border: '1px solid #334155', borderRadius: '6px',
                padding: '6px 10px', color: GREEN, fontSize: '12px', boxSizing: 'border-box',
              }}
            />
          </div>

          <div style={{ marginBottom: '12px' }}>
            <label style={{ color: TEXT, opacity: 0.7, fontSize: '12px', display: 'block', marginBottom: '6px' }}>
              Входные поля (input)
            </label>
            {fields.map((field, i) => (
              <div key={i} style={{ display: 'flex', gap: '4px', marginBottom: '4px' }}>
                <input
                  value={field.name}
                  onChange={e => updateField(i, 'name', e.target.value)}
                  style={{ flex: 1, background: BG, border: '1px solid #334155', borderRadius: '4px', padding: '4px 6px', color: TEXT, fontSize: '11px' }}
                />
                <select
                  value={field.type}
                  onChange={e => updateField(i, 'type', e.target.value)}
                  style={{ background: BG, border: '1px solid #334155', borderRadius: '4px', padding: '4px', color: YELLOW, fontSize: '11px' }}
                >
                  <option value="string">string</option>
                  <option value="number">number</option>
                  <option value="boolean">boolean</option>
                </select>
                <button onClick={() => removeField(i)} style={{ background: '#ef444420', border: '1px solid #ef4444', borderRadius: '4px', color: '#ef4444', padding: '4px 6px', cursor: 'pointer', fontSize: '11px' }}>✕</button>
              </div>
            ))}
            <button onClick={addField} style={{ width: '100%', background: 'rgba(57,140,203,0.1)', border: '1px dashed ' + TRPC, borderRadius: '6px', color: TRPC, padding: '5px', cursor: 'pointer', fontSize: '11px', marginTop: '4px' }}>
              + Добавить поле
            </button>
          </div>

          <label style={{ display: 'flex', alignItems: 'center', gap: '8px', cursor: 'pointer', fontSize: '12px', color: TEXT }}>
            <input type="checkbox" checked={hasMiddleware} onChange={e => setHasMiddleware(e.target.checked)} />
            Добавить .use(isAuthed)
          </label>
        </div>

        <div style={{ flex: '1', minWidth: '280px', display: 'flex', flexDirection: 'column', gap: '12px' }}>
          <div style={{ background: CARD, border: '1px solid ' + TRPC, borderRadius: '10px', padding: '16px' }}>
            <div style={{ color: TRPC, fontWeight: 'bold', fontSize: '12px', marginBottom: '10px' }}>
              🖥 Сгенерированная процедура (сервер)
            </div>
            <pre style={{ margin: 0, fontSize: '11px', color: TEXT, lineHeight: '1.7', overflowX: 'auto', whiteSpace: 'pre-wrap' }}>
              {generated}
            </pre>
          </div>
          <div style={{ background: CARD, border: '1px solid ' + GREEN, borderRadius: '10px', padding: '16px' }}>
            <div style={{ color: GREEN, fontWeight: 'bold', fontSize: '12px', marginBottom: '10px' }}>
              💻 Использование на клиенте (React)
            </div>
            <pre style={{ margin: 0, fontSize: '11px', color: TEXT, lineHeight: '1.7', overflowX: 'auto', whiteSpace: 'pre-wrap' }}>
              {clientUsage}
            </pre>
          </div>
          <div style={{ background: 'rgba(57,140,203,0.08)', border: '1px solid rgba(57,140,203,0.3)', borderRadius: '8px', padding: '12px' }}>
            <div style={{ color: TRPC, fontSize: '12px', fontWeight: 'bold', marginBottom: '6px' }}>💡 Вывод типов</div>
            <div style={{ color: TEXT, opacity: 0.7, fontSize: '11px', lineHeight: '1.5' }}>
              TypeScript автоматически выводит тип <span style={{ color: GREEN }}>{returnType}</span> из возвращаемого значения процедуры. Не нужно явно указывать тип ответа на клиенте!
            </div>
          </div>
        </div>
      </div>
    </div>
  );
}
` }} />
