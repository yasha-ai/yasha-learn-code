Middleware в tRPC — это функции, которые выполняются между получением запроса и вызовом основного обработчика процедуры. Они используются для логирования, аутентификации, авторизации, ограничения частоты запросов и многого другого.

## Создание middleware

Middleware создаётся через `t.middleware()`. Функция middleware получает объект `{ ctx, next, input, type, path }`. Для продолжения выполнения необходимо вызвать `next()`. Можно также расширить контекст, передав новый объект в `next({ ctx: newCtx })`.

## Middleware аутентификации

Самый распространённый middleware — проверка аутентификации. Он проверяет `ctx.user`, и если пользователь не авторизован, выбрасывает ошибку `TRPCError({ code: 'UNAUTHORIZED' })`. Если авторизован — обогащает контекст гарантированно не-null пользователем.

## Цепочки middleware

Middleware можно выстраивать в цепочки: один middleware вызывает следующий через `next()`. Порядок имеет значение: первым выполняется первый middleware в цепочке. Это позволяет строить сложные pipelines обработки запросов.

## Логирование

Logging middleware измеряет время выполнения процедуры: сохраняет время начала до вызова `next()` и логирует разницу после. Это мощный способ мониторинга производительности без изменения бизнес-логики.

## Rate limiting

Middleware ограничения частоты запросов проверяет количество вызовов от конкретного пользователя или IP за временной интервал. При превышении лимита выбрасывается ошибка `TOO_MANY_REQUESTS`.

interface LogEntry {
  id: number;
  middlewareId: string;
  status: 'running' | 'passed' | 'failed' | 'skipped';
  message: string;
  time?: number;
}

let logIdCounter = 0;

export default function App() {
  const [middlewares, setMiddlewares] = useState<Middleware[]>(defaultMiddlewares);
  const [isAuthed, setIsAuthed] = useState(false);
  const [isRunning, setIsRunning] = useState(false);
  const [logs, setLogs] = useState<LogEntry[]>([]);
  const [selectedMw, setSelectedMw] = useState<Middleware | null>(null);
  const [activeIndex, setActiveIndex] = useState(-1);

  const toggleMiddleware = (id: string) => {
    setMiddlewares(mws => mws.map(mw => mw.id === id ? { ...mw, enabled: !mw.enabled } : mw));
  };

  const runPipeline = async () => {
    if (isRunning) return;
    setIsRunning(true);
    setLogs([]);
    setActiveIndex(-1);

    const enabled = middlewares.filter(mw => mw.enabled);

    for (let i = 0; i < enabled.length; i++) {
      const mw = enabled[i];
      setActiveIndex(i);

      await new Promise(resolve => setTimeout(resolve, 600));

      const startTime = Date.now();
      let passed = true;
      let msg = '';

      if (mw.id === 'auth' && !isAuthed && mw.canFail) {
        passed = false;
        msg = mw.failMessage || 'Error';
      }

      await new Promise(resolve => setTimeout(resolve, 400));
      const elapsed = Date.now() - startTime;

      setLogs(l => [...l, {
        id: ++logIdCounter,
        middlewareId: mw.id,
        status: passed ? 'passed' : 'failed',
        message: passed ? mw.name + ' — пройден (' + elapsed + 'ms)' : mw.name + ' — ' + msg,
        time: elapsed,
      }]);

      if (!passed) {
        setActiveIndex(-1);
        setIsRunning(false);
        return;
      }
    }

    await new Promise(resolve => setTimeout(resolve, 400));
    setLogs(l => [...l, {
      id: ++logIdCounter,
      middlewareId: 'procedure',
      status: 'passed',
      message: '🎯 Процедура выполнена успешно',
    }]);
    setActiveIndex(-1);
    setIsRunning(false);
  };

            {middlewares.map((mw, i) => {
              const enabledList = middlewares.filter(m => m.enabled);
              const pos = enabledList.findIndex(m => m.id === mw.id);
              const isActive = pos === activeIndex;
              return (
                <div key={mw.id}
                  onClick={() => setSelectedMw(mw === selectedMw ? null : mw)}
                  style={{
                    background: isActive ? 'rgba(57,140,203,0.2)' : BG,
                    border: '1px solid ' + (selectedMw?.id === mw.id ? mw.color : (mw.enabled ? mw.color + '44' : '#334155')),
                    borderRadius: '8px', padding: '8px 12px', marginBottom: '6px', cursor: 'pointer',
                    opacity: mw.enabled ? 1 : 0.5, transition: 'all 0.2s',
                  }}
                >
                  <div style={{ display: 'flex', justifyContent: 'space-between', alignItems: 'center' }}>
                    <div style={{ color: mw.enabled ? mw.color : '#475569', fontSize: '12px', fontWeight: 'bold' }}>
                      {mw.icon} {mw.name}
                    </div>
                    <button onClick={e => { e.stopPropagation(); toggleMiddleware(mw.id); }} style={{
                      background: mw.enabled ? mw.color + '33' : '#334155', color: mw.enabled ? mw.color : '#475569',
                      border: '1px solid ' + (mw.enabled ? mw.color : '#475569'), borderRadius: '4px',

                    }}>
                      {mw.enabled ? 'Вкл' : 'Выкл'}
                    </button>
                  </div>
                  <div style={{ color: TEXT, opacity: 0.5, fontSize: '10px', marginTop: '2px' }}>{mw.description}</div>
                </div>
              );
            })}
          </div>

          <div style={{ background: CARD, border: '1px solid #334155', borderRadius: '8px', padding: '12px', marginBottom: '10px' }}>
            <label style={{ display: 'flex', alignItems: 'center', gap: '8px', cursor: 'pointer', fontSize: '12px' }}>
              <input type="checkbox" checked={isAuthed} onChange={e => setIsAuthed(e.target.checked)} />
              <span style={{ color: TEXT }}>Авторизованный запрос</span>
            </label>
          </div>

          </button>
        </div>

        <div style={{ flex: '1', minWidth: '250px', display: 'flex', flexDirection: 'column', gap: '10px' }}>

        </div>
