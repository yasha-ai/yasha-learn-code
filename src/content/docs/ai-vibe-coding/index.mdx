**Vibe Coding** — это подход к разработке, где ты описываешь что хочешь получить, а AI генерирует код. Ты задаёшь направление, AI делает работу.

В 2026 году это не эксперимент — это реальный рабочий процесс тысяч разработчиков.

## Что изучим в курсе

### Платформы для старта
- **Replit** — облачная среда с AI-агентом, деплой в один клик
- **Lovable** — React + Supabase из промпта, идеально для SaaS

### AI-редакторы
- **Cursor** — VS Code с агентами и собственной моделью Composer
- **Google AntiGravity** — multi-agent IDE с mission control и встроенным браузером

### Специализированные инструменты
- **OpenAI Codex CLI** — агент в терминале
- **Claude Code** — CLI с MCP серверами и skills

### Генерация медиа
- **Изображения** — ChatGPT Image, Gemini, Nano Banana Pro
- **Звук и голос** — Suno AI, ElevenLabs

### Публикация и монетизация
- **Деплой** — Vercel, Railway, Dokploy
- **Контент с AI** — автоматизация постинга, SEO

### Мобильные и Desktop приложения
- **React Native + AI** — мобилки без нативного опыта
- **Tauri/Electron + AI** — Desktop из веб-технологий

## Для кого курс

- Разработчики, которые хотят работать быстрее
- Дизайнеры и продакты, которые хотят прототипировать без помощи разработчиков
- Предприниматели, которые хотят запустить продукт с минимальными затратами

## Что нужно для старта

- Базовое понимание HTML/CSS/JavaScript (достаточно знать что это такое)
- Аккаунт GitHub
- Кредитная карта (для некоторых платных инструментов, но начать можно бесплатно)
