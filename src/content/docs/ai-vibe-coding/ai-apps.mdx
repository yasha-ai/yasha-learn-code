## Экосистема AI-приложений в 2026

Время, когда "AI" означало только вкладку в браузере, прошло. Сейчас у каждой крупной лаборатории есть нативные десктопные приложения — со своей уникальной механикой, суперсилами и косяками. И разработчик, который знает, когда и что использовать, работает в несколько раз быстрее тех, кто сидит только на одном инструменте.

В этом уроке — полный разбор экосистемы: от Claude Desktop до Grok.

## Claude Desktop

Claude Desktop — это не просто Anthropic в нативном окне. Это **платформа** с доступом к локальным файлам, базам данных и инструментам через MCP.

### Установка

<Steps>
1. Скачай с [claude.ai/download](https://claude.ai/download) для macOS или Windows

2. Войди в свой Anthropic аккаунт

3. Готово — у тебя уже есть доступ к Claude Sonnet 4 / Opus 4 в нативном приложении
</Steps>

### Ключевые фичи

**Расширенные проекты (Projects)**

Создаёшь проект → загружаешь документы, кодовую базу, заметки → Claude помнит всё это в рамках проекта. Идеально для: работы над одним продуктом долго, без повторного вбрасывания контекста.

**Работа с файлами**

Перетаскиваешь PDF, изображения, таблицы — Claude их читает и анализирует прямо в диалоге. Никаких ограничений на "загрузи файл через форму".

**Нативный интерфейс**

Горячие клавиши, быстрый вызов (Cmd+Shift+Space / Ctrl+Shift+Space), поиск по истории диалогов. Живёт в меню-баре, не засоряет рабочий стол.

## MCP — почему это меняет всё

**Model Context Protocol (MCP)** — открытый стандарт, придуманный Anthropic в конце 2024 года. По сути, это USB-порт для AI-приложений: единый способ подключить внешние инструменты к языковой модели.

### Как это работает

```
Claude Desktop
    ↓ MCP Client
MCP Server (твой инструмент)
    ↓
Реальные данные: файлы, БД, API, браузер...
```

До MCP каждый AI-продукт изобретал свои велосипеды для тулзов. Теперь — один сервер, работающий с любым MCP-совместимым клиентом.

### Что подключают через MCP

| MCP-сервер | Что даёт |
|------------|----------|
| `filesystem` | Чтение/запись локальных файлов |
| `sqlite` / `postgres` | Прямые запросы к БД |
| `github` | Issues, PR, коммиты прямо в чате |
| `puppeteer` | Браузерная автоматизация |
| `slack` / `notion` | Рабочие инструменты |
| `memory` | Персистентная память между сессиями |

### Настройка MCP в Claude Desktop

```json
// ~/Library/Application Support/Claude/claude_desktop_config.json
{
  "mcpServers": {
    "filesystem": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/Users/me/projects"]
    },
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_PERSONAL_ACCESS_TOKEN": "ghp_..."
      }
    },
    "postgres": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-postgres", "postgresql://localhost/mydb"]
    }
  }
}
```

После перезапуска приложения — модель видит все подключённые инструменты. Говоришь: _"Посмотри на файлы в /projects/my-app и найди где мы обрабатываем авторизацию"_ — и Клод реально читает твои файлы.

<Aside type="tip">
Реестр готовых MCP-серверов: [modelcontextprotocol.io/servers](https://modelcontextprotocol.io/servers). Там сотни серверов от сообщества — для Stripe, Jira, Home Assistant, Linear и много другого.
</Aside>

## ChatGPT Desktop

ChatGPT Desktop вышел в 2024 году и быстро набрал аудиторию за счёт трёх убийственных фич: **голосовой режим**, **работа с экраном** и **memory**.

### Установка

Скачай с [openai.com/chatgpt/download](https://openai.com/chatgpt/download). Нужен ChatGPT Plus ($20/мес) для полного доступа к GPT-4o и голосовому режиму.

### Advanced Voice Mode

Это не просто TTS. ChatGPT слышит тебя в реальном времени, отвечает голосом с паузами и интонацией, понимает прерывания. Можно вести полноценный разговор, как с живым человеком:

- Объяснять архитектурные решения "на пальцах" и получать фидбек голосом
- Делать code review в формате диалога
- Готовиться к техническому интервью в режиме реального разговора

### Screen Sharing

ChatGPT видит твой экран и комментирует происходящее. Сценарии для разработчика:

- "Посмотри на этот стектрейс, что не так?" — и ты просто показываешь экран
- Дебаг в реальном времени: показываешь браузер, объясняешь проблему голосом
- Разбор чужого кода с вопросами по ходу

### Memory

ChatGPT запоминает факты между сессиями: твой стек, предпочтения, проекты. Управляй памятью: Settings → Personalization → Memory.

```
Помнит: "Пользователь работает с Next.js 15, 
предпочитает TypeScript, использует Prisma ORM"
```

<Aside type="note">
GPT-4o в голосовом режиме иногда "галлюцинирует" API-детали. Для технических задач с кодом — сначала текст, голос для обсуждения.
</Aside>

## Gemini (Google)

Gemini — главный козырь Google. Не потому что лучшая модель (хотя 2.5 Pro действительно крут), а потому что **глубокая интеграция с Google Workspace**.

### Gemini Desktop App

Скачай с [gemini.google.com](https://gemini.google.com) — нативное приложение для macOS и Windows (с Gemini Advanced).

### Google Workspace интеграция

Если ты юзаешь Gmail, Google Docs, Drive, Calendar — Gemini превращается в суперсилу:

```
Gemini + Gmail:
"Найди все письма от клиента X за последний месяц
и сделай саммари обсуждений про проект Y"

Gemini + Google Docs:
"Открой документ 'Архитектура' и добавь раздел 
про API endpoints на основе нашего чата"

Gemini + Drive:
"Проанализируй все PDF в папке /specs и 
выдели противоречия в требованиях"
```

### Deep Research

Убийственная фича для ресёрча. Gemini запускает многошаговое исследование: читает статьи, сравнивает источники, строит структурированный отчёт с цитатами.

**Для разработчика:** сравнить технологии перед выбором стека, изучить новый фреймворк, разобраться в сложной теме за 15 минут.

### Gemini 2.5 Pro — контекстное окно

2 миллиона токенов. Это значит — вставить всю кодовую базу среднего проекта и спросить _что угодно_. Это реально меняет как работаешь с большими репами.

## Perplexity Desktop

Perplexity — гибрид поисковика и AI. Не просто отвечает на вопросы, а **ищет актуальную информацию в реальном времени** и показывает источники.

### Зачем разработчику

- **Актуальная документация**: "Как настроить Next.js 15 с App Router и Prisma?" — Perplexity найдёт свежие статьи, не ответит устаревшим контентом из тренировочных данных
- **Баги и решения**: "Error: Cannot read properties of undefined (reading 'map') в React 19" — ищет в GitHub Issues, StackOverflow, Reddit прямо сейчас
- **Новости экосистемы**: что нового в TypeScript 6, что релизнул Vercel — без риска получить устаревший ответ

### Collections

Создай коллекцию → добавь в неё ссылки, документы, заметки → задавай вопросы по этому контексту. По сути, персональная RAG-система без единой строчки кода.

```
Коллекция "Мой проект":
- README.md проекта
- Ссылки на docs используемых библиотек
- Заметки с ретро
→ Задаёшь вопросы, Perplexity отвечает в контексте
```

## Copilot (Microsoft)

Microsoft Copilot — это их ставка на AI везде: в Windows, Office 365, браузере Edge.

### Copilot в Windows 11

Встроен прямо в систему (Win+C или кнопка на таскбаре). Умеет:
- Управлять настройками системы голосом: "Включи тёмный режим", "Подключись к VPN"
- Отвечать на вопросы с учётом контекста экрана
- Работать с файлами через Проводник

### Copilot в Edge

Боковая панель с AI прямо в браузере:
- Суммаризация страницы одним кликом
- Генерация email по контексту открытой страницы
- Перевод + объяснение технических статей

### Для разработчика

Если ты на Windows и в экосистеме Microsoft (Azure, VS Code, Teams) — Copilot сэкономит время через глубокую интеграцию. Но если ты не в этой экосистеме — смотри на Claude и ChatGPT.

<Aside type="note">
Copilot в VS Code — это отдельная история и отдельный урок. Здесь речь о системном Copilot.
</Aside>

## Grok Desktop / xAI

Grok от Elon Musk's xAI — это AI с доступом к реальному времени Twitter/X и весьма нестандартным подходом к цензуре (его почти нет).

### Что умеет

- **Real-time X данные**: тренды, твиты, настроения рынка прямо сейчас
- **Расширенный режим "Fun"**: без корпоративного фильтра, отвечает прямо
- **Grok-3**: мощная модель, часто бьёт GPT-4o на бенчах рассуждений
- **Aurora**: генерация изображений, встроенная в Grok

### Когда использовать

- Анализ трендов и настроений в X/Twitter
- Когда нужен AI без политкорректных ограничений
- Crypto/финансовый ресёрч с учётом свежих новостей
- Технические задачи, где нужна модель с сильным reasoning

## MCP в деталях: меняем правила игры

MCP — это не просто "фича Claude Desktop". Это новый стандарт для всей AI-индустрии. Разберём глубже.

### Архитектура MCP

```
┌─────────────────────────────────────────┐
│           MCP-совместимый клиент         │
│  (Claude Desktop, Cursor, OpenClaw...)  │
└──────────────────┬──────────────────────┘
                   │ MCP Protocol (JSON-RPC)
        ┌──────────┴──────────┐
        │                     │
   MCP Server #1         MCP Server #2
   (filesystem)          (postgres)
        │                     │
   ~/projects/          база данных
```

### Создать свой MCP-сервер

```typescript
// my-mcp-server.ts
import { Server } from "@modelcontextprotocol/sdk/server/index.js";
import { StdioServerTransport } from "@modelcontextprotocol/sdk/server/stdio.js";

const server = new Server({ name: "my-tools", version: "1.0.0" });

// Объявляем инструмент
server.setRequestHandler("tools/list", async () => ({
  tools: [{
    name: "get_user_stats",
    description: "Возвращает статистику пользователя из БД",
    inputSchema: {
      type: "object",
      properties: { userId: { type: "string" } }
    }
  }]
}));

// Реализуем логику
server.setRequestHandler("tools/call", async (request) => {
  if (request.params.name === "get_user_stats") {
    const stats = await db.getUserStats(request.params.arguments.userId);
    return { content: [{ type: "text", text: JSON.stringify(stats) }] };
  }
});

// Запускаем
const transport = new StdioServerTransport();
await server.connect(transport);
```

Добавляешь в конфиг Claude Desktop — и теперь Claude умеет запрашивать твою БД напрямую, прямо из разговора.

<Aside type="tip">
**MCP — не только Claude.** Cursor, Windsurf, OpenClaw, Cline — все крупные AI-инструменты уже добавили поддержку MCP. Написал один сервер — работает везде.
</Aside>

## Выводы: твой арсенал в 2026

Не надо выбирать один инструмент. Выигрывает тот, кто знает, когда что брать:

1. **Claude Desktop** — дефолтный выбор для кодинга. Особенно с MCP и локальными файлами.
2. **ChatGPT Desktop** — когда нужен голос или работа с экраном. Объяснения на ходу.
3. **Gemini** — Google Workspace + Deep Research. Лучший ресёрч-инструмент.
4. **Perplexity** — актуальная информация. Всегда когда важна свежесть данных.
5. **Copilot** — если ты в экосистеме Microsoft. Иначе — пропускай.
6. **Grok** — X/Twitter контекст и нестандартные задачи.

Реальный стек разработчика в 2026: **Claude + Perplexity + ChatGPT голос**. Всё остальное — по ситуации.
