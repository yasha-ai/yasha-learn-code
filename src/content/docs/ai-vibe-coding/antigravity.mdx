## Что такое Google AntiGravity?

**Google AntiGravity** — агентная IDE от Google: форк VS Code, заточенный под multi-agent разработку. Вместо одного AI-помощника — несколько специализированных агентов, которые работают параллельно в редакторе, терминале и встроенном браузере.

Запущен в 2026 году. По сути — ответ Google на Cursor и Windsurf, но с уклоном в mission control и аудируемые агентные workflows.

## Установка

1. Скачай с [antigravity.google](https://antigravity.google) (Mac, Windows, Linux)
2. Установи как обычное приложение (VS Code форк — привычный интерфейс)
3. Дай разрешение на сетевой доступ для встроенного браузера
4. Войди через Google аккаунт (в превью — личный Gmail)

## Два главных интерфейса

### Manager — Mission Control

Центр управления агентами:

- Создай **миссию** (высокоуровневая задача)
- Выбери модель для каждого агента
- Смотри артефакты в реальном времени: планы, логи, скриншоты
- Перераспредели задачи прямо в процессе

### Editor — привычный VS Code

- Inline chat для быстрых правок
- Diff-вью изменений от агентов
- Знакомые шорткаты и расширения

## Как работает multi-agent workflow

Классический пример: ты говоришь «Сделай лендинг с A/B тестом для заголовка».

AntiGravity разбивает задачу:

1. **Planning agent** — составляет план: компоненты, роуты, структура данных
2. **Editor agent** — пишет код в файлах
3. **Terminal agent** — запускает тесты, устанавливает зависимости
4. **Browser agent** — открывает страницу, проверяет UI, делает скриншот

Всё параллельно. Артефакты каждого агента видны в Manager.

## Артефакты — что это

Каждый агент оставляет **артефакты**:
- Список задач и планов
- Скриншоты браузера
- Записи терминальных команд
- Результаты тестов

Это не просто логи — это аудит-трейл. Ты можешь откатиться, понять почему агент принял решение, скорректировать направление.

## Встроенный браузер

Headless-but-visible Chromium прямо в IDE:

- E2E тесты без внешних инструментов
- Скрапинг данных для контекста
- Рендер UI изменений в реальном времени
- Скриншоты как артефакты миссии

Полезно для сложных задач: «Отрефактори авторизацию, напиши доки и проверь checkout flow» — всё в одной миссии.

## Доступные модели

| Модель | Для чего |
|--------|----------|
| Gemini 3 Pro | Рассуждения, планирование, большой контекст |
| Claude Sonnet 4.5 | Длинные тексты, рефакторинг |
| GPT-OSS | Лёгкие правки, быстрые задачи |

Можно назначить разные модели разным агентам внутри одной миссии.

## Цена 2026

- **$0/мес** — бесплатный preview с rotating rate limits
- Лимиты сбрасываются каждые ~5 часов
- Платные тиры ожидаются когда превью стабилизируется

Для тяжёлых сессий (долгие тесты, браузерные задачи) = больше токенов. Следи за roadmap перед бюджетированием.

## Практические примеры промптов

### Создать фичу с проверкой

```
Создай страницу профиля пользователя.
Требования:
- Next.js 15 App Router
- Форма редактирования (имя, email, аватар)
- Сохранение через Server Action
- Оптимистичное обновление UI
- Тесты на основные сценарии
Проверь в браузере что форма работает.
```

### Рефакторинг с аудитом

```
Отрефактори модуль /lib/auth:
- Раздели на мелкие файлы по ответственности
- Добавь типы
- Не меняй публичный API
Покажи план до начала работы.
```

### Полная фича с CI

```
Добавь dark mode в дашборд:
- Переключатель в хедере
- Сохранение в localStorage
- CSS переменные для всех цветов
- Тесты на переключение
Запусти тесты и проверь что ничего не сломалось.
```

## Настройка guardrails

При старте миссии можно задать ограничения:

```
Разрешённый путь: /src/components/auth
Запрещённые команды: rm -rf, git push
Браузер: только localhost:3000
```

Критично для репозиториев с чувствительным кодом — агенты работают в рамках, которые ты задал.

## AntiGravity vs Cursor — что выбрать?

| Критерий | Cursor | AntiGravity |
|----------|--------|------------|
| Зрелость | Более зрелый (2+ года) | Новый (2026) |
| Agentность | Хорошо | Нативно, mission control |
| Встроенный браузер | Есть | Более мощный |
| Аудит действий | Ограниченно | Артефакты и логи |
| Цена | $20+/мес | Бесплатно (preview) |
| Модели | Больше выбора | Gemini/Claude/GPT-OSS |
| VS Code совместимость | Полная | Полная (форк) |

**Вывод:** Cursor — боевая лошадка с богатым функционалом и большой базой пользователей. AntiGravity — интересен если нужна прозрачность агентных решений и встроенный браузер. Попробуй оба — оба бесплатны для старта.

## Связка AI Studio + AntiGravity

Мощный workflow:

1. **Google AI Studio** — быстро прототипируй идею, проверяй промпты, работай с Gemini напрямую
2. **AntiGravity** — берёшь готовый промпт/прототип и запускаешь полноценную разработку с агентами

До интеграции: копировал промпт → открывал AntiGravity → настраивал проект. Теперь можно делать прямо из AI Studio.

## Советы по работе

**Начни с малых миссий:** Понятная, конкретная задача → агенты не уходят в сторону.

**Пиши acceptance criteria:** "Тест должен пройти", "страница грузится за <2с", "нет TypeScript ошибок". Агент будет стремиться к конкретному результату.

**Смотри артефакты:** Не просто принимай изменения — изучи план агента. Если он пошёл не туда — поправь на раннем этапе.

**Запускай в sandbox:** Мощные агенты = агрессивные команды. Для sensitive репозиториев — VM или контейнер.
