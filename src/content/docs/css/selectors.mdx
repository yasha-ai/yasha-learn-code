Селекторы — это способ «нацелиться» на HTML-элементы, чтобы применить к ним стили.

## Интерактивная практика

Попробуй изменить стили!
1. Поменяй цвет для всех `p` на `blue`.
2. Измени фон у `.highlight` на `lightgreen`.
3. Попробуй перекрасить заголовок `#main-title`.
👇

<Playground client:visible
  template="static"
  files={{
    "/index.html": `<!DOCTYPE html>
<html>
  <head>
    <style>
      /* 1. Селектор по Тегу (самый слабый) */
      p {
        color: #333;
        font-family: sans-serif;
      }

      /* 2. Селектор по Классу (сильнее тега) */
      .highlight {
        background: yellow;
        padding: 5px;
        border-radius: 4px;
        font-weight: bold;
      }

      /* 3. Селектор по ID (самый сильный) */
      #main-title {
        color: crimson;
        text-decoration: underline;
      }

      /* 4. Селектор Атрибута */
      a[target="_blank"] {
        color: green;
      }

      /* 5. Псевдо-класс (наведение) */
      button:hover {
        background: #0056b3;
        cursor: pointer;
      }

      /* Базовые стили для красоты */
      body { padding: 20px; }
      button { 
        background: #007bff; 
        color: white; 
        border: none; 
        padding: 10px 20px; 
        border-radius: 5px;
        font-size: 16px;
        transition: 0.3s;
      }
    </style>
  </head>
  <body>
    
    <h1 id="main-title">CSS Селекторы 🎯</h1>
    
    <p>Это обычный параграф текста.</p>
    
    <p class="highlight">А это параграф с классом .highlight!</p>
    
    <p>
      Ссылка: <a href="https://google.com" target="_blank">Google (откроется в новом окне)</a>
    </p>

    <button>Наведи на меня!</button>

  </body>
</html>`
  }}
