Каждый элемент в CSS — это коробка (Box). Понимание того, как работают отступы (margin/padding) — ключ к верстке.

## Интерактивная коробка 📦

Попробуй изменить значения!
1. Увеличь `padding` до `50px` (внутренний отступ).
2. Добавь `margin: 20px` (внешний отступ).
3. Измени толщину границы `border`.
👇
