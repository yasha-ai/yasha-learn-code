Сравнение популярных ORM/ODM.

## Feature Comparison

| Feature | TypeORM | Mongoose | Prisma |
|---|---|---|---|
| DB | SQL | MongoDB | SQL + MongoDB |
| Type Safety | ✅ | ⚠️ Partial | ✅✅ Best |
| Migrations | ✅ | ❌ | ✅ |
| Relations | ✅ | ❌ | ✅ |
| Active Record | ✅ | ✅ | ❌ |
| Query Builder | ✅ | ⚠️ | ✅ |

## TypeORM (SQL)

```typescript
@Entity()
class User {
  @PrimaryGeneratedColumn()
  id: number;
  
  @Column()
  email: string;
  
  @OneToMany(() => Post, post => post.author)
  posts: Post[];
}

const users = await userRepository.find({ where: { email: Like('%@example.com') } });
```

## Mongoose (MongoDB)

```typescript
const userSchema = new Schema({
  email: { type: String, required: true },
  name: String
});

const User = model('User', userSchema);

const users = await User.find({ email: /@example\.com/ });
```

## Prisma (Universal)

```typescript
const users = await prisma.user.findMany({
  where: { email: { contains: '@example.com' } },
  include: { posts: true }
});
```

**Рекомендация:** Prisma для новых проектов (лучший DX).

// Active Record vs Data Mapper vs Query Builder

const db = [
  { id: 1, name: "Алиса", email: "alice@mail.ru" },
  { id: 2, name: "Борис", email: "boris@mail.ru" },
];

// 1. Active Record (модель = запрос к БД)
class User {
  constructor(data) { Object.assign(this, data); }
  save() { console.log("AR: Сохранён", this.name); }
  static find(id) { return new User(db.find(u => u.id === id)); }
  static where(fn) { return db.filter(fn).map(d => new User(d)); }
}

const alice = User.find(1);
alice.name = "Алиса Обновлённая";
alice.save();

// 2. Query Builder (цепочка методов)
class QueryBuilder {
  constructor(data) { this.data = data; this.result = data; }
  select(...fields) { this.fields = fields; return this; }
  where(fn) { this.result = this.result.filter(fn); return this; }
  orderBy(key) { this.result.sort((a, b) => a[key] > b[key] ? 1 : -1); return this; }
  execute() {
    if (this.fields) return this.result.map(r =>
      Object.fromEntries(this.fields.map(f => [f, r[f]])));
    return this.result;
  }
}

const qb = new QueryBuilder(db)
  .select("name", "email")
  .where(u => u.id > 0)
  .orderBy("name")
  .execute();
console.log("\\nQuery Builder:", qb);

console.log("\\nActive Record: простой, модель = БД");
console.log("Data Mapper: гибкий, разделяет модель и БД");
console.log("Query Builder: контроль над SQL");
