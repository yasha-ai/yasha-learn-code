EXPLAIN показывает план выполнения запроса.

## Базовое использование

```sql
EXPLAIN SELECT * FROM users WHERE email = 'john@example.com';
```

## EXPLAIN ANALYZE (реальное выполнение)

```sql
EXPLAIN ANALYZE
SELECT u.username, COUNT(p.id) as post_count
FROM users u
LEFT JOIN posts p ON u.id = p.user_id
GROUP BY u.id, u.username;
```

## Ключевые понятия

- **Seq Scan:** Полное сканирование таблицы (медленно!)
- **Index Scan:** Использует индекс (быстро)
- **Nested Loop:** JOIN через вложенный цикл
- **Hash Join:** JOIN через хеш-таблицу
- **Cost:** Оценка стоимости (относительная)
- **Rows:** Ожидаемое количество строк

## Пример анализа

```sql
EXPLAIN ANALYZE
SELECT * FROM orders WHERE status = 'pending';

-- Результат:
Seq Scan on orders  (cost=0.00..1523.00 rows=150 width=128) (actual time=0.045..12.389 rows=142 loops=1)
  Filter: (status = 'pending'::text)
  Rows Removed by Filter: 9858
Planning Time: 0.123 ms
Execution Time: 12.456 ms
```

**Проблема:** Seq Scan! Нужен индекс.

```sql
CREATE INDEX idx_orders_status ON orders(status);

-- Теперь:
Index Scan using idx_orders_status on orders  (cost=0.29..8.31 rows=142 width=128) (actual time=0.021..0.089 rows=142 loops=1)
```

// EXPLAIN показывает план выполнения запроса

const data = Array.from({ length: 1000 }, (_, i) => ({
  id: i, name: "User " + i, age: 18 + (i % 50), city: ["Москва", "Питер", "Казань"][i % 3]
}));

function explain(queryName, queryFn) {
  const plan = { query: queryName, steps: [] };
  const start = performance.now();
  const result = queryFn(plan);
  const time = (performance.now() - start).toFixed(3);
  plan.executionTime = time + "ms";
  plan.rowsReturned = result.length;
  console.log("EXPLAIN ANALYZE:", queryName);
  plan.steps.forEach(s => console.log("  ->", s));
  console.log("  Время:", time + "ms, строк:", result.length);
  return result;
}

// Seq Scan (полное сканирование)
explain("SELECT * WHERE age > 40", (plan) => {
  plan.steps.push("Seq Scan on data (rows=1000)");
  plan.steps.push("Filter: age > 40");
  return data.filter(d => d.age > 40);
});

// Index Scan (с индексом)
const idx = new Map();
data.forEach(d => { const k = d.city; (idx.get(k) || (idx.set(k, []), idx.get(k))).push(d); });

explain("\\nSELECT * WHERE city = Москва", (plan) => {
  plan.steps.push("Index Scan using city_idx");
  return idx.get("Москва");
});
