Безопасная эволюция схемы базы данных.

## Типы миграций

### 1. Up/Down migrations

```sql
-- migrations/001_create_users.up.sql
CREATE TABLE users (
    id SERIAL PRIMARY KEY,
    email VARCHAR(200) UNIQUE NOT NULL
);

-- migrations/001_create_users.down.sql
DROP TABLE users;
```

### 2. Versioned migrations

```bash
npx prisma migrate dev --name add_users
```

## Правила безопасных миграций

✅ **Безопасные:**
- ADD COLUMN (с DEFAULT)
- CREATE INDEX CONCURRENTLY
- ADD CONSTRAINT (NOT VALID)

❌ **Опасные:**
- DROP COLUMN
- ALTER COLUMN TYPE
- ADD COLUMN NOT NULL (без DEFAULT)

## Пример: добавление NOT NULL

```sql
-- ❌ Опасно (блокирует таблицу!)
ALTER TABLE users ADD COLUMN name VARCHAR(100) NOT NULL;

-- ✅ Безопасно (поэтапно)
-- 1. Добавить nullable
ALTER TABLE users ADD COLUMN name VARCHAR(100);

-- 2. Заполнить данными
UPDATE users SET name = 'Unknown' WHERE name IS NULL;

-- 3. Добавить constraint
ALTER TABLE users ALTER COLUMN name SET NOT NULL;
```

  template="vanilla"
  files={{
    "/index.js": `// JavaScript-эквивалент миграций схемы базы данных

const migrations = [];
let schema = {};

function createMigration(name, up, down) {
  migrations.push({ name, up, down, applied: false });
}

// Миграция 1: Создать таблицу users
createMigration("001_create_users",
  () => { schema.users = { columns: ["id", "name", "email"] }; },
  () => { delete schema.users; }
);

// Миграция 2: Добавить поле age
createMigration("002_add_age",
  () => { schema.users.columns.push("age"); },
  () => { schema.users.columns = schema.users.columns.filter(c => c !== "age"); }
);

// Миграция 3: Создать таблицу posts
createMigration("003_create_posts",
  () => { schema.posts = { columns: ["id", "userId", "title", "body"] }; },
  () => { delete schema.posts; }
);

// Применяем все миграции (migrate up)
console.log("--- Migrate UP ---");
migrations.forEach(m => { m.up(); m.applied = true; console.log("Applied:", m.name); });
console.log("Схема:", JSON.stringify(schema, null, 2));

// Откат последней миграции (migrate down)
const last = [...migrations].reverse().find(m => m.applied);
last.down(); last.applied = false;
console.log("\\n--- Rollback ---");
console.log("Rolled back:", last.name);
console.log("Схема:", JSON.stringify(schema, null, 2));
`
