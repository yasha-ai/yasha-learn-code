Webpack — это **статический сборщик модулей** для JavaScript-приложений. В отличие от старых инструментов вроде Grunt и Gulp, которые выполняли задачи последовательно, Webpack строит **граф зависимостей** вашего приложения и оптимально упаковывает всё в бандлы.

## Историческая справка

До Webpack JavaScript-разработчики сталкивались с серьёзными проблемами:
- Управление порядком `<script>` тегов в HTML вручную
- Глобальные переменные и конфликты имён
- Невозможность использовать `require`/`import` в браузере
- Ручная конкатенация файлов для оптимизации

Webpack решил все эти проблемы, введя концепцию **модульной системы** для браузера.

## Граф зависимостей (Dependency Graph)

Когда Webpack обрабатывает ваш проект, он начинает с **точки входа (entry point)** и рекурсивно строит граф всех зависимостей:

```js
// index.js (entry)
import App from './App';      // → App.js → Button.js, Header.js
import './styles.css';         // → styles.css → variables.css
import logo from './logo.svg'; // → logo.svg (asset)
```

Webpack "видит" весь этот граф и принимает оптимальные решения о бандлинге:
- Какие модули объединить в один бандл
- Какие вынести в отдельные чанки (code splitting)
- Какие оставить как есть (external)

## Pipeline обработки: Entry → Loader → Plugin → Output

```
┌─────────┐    ┌──────────────────────────────────┐    ┌────────┐
│  Entry  │───▶│  Dependency Graph Resolution     │───▶│        │
└─────────┘    │  ┌─────────────────────────────┐ │    │        │
               │  │  Module Processing           │ │    │ Output │
               │  │  TS → Babel → JS             │ │    │        │
               │  │  SCSS → PostCSS → CSS        │ │    │ dist/  │
               │  │  PNG → base64/file           │ │    │        │
               │  └─────────────────────────────┘ │    │        │
               │  ┌─────────────────────────────┐ │    │        │
               │  │  Plugin Hooks                │ │    │        │
               │  │  HtmlWebpackPlugin           │ │    │        │
               │  │  MiniCssExtractPlugin        │ │    │        │
               │  └─────────────────────────────┘ │    │        │
               └──────────────────────────────────┘    └────────┘
```

## Основные концепции

### Mode
```js
module.exports = {
  mode: 'development', // 'production' | 'none'
};
```
- **development**: быстрая сборка, source maps, удобный debug
- **production**: минификация, tree shaking, оптимизация

### Module System Support
Webpack понимает все форматы модулей:
- **ESM**: `import` / `export`
- **CommonJS**: `require()` / `module.exports`
- **AMD**: `define()` / `require()`
- **Assets**: CSS, изображения, шрифты

### Resolve
Webpack умеет разрешать пути:
```js
resolve: {
  extensions: ['.tsx', '.ts', '.js'],
  alias: {
    '@': path.resolve(__dirname, 'src'),
  },
}
```

## Жизненный цикл сборки

1. **Initialization** — загрузка конфига, создание компилятора
2. **Make** — обход графа зависимостей, применение лоадеров
3. **Seal** — оптимизация, разбивка на чанки
4. **Emit** — запись файлов в `output.path`
5. **Done** — финальные хуки, вывод статистики

const BG = '#0f172a';
const WEBPACK_COLOR = '#8dd6f9';
const CARD = '#1e293b';
const TEXT = '#e2e8f0';
const ACCENT = '#fb923c';
const SUCCESS = '#4ade80';
const MUTED = '#64748b';

interface Node {
  id: string;
  label: string;
  type: 'entry' | 'js' | 'css' | 'asset' | 'external';
  x: number;
  y: number;
  deps: string[];
  size: string;
  description: string;
}

const NODES: Node[] = [
  { id: 'entry', label: 'index.ts', type: 'entry', x: 400, y: 40, deps: ['app', 'styles', 'logo'], size: '1.2 KB', description: 'Точка входа приложения' },
  { id: 'app', label: 'App.tsx', type: 'js', x: 200, y: 140, deps: ['button', 'header', 'api'], size: '4.8 KB', description: 'Корневой React компонент' },
  { id: 'styles', label: 'global.css', type: 'css', x: 400, y: 140, deps: ['vars'], size: '3.1 KB', description: 'Глобальные стили' },
  { id: 'logo', label: 'logo.svg', type: 'asset', x: 600, y: 140, deps: [], size: '2.1 KB', description: 'SVG логотип (asset)' },
  { id: 'button', label: 'Button.tsx', type: 'js', x: 80, y: 260, deps: ['utils'], size: '1.1 KB', description: 'Компонент кнопки' },
  { id: 'header', label: 'Header.tsx', type: 'js', x: 240, y: 260, deps: ['utils'], size: '2.3 KB', description: 'Компонент заголовка' },
  { id: 'api', label: 'api.ts', type: 'js', x: 380, y: 260, deps: ['axios'], size: '3.5 KB', description: 'HTTP клиент' },
  { id: 'vars', label: 'vars.css', type: 'css', x: 520, y: 260, deps: [], size: '0.5 KB', description: 'CSS переменные' },
  { id: 'utils', label: 'utils.ts', type: 'js', x: 140, y: 380, deps: [], size: '0.8 KB', description: 'Утилиты и хелперы' },
  { id: 'axios', label: 'axios', type: 'external', x: 420, y: 380, deps: [], size: '48.2 KB', description: 'NPM пакет (node_modules)' },
];

const TYPE_COLORS: Record<string, string> = {
  entry: ACCENT,
  js: WEBPACK_COLOR,
  css: '#a78bfa',
  asset: SUCCESS,
  external: '#f87171',
};

const TYPE_LABELS: Record<string, string> = {
  entry: 'Entry',
  js: 'JS/TS',
  css: 'CSS',
  asset: 'Asset',
  external: 'External',
};

export default function App() {
  const [selected, setSelected] = useState<string | null>(null);
  const [highlighted, setHighlighted] = useState<Set<string>>(new Set());
  const [showPipeline, setShowPipeline] = useState(false);
  const [pipelineStep, setPipelineStep] = useState(0);

  const selectedNode = NODES.find(n => n.id === selected);

  useEffect(() => {
    if (selected) {
      const node = NODES.find(n => n.id === selected);
      if (node) {
        const related = new Set([selected, ...node.deps]);
        NODES.forEach(n => {
          if (n.deps.includes(selected)) related.add(n.id);
        });
        setHighlighted(related);
      }
    } else {
      setHighlighted(new Set());
    }
  }, [selected]);

  const pipelineSteps = [
    { label: 'Entry', desc: 'Webpack читает точку входа index.ts', color: ACCENT },
    { label: 'Resolve', desc: 'Разрешаются пути import/require', color: WEBPACK_COLOR },
    { label: 'Load', desc: 'Лоадеры трансформируют файлы (TS→JS, CSS→JS)', color: '#a78bfa' },
    { label: 'Seal', desc: 'Модули группируются в чанки', color: SUCCESS },
    { label: 'Emit', desc: 'Файлы записываются в dist/', color: '#4ade80' },
  ];

  const runPipeline = () => {
    setShowPipeline(true);
    setPipelineStep(0);
    let step = 0;
    const iv = setInterval(() => {
      step++;
      if (step >= pipelineSteps.length) { clearInterval(iv); return; }
      setPipelineStep(step);
    }, 700);
  };

  const getEdges = () => {
    const edges: { from: Node; to: Node; active: boolean }[] = [];
    NODES.forEach(node => {
      node.deps.forEach(depId => {
        const dep = NODES.find(n => n.id === depId);
        if (dep) {
          const active = highlighted.size === 0 || (highlighted.has(node.id) && highlighted.has(depId));
          edges.push({ from: node, to: dep, active });
        }
      });
    });
    return edges;
  };

  return (
    <div style={{ background: BG, minHeight: '100vh', color: TEXT, fontFamily: 'monospace', padding: '24px' }}>
      <div style={{ maxWidth: 900, margin: '0 auto' }}>
        <div style={{ textAlign: 'center', marginBottom: 24 }}>
          <div style={{ fontSize: 26, fontWeight: 700, color: WEBPACK_COLOR, marginBottom: 6 }}>
            🕸️ Dependency Graph Visualizer
          </div>
          <div style={{ color: MUTED, fontSize: 13 }}>Нажмите на узел чтобы исследовать зависимости</div>
        </div>

        {/* Legend */}
        <div style={{ display: 'flex', gap: 12, marginBottom: 20, flexWrap: 'wrap', justifyContent: 'center' }}>
          {Object.entries(TYPE_LABELS).map(([type, label]) => (
            <div key={type} style={{ display: 'flex', alignItems: 'center', gap: 6 }}>
              <div style={{ width: 10, height: 10, borderRadius: '50%', background: TYPE_COLORS[type] }} />
              <span style={{ fontSize: 11, color: MUTED }}>{label}</span>
            </div>
          ))}
        </div>

        {/* SVG Graph */}
        <div style={{ background: CARD, borderRadius: 12, padding: 16, marginBottom: 20, overflow: 'hidden' }}>
          <svg width="100%" viewBox="0 0 720 440" style={{ display: 'block' }}>
            {/* Edges */}
            {getEdges().map((edge, i) => (
              <line
                key={i}
                x1={edge.from.x} y1={edge.from.y}
                x2={edge.to.x} y2={edge.to.y}
                stroke={edge.active ? WEBPACK_COLOR : '#334155'}
                strokeWidth={edge.active ? 2 : 1}
                strokeDasharray={edge.active ? 'none' : '4,4'}
                opacity={edge.active ? 0.8 : 0.3}
              />
            ))}
            {/* Nodes */}
            {NODES.map(node => {
              const color = TYPE_COLORS[node.type];
              const isSelected = selected === node.id;
              const isHighlighted = highlighted.has(node.id);
              const isDimmed = highlighted.size > 0 && !isHighlighted;
              return (
                <g key={node.id} onClick={() => setSelected(isSelected ? null : node.id)} style={{ cursor: 'pointer' }}>
                  <circle
                    cx={node.x} cy={node.y} r={isSelected ? 24 : 20}
                    fill={isSelected ? color : isDimmed ? '#1e293b' : '#0f2744'}
                    stroke={color}
                    strokeWidth={isSelected ? 3 : isDimmed ? 1 : 2}
                    opacity={isDimmed ? 0.4 : 1}
                  />
                  <text x={node.x} y={node.y + 4} textAnchor="middle" fontSize={9} fill={isDimmed ? MUTED : (isSelected ? BG : color)} fontFamily="monospace">
                    {node.label.length > 10 ? node.label.slice(0, 9) + '…' : node.label}
                  </text>
                </g>
              );
            })}
          </svg>
        </div>

        <div style={{ display: 'grid', gridTemplateColumns: '1fr 1fr', gap: 16 }}>
          {/* Node info */}
          <div style={{ background: CARD, borderRadius: 12, padding: 16 }}>
            {selectedNode ? (
              <>
                <div style={{ fontSize: 12, color: MUTED, marginBottom: 8, textTransform: 'uppercase' }}>Модуль</div>
                <div style={{ fontSize: 16, color: TYPE_COLORS[selectedNode.type], fontWeight: 700, marginBottom: 4 }}>
                  {selectedNode.label}
                </div>
                <div style={{ fontSize: 12, color: TEXT, marginBottom: 4 }}>{selectedNode.description}</div>
                <div style={{ fontSize: 11, color: MUTED }}>Тип: <span style={{ color: TYPE_COLORS[selectedNode.type] }}>{TYPE_LABELS[selectedNode.type]}</span></div>
                <div style={{ fontSize: 11, color: MUTED }}>Размер: <span style={{ color: TEXT }}>{selectedNode.size}</span></div>
                {selectedNode.deps.length > 0 && (
                  <div style={{ marginTop: 8 }}>
                    <div style={{ fontSize: 11, color: MUTED, marginBottom: 4 }}>Импортирует:</div>
                    {selectedNode.deps.map(d => {
                      const dep = NODES.find(n => n.id === d);
                      return dep ? <div key={d} style={{ fontSize: 11, color: SUCCESS }}>→ {dep.label}</div> : null;
                    })}
                  </div>
                )}
              </>
            ) : (
              <div style={{ color: MUTED, fontSize: 12, textAlign: 'center', paddingTop: 20 }}>
                Выберите узел для просмотра деталей
              </div>
            )}
          </div>

          {/* Pipeline */}
          <div style={{ background: CARD, borderRadius: 12, padding: 16 }}>
            <div style={{ fontSize: 12, color: MUTED, marginBottom: 12, textTransform: 'uppercase' }}>Build Pipeline</div>
            <button
              onClick={runPipeline}
              style={{ width: '100%', padding: '8px', background: WEBPACK_COLOR, color: BG, border: 'none', borderRadius: 6, fontWeight: 700, cursor: 'pointer', marginBottom: 12, fontFamily: 'monospace', fontSize: 13 }}
            >
              ▶ Run Build
            </button>
            {showPipeline && pipelineSteps.map((step, i) => (
              <div
                key={i}
                style={{
                  padding: '8px 12px',
                  background: i <= pipelineStep ? '#0f2744' : '#0a0f1e',
                  borderLeft: \`3px solid \${i <= pipelineStep ? step.color : '#334155'}\`,
                  marginBottom: 4,
                  borderRadius: '0 6px 6px 0',
                  opacity: i <= pipelineStep ? 1 : 0.3,
                  transition: 'all 0.3s',
                }}
              >
                <span style={{ fontSize: 11, color: step.color, fontWeight: 700 }}>{step.label}</span>
                <span style={{ fontSize: 10, color: MUTED, marginLeft: 8 }}>{step.desc}</span>
              </div>
            ))}
          </div>
        </div>
      </div>
    </div>
  );
}
` }} />
