Теги (tags) в Git — это способ отметить важные точки в истории вашего проекта. Чаще всего они используются для маркировки версий ПО (v1.0.0, v2.1.0 и т.д.).

## Типы тегов

В Git существует два основных типа тегов: **легковесные (lightweight)** и **аннотированные (annotated)**.

### 1. Легковесные теги (Lightweight)
Это просто указатель на конкретный [коммит](/git/init-commit/). Они похожи на [ветки](/git/branches/), которые не меняются.

```bash
git tag v1.0.0-lw
```

### 2. Аннотированные теги (Annotated)
Рекомендуемый способ. Они хранятся как полноценные объекты в базе данных Git. Содержат имя создателя, email, дату и сообщение.

```bash
git tag -a v1.0.0 -m "Release version 1.0.0"
```

## Работа с тегами

### Просмотр тегов
```bash
git tag             # Список всех тегов
git show v1.0.0     # Информация о конкретном теге
```

### Удаление тегов
Локально:
```bash
git tag -d v1.0.0
```

Удаление с сервера:
```bash
git push origin --delete v1.0.0
```

## Отправка тегов на сервер

По умолчанию [`git push`](/git/remote-push-pull/) не отправляет теги. Их нужно пушить явно.

```bash
git push origin v1.0.0          # Один тег
git push origin --tags          # Все локальные теги
```

## Releases (Релизы)

На платформах вроде [GitHub](/git/github-workflow/) или GitLab "Релиз" — это надстройка над Git Tag. К тегу прикрепляются:
- Скомпилированные файлы (бинарники, архивы).
- Release Notes (описание изменений).
- Changelog.

## Интерактивный пример

Управление тегами и семантическое версионирование:

<Playground client:visible
  template="vite-react"
  files={{
    "/App.jsx": `import { useState } from "react";

export default function App() {
  const [ver, setVer] = useState({ major: 1, minor: 0, patch: 0 });
  const [tags, setTags] = useState([]);

  const bump = (type) => {
    const v = { ...ver };
    if (type === "major") { v.major++; v.minor = 0; v.patch = 0; }
    else if (type === "minor") { v.minor++; v.patch = 0; }
    else { v.patch++; }
    setVer(v);
    setTags([{ ver: "v" + v.major + "." + v.minor + "." + v.patch, type }, ...tags]);
  };

  const colors = { major: "#ef4444", minor: "#f59e0b", patch: "#22c55e" };
  const bgColors = { major: "#450a0a", minor: "#451a03", patch: "#052e16" };
  const btn = (type, label) => (
    <button key={type} onClick={() => bump(type)} style={{ padding: "6px 14px", borderRadius: 6, border: "none", cursor: "pointer", fontWeight: 700, fontSize: 12, background: colors[type], color: type === "major" ? "#fff" : "#0f172a" }}>{label}</button>
  );

  const Part = ({ value, label, color }) => (
    <div style={{ textAlign: "center" }}>
      <div style={{ fontSize: 32, fontWeight: 700, color }}>{value}</div>
      <div style={{ fontSize: 10, color: "#64748b" }}>{label}</div>
    </div>
  );

  return (
    <div style={{ fontFamily: "monospace", background: "#0f172a", color: "#e2e8f0", padding: 16, minHeight: "100vh" }}>
      <h3 style={{ color: "#818cf8", marginBottom: 12 }}>Semantic Versioning (SemVer)</h3>
      <div style={{ display: "flex", alignItems: "center", gap: 4, justifyContent: "center", marginBottom: 16 }}>
        <Part value={ver.major} label="MAJOR" color="#ef4444" />
        <span style={{ fontSize: 32, color: "#64748b" }}>.</span>
        <Part value={ver.minor} label="MINOR" color="#f59e0b" />
        <span style={{ fontSize: 32, color: "#64748b" }}>.</span>
        <Part value={ver.patch} label="PATCH" color="#22c55e" />
      </div>
      <div style={{ display: "flex", gap: 6, justifyContent: "center", marginBottom: 14 }}>
        {btn("major", "Major (breaking)")}
        {btn("minor", "Minor (feature)")}
        {btn("patch", "Patch (fix)")}
      </div>
      <div style={{ display: "flex", flexDirection: "column", gap: 4, maxHeight: 120, overflowY: "auto" }}>
        {tags.map((t, i) => (
          <div key={i} style={{ display: "flex", alignItems: "center", gap: 8, padding: "5px 10px", background: "#1e293b", borderRadius: 6, fontSize: 12 }}>
            <span style={{ color: "#f59e0b", minWidth: 60 }}>{t.ver}</span>
            <span style={{ fontSize: 10, padding: "2px 6px", borderRadius: 3, background: bgColors[t.type], color: colors[t.type] }}>{t.type}</span>
          </div>
        ))}
      </div>
    </div>
  );
}
`
  }}
/>
