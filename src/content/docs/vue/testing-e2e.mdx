## 🎭 Урок 25: E2E-тестирование с Playwright

End-to-End (E2E) тестирование проверяет твоё приложение **глазами реального пользователя** — открывает браузер, кликает кнопки, заполняет формы и проверяет, что всё работает от начала до конца. Playwright — современный инструмент от Microsoft, который делает это быстро, надёжно и с кучей полезных фишек. 🚀

---

### 🤔 Зачем E2E-тесты?

Unit-тесты проверяют отдельные функции, интеграционные — связи между компонентами, а E2E-тесты — **весь пользовательский сценарий** целиком:

```
Unit тест:       [функция] → ✅
Integration:     [компонент A] + [компонент B] → ✅
E2E тест:        [браузер] → открыть → кликнуть → заполнить → отправить → ✅
```

Это последний рубеж защиты. Именно E2E-тесты ловят баги, которые возникают только когда всё работает вместе.

### 🏗️ Page Object Model (POM)

Page Object Model — паттерн, который делает тесты **читаемыми и поддерживаемыми**. Вместо прямых селекторов в тестах — объекты, инкапсулирующие взаимодействие со страницей.

```typescript
// tests/pages/LoginPage.ts
import { Page, Locator, expect } from '@playwright/test';

export class LoginPage {
  readonly page: Page;
  readonly emailInput: Locator;
  readonly passwordInput: Locator;
  readonly submitButton: Locator;
  readonly errorMessage: Locator;

  constructor(page: Page) {
    this.page = page;
    this.emailInput = page.getByLabel('Email');
    this.passwordInput = page.getByLabel('Пароль');
    this.submitButton = page.getByRole('button', { name: 'Войти' });
    this.errorMessage = page.getByRole('alert');
  }

  async goto() {
    await this.page.goto('/login');
  }

  async login(email: string, password: string) {
    await this.emailInput.fill(email);
    await this.passwordInput.fill(password);
    await this.submitButton.click();
  }

  async expectError(message: string) {
    await expect(this.errorMessage).toContainText(message);
  }
}
```

### 🎯 Атрибуты `data-testid`

Лучшая практика — помечать элементы специальными атрибутами для тестов. Они не зависят от CSS-классов или структуры DOM.

```vue
<!-- В Vue-компоненте -->
<template>
  <div class="todo-app">
    <input
      data-testid="todo-input"
      v-model="newTodo"
      placeholder="Новая задача..."
      @keyup.enter="addTodo"
    />
    <button data-testid="add-todo-btn" @click="addTodo">
      Добавить
    </button>

    <ul data-testid="todo-list">
      <li
        v-for="todo in todos"
        :key="todo.id"
        :data-testid="`todo-item-${todo.id}`"
      >
        <input
          type="checkbox"
          :data-testid="`todo-checkbox-${todo.id}`"
          :checked="todo.done"
          @change="toggleTodo(todo.id)"
        />
        <span :data-testid="`todo-text-${todo.id}`">{{ todo.text }}</span>
        <button :data-testid="`delete-todo-${todo.id}`" @click="deleteTodo(todo.id)">
          ✕
        </button>
      </li>
    </ul>

    <footer data-testid="todo-footer">
      <span data-testid="todo-count">{{ remaining }} задач осталось</span>
    </footer>
  </div>
</template>
```


### 💡 Лучшие практики

**1. Используй роли и метки вместо CSS-селекторов:**
```typescript
// ❌ Плохо — ломается при изменении CSS
page.locator('.btn-primary.submit-form')

// ✅ Хорошо — устойчиво к рефакторингу
page.getByRole('button', { name: 'Отправить' })
page.getByLabel('Email')
page.getByTestId('submit-btn')
```

**2. Избегай жёстких таймаутов:**
```typescript
// ❌ Плохо — ненадёжно
await page.waitForTimeout(2000);

// ✅ Хорошо — ждём конкретного состояния
await page.waitForSelector('[data-testid="loader"]', { state: 'hidden' });
await expect(page.getByText('Загружено')).toBeVisible();
```

**3. Изолируй тесты:**
```typescript
// Каждый тест создаёт свои данные и не зависит от других
test.beforeEach(async ({ page }) => {
  // Очистить состояние через API
  await page.request.post('/api/test/reset');
});
```

**4. Тестируй доступность:**
```typescript
test('доступность форм', async ({ page }) => {
  await page.goto('/contact');
  // Проверить наличие меток у полей
  await expect(page.getByLabel('Имя')).toBeVisible();
  // Проверить keyboard navigation
  await page.keyboard.press('Tab');
  await expect(page.getByLabel('Email')).toBeFocused();
});
```

          addLog(\`  ✓ \${step.action}(\${step.selector}) [\${dur}ms]\`);
        }
        await sleep(100);
      }

      test.status = testFailed ? "fail" : "pass";
      setTests([...newTests]);
      await sleep(300);
    }

    const passed = newTests.filter(t => t.status === "pass").length;
    const failed = newTests.filter(t => t.status === "fail").length;
    addLog(\`\\n📊 Результат: \${passed} прошло, \${failed} упало\`);
    setRunning(false);
    setActiveTest(null);
  };

  const statusIcon = (s: string) =>
    s === "pass" ? "✓" : s === "fail" ? "✗" : s === "running" ? "⟳" : "○";
  const statusColor = (s: string) =>
    s === "pass" ? GREEN : s === "fail" ? "#ef4444" : s === "running" ? "#f59e0b" : "#475569";

  const passed = tests.filter(t => t.status === "pass").length;
  const failed = tests.filter(t => t.status === "fail").length;
  const total = tests.length;

  return (
    <div style={{ background: BG, minHeight: "100vh", padding: "1rem", fontFamily: "monospace", fontSize: "0.78rem" }}>
      {/* Header */}
      <div style={{ display: "flex", alignItems: "center", gap: "0.75rem", marginBottom: "1rem" }}>
        <div style={{ background: DARK, borderRadius: "8px", padding: "0.4rem 0.75rem", color: GREEN, fontWeight: 700, fontSize: "0.85rem" }}>
          🎭 Playwright
        </div>

            <button key={b} onClick={() => !running && setBrowser(b)} style={{

              background: browser === b ? GREEN : BORDER, color: browser === b ? DARK : "#94a3b8", fontSize: "0.7rem", fontWeight: 600,
            }}>

            </button>
          ))}
        </div>
      </div>

      {/* Stats */}
      <div style={{ display: "grid", gridTemplateColumns: "repeat(3,1fr)", gap: "0.5rem", marginBottom: "1rem" }}>
        {[
          { label: "Всего", value: total, color: "#60a5fa" },
          { label: "Прошло", value: passed, color: GREEN },
          { label: "Упало", value: failed, color: "#ef4444" },
        ].map(s => (
          <div key={s.label} style={{ background: CARD, borderRadius: "6px", padding: "0.5rem", border: \`1px solid \${BORDER}\`, textAlign: "center" }}>
            <div style={{ color: s.color, fontSize: "1.4rem", fontWeight: 700 }}>{s.value}</div>
            <div style={{ color: "#475569", fontSize: "0.68rem" }}>{s.label}</div>
          </div>
        ))}
      </div>

      <div style={{ display: "grid", gridTemplateColumns: "1fr 1fr", gap: "0.75rem" }}>
        {/* Test list */}
        <div>
          <div style={{ color: "#94a3b8", fontSize: "0.7rem", marginBottom: "0.5rem" }}>ТЕСТ-КЕЙСЫ</div>
          <div style={{ display: "flex", flexDirection: "column", gap: "0.5rem" }}>
            {tests.map(t => (
              <div key={t.id} style={{
                background: CARD, borderRadius: "6px", padding: "0.6rem",
                border: \`1px solid \${activeTest === t.id ? GREEN : BORDER}\`,
                transition: "border-color 0.2s",
              }}>
                <div style={{ display: "flex", alignItems: "center", gap: "0.4rem", marginBottom: "0.4rem" }}>
                  <span style={{ color: statusColor(t.status), fontWeight: 700 }}>{statusIcon(t.status)}</span>
                  <span style={{ color: "#e2e8f0", fontSize: "0.72rem" }}>{t.name}</span>
                </div>
                <div style={{ display: "flex", flexDirection: "column", gap: "0.2rem", paddingLeft: "1rem" }}>
                  {t.steps.map(s => (
                    <div key={s.id} style={{ display: "flex", alignItems: "center", gap: "0.35rem" }}>
                      <span style={{ color: statusColor(s.status), fontSize: "0.65rem" }}>{statusIcon(s.status)}</span>
                      <span style={{
                        background: \`\${ACTION_COLORS[s.action] || GREEN}22\`,
                        color: ACTION_COLORS[s.action] || GREEN,
                        borderRadius: "3px", padding: "0 0.3rem", fontSize: "0.62rem",
                      }}>{s.action}</span>
                      <span style={{ color: "#64748b", fontSize: "0.62rem", overflow: "hidden", textOverflow: "ellipsis", whiteSpace: "nowrap", maxWidth: "100px" }}>{s.selector}</span>
                      {s.duration && <span style={{ color: "#475569", fontSize: "0.6rem", marginLeft: "auto" }}>{s.duration}ms</span>}
                    </div>
                  ))}
                </div>
              </div>
            ))}
          </div>
        </div>

        {/* Log */}
        <div>
          <div style={{ color: "#94a3b8", fontSize: "0.7rem", marginBottom: "0.5rem" }}>КОНСОЛЬ</div>
          <div style={{
            background: "#0a0f1a", borderRadius: "6px", padding: "0.6rem",
            border: \`1px solid \${BORDER}\`, height: "280px", overflowY: "auto",
            display: "flex", flexDirection: "column", gap: "0.15rem",
          }}>

            {log.map((l, i) => (

      <div style={{ display: "flex", gap: "0.5rem", marginTop: "1rem" }}>
        <button onClick={runTests} disabled={running} style={{
          padding: "0.5rem 1.25rem", borderRadius: "6px", border: "none",
          background: running ? BORDER : GREEN, color: running ? "#64748b" : DARK,
          cursor: running ? "not-allowed" : "pointer", fontWeight: 700, fontSize: "0.78rem",
