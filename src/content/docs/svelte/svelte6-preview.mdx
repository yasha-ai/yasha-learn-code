## 📊 Svelte + D3: мощная визуализация данных

Svelte и D3.js — идеальная пара для создания интерактивных визуализаций:

```svelte
<script lang="ts">
  import * as d3 from 'd3';
  import { onMount } from 'svelte';

  let { data }: { data: { name: string; value: number }[] } = $props();

  let svgElement: SVGSVGElement;
  let width = $state(400);
  let height = $state(300);

  // Svelte + D3 паттерн: D3 для расчётов, Svelte для DOM
  let xScale = $derived(
    d3.scaleBand()
      .domain(data.map(d => d.name))
      .range([0, width])
      .padding(0.1)
  );

  let yScale = $derived(
    d3.scaleLinear()
      .domain([0, d3.max(data, d => d.value) ?? 0])
      .range([height, 0])
  );

  let bars = $derived(
    data.map(d => ({
      x: xScale(d.name) ?? 0,
      y: yScale(d.value),
      width: xScale.bandwidth(),
      height: height - yScale(d.value),
      name: d.name,
      value: d.value,
    }))
  );

  let hoveredBar = $state<string | null>(null);
</script>

<svg bind:this={svgElement} {width} {height}>
  {#each bars as bar}
    <rect
      x={bar.x}
      y={bar.y}
      width={bar.width}
      height={bar.height}
      fill={hoveredBar === bar.name ? '#ffba00' : '#ff3e00'}
      onmouseenter={() => hoveredBar = bar.name}
      onmouseleave={() => hoveredBar = null}
      style="transition: fill 0.2s"
    />
    <text
      x={bar.x + bar.width / 2}
      y={bar.y - 5}
      text-anchor="middle"
      fill="white"
      font-size="12"
    >
      {bar.value}
    </text>
  {/each}
</svg>
```

**Почему Svelte + D3 лучше чем React + D3:**

```
React + D3:
  - Два разных DOM управления → конфликты
  - Нужен useEffect для D3 операций
  - Re-render уничтожает D3 state

Svelte + D3:
  - Svelte управляет DOM, D3 только считает
  - $derived для scales и calculations
  - Реактивность без useEffect
  - Плавные transitions через Svelte
```

```
1. ПЕРВОЕ МЕСТО по retention в State of JS 3 года подряд

   → Ресурсы, стабильность, долгосрочная перспектива

3. Svelte 5 решил главные проблемы
   → Reusable logic (runes работают вне компонентов)
   → Лучший TypeScript
   → Предсказуемость

4. SvelteKit == полный стек
   → Один фреймворк от frontend до backend
   → Не нужен отдельный Express/Fastify

5. Растущий рынок труда
   → +30% вакансий год к году
   → Нишевое знание = более высокая ценность

   → Официальные туториалы на svelte.dev
```

// Mini D3-like bar chart demo
function D3Demo() {
  const [data, setData] = useState([
    { name: "Svelte", value: 77 },
    { name: "React", value: 220 },
    { name: "Vue", value: 207 },
    { name: "Angular", value: 90 },
    { name: "Solid", value: 30 },
  ]);
  const [hovered, setHovered] = useState<string | null>(null);

  const maxVal = Math.max(...data.map((d) => d.value));
  const width = 300;
  const height = 180;
  const barWidth = width / data.length - 6;

  const frameworkColors: Record<string, string> = {
    Svelte: SVELTE_ORANGE, React: "#61dafb", Vue: "#4fc08d", Angular: "#dd0031", Solid: "#2c4f7c",
  };

  const randomize = () => {
    setData((prev) =>
      prev.map((d) => ({ ...d, value: Math.floor(Math.random() * 250) + 10 }))
    );
  };

  return (
    <div>
      <div style={{ fontSize: "11px", color: "#64748b", fontFamily: "monospace", marginBottom: "10px" }}>
        // D3 считает → Svelte рендерит (без VDOM конфликтов)
      </div>

      <div style={{ background: BG3, borderRadius: "8px", padding: "12px", marginBottom: "10px" }}>
        <svg viewBox={"0 0 " + width + " " + height} style={{ width: "100%", height: "auto" }}>
          {data.map((d, i) => {
            const barH = (d.value / maxVal) * (height - 30);
            const x = i * (barWidth + 6) + 3;
            const y = height - barH - 20;
            const color = frameworkColors[d.name] ?? "#64748b";
            const isHovered = hovered === d.name;

            return (
              <g key={d.name}
                onMouseEnter={() => setHovered(d.name)}
                onMouseLeave={() => setHovered(null)}
                style={{ cursor: "pointer" }}
              >
                <rect
                  x={x} y={y}
                  width={barWidth} height={barH}
                  fill={isHovered ? color : color + "88"}
                  rx={3}
                  style={{ transition: "all 0.3s ease" }}
                />
                <text
                  x={x + barWidth / 2} y={y - 3}
                  textAnchor="middle" fontSize={9}
                  fill={color}
                >
                  {isHovered ? d.value + "k" : ""}
                </text>
                <text
                  x={x + barWidth / 2} y={height - 5}
                  textAnchor="middle" fontSize={8}
                  fill={isHovered ? color : "#64748b"}
                >
                  {d.name.slice(0, 4)}
                </text>
              </g>
            );
          })}
        </svg>
      </div>

      {hovered && (
        <div style={{ background: BG, borderRadius: "6px", padding: "8px", fontSize: "12px", color: frameworkColors[hovered] ?? "#e2e8f0", marginBottom: "8px" }}>
          {hovered}: {data.find((d) => d.name === hovered)?.value}k GitHub Stars
        </div>
      )}

      <button onClick={randomize} style={{ width: "100%", background: SVELTE_ORANGE, color: "#fff", border: "none", borderRadius: "8px", padding: "8px", cursor: "pointer", fontSize: "12px" }}>
        🎲 Рандомизировать (Svelte реактивность!)
      </button>
    </div>
  );
}
