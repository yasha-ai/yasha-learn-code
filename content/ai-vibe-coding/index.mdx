# AI Vibe Coding: Строим продукты с ИИ 🤖

Vibe coding — это новый способ создавать программное обеспечение. Вместо того чтобы писать каждую строку кода вручную, ты описываешь что хочешь получить на естественном языке, а ИИ генерирует код за тебя.

**Это не будущее — это настоящее 2026 года.**

## Что такое Vibe Coding?

Термин ввёл Андрей Карпатий в 2025 году: ты «вайбуешь» с ИИ-инструментом, описывая своё видение, а не пишешь код посимвольно. ИИ берёт на себя 80% работы — ты управляешь процессом.

Ключевые принципы:
- **Промпт > Синтаксис** — ты думаешь о продукте, не о переменных
- **Итерации быстрее** — от идеи до рабочего прототипа за минуты
- **Владение кодом** — код реальный, ты его понимаешь и можешь редактировать
- **Без потолка** — можно строить полноценные продукты, не только игрушки

## Чему учит этот курс

Здесь нет «нарисуй кнопку за 5 минут». Здесь — реальные инструменты, которыми пользуются профессиональные разработчики и фаундеры прямо сейчас:

### Урок 1 — Replit и Lovable
Два топовых инструмента для быстрого старта. Replit — cloud IDE с мощным AI агентом. Lovable — генератор полноценных веб-приложений на React + Supabase.

### Урок 2 — Cursor и Antigravity
Cursor — AI-редактор кода, который знает весь твой проект. Antigravity (Google AI Studio + Firebase Genkit) — гугловый стек для AI-разработки.

### Урок 3 — Codex и Claude Code (CLI, MCP, плагины)
Кодирование прямо из терминала. OpenAI Codex CLI и Anthropic Claude Code — агентные инструменты для серьёзных задач.

### Урок 4 — Генерация изображений
ChatGPT Image Generation (GPT-4o), Gemini Imagen, Nano Banana Pro — создаём визуалы для проектов без дизайнера.

### Урок 5 — Генерация звука
Suno для музыки, ElevenLabs для голоса и озвучки. Полный стек для медиапродуктов.

### Урок 6 — OpenClaw AI агент
OpenClaw — платформа для персональных AI агентов с интеграцией в Telegram, cron-задачами и skills.

### Урок 7 — Публикация с ИИ
Деплой на Vercel и Railway, AI-контент для соцсетей, SEO-автоматизация.

### Урок 8 (Бонус) — Desktop и Mobile
React Native + AI, Flutter + AI, Tauri + AI. Выходим за пределы браузера.

## Для кого этот курс

- **Разработчики** — хотят ускорить работу в 5-10 раз
- **Фаундеры** — нужно быстро проверить гипотезу без найма команды
- **Дизайнеры** — хотят сами строить то, что придумывают
- **Все остальные** — хотят создавать продукты, не зная как программировать

## Что нужно для старта

- Базовое понимание что такое веб (HTML/CSS — опционально, но помогает)
- Аккаунт на GitHub
- Кредитка для платных планов (у большинства есть бесплатный tier)
- Готовность экспериментировать
