# Оптимизация изображений

Изображения — это часто самый тяжёлый ресурс на странице. Astro решает эту проблему через встроенный компонент `<Image />`, который автоматически конвертирует, сжимает и добавляет атрибуты `width`/`height` — никакого layout shift!

Думай об `<Image />` как о личном фотографе-ретушёре: ты даёшь ему исходник, говоришь «сделай поменьше и в WebP», и получаешь идеально оптимизированный результат.

---

## Компонент &lt;Image /&gt;

```astro
---
import { Image } from 'astro:assets';
import myPhoto from '../assets/photo.jpg';
---

<!-- Локальное изображение — Astro знает размеры автоматически -->
<Image src={myPhoto} alt="Мой фото" />

<!-- С явными параметрами -->
<Image
  src={myPhoto}
  alt="Мой фото"
  width={800}
  height={600}
  format="webp"
  quality={80}
/>
```

---

## Ключевые пропсы

| Проп | Тип | Описание |
|------|-----|----------|
| `src` | `ImageMetadata \| string` | Источник изображения |
| `alt` | `string` | Обязательный alt-текст |
| `width` | `number` | Ширина в пикселях |
| `height` | `number` | Высота в пикселях |
| `format` | `'webp' \| 'avif' \| 'jpeg' \| 'png'` | Формат вывода |
| `quality` | `number \| 'low' \| 'mid' \| 'high'` | Качество (1–100) |
| `loading` | `'lazy' \| 'eager'` | Стратегия загрузки |
| `densities` | `number[]` | Пиксельные плотности для srcset |
| `widths` | `number[]` | Несколько ширин для srcset |

## Компонент &lt;Picture /&gt; для art direction

```astro
---
import { Picture } from 'astro:assets';
import hero from '../assets/hero.jpg';
---

<Picture
  src={hero}
  alt="Hero"
  formats={['avif', 'webp']}
  widths={[400, 800, 1200]}
  sizes="(max-width: 600px) 400px, (max-width: 900px) 800px, 1200px"
/>
```

`<Picture />` генерирует `<source>` элементы с `srcset` — браузер выбирает лучший вариант автоматически.

## Удалённые изображения

```ts
// astro.config.mjs
export default defineConfig({
  image: {
    domains: ['images.unsplash.com', 'cdn.mysite.com'],
    remotePatterns: [
      { protocol: 'https', hostname: '**.cloudinary.com' }
    ],
  },
});
```

```astro
<Image
  src="https://images.unsplash.com/photo-123"
  alt="Remote image"
  width={800}
  height={600}
/>
```

---

## Фоновые изображения через CSS

Для фоновых изображений используй `getImage()`:

```astro
---
import { getImage } from 'astro:assets';
import bg from '../assets/bg.jpg';

const optimizedBg = await getImage({ src: bg, format: 'webp', quality: 60 });
---

<div style={\`background-image: url(\${optimizedBg.src})\`}>
  Контент с оптимизированным фоном
</div>
```

---

## Изображения в Markdown и MDX

В MDX просто используй компонент напрямую. В Markdown Astro автоматически оптимизирует изображения если указана настройка:

```ts
// astro.config.mjs
export default defineConfig({
  markdown: {
    shikiConfig: { theme: 'dracula' },
  },
  experimental: {
    optimizeHoistedScript: true,
  },
});
```

## Интерактивная практика

<Playground template="react" files={{ 'App.tsx': `import { useState } from "react";

const BG = "#0f172a";
const ASTRO = "#FF5D01";
const CARD = "#1e293b";
const TEXT = "#e2e8f0";
const MUTED = "#64748b";
const GREEN = "#22c55e";
const BLUE = "#38bdf8";

// Симулируем размеры файлов для разных форматов и качества
function estimateSize(format, quality) {
  const base = { jpeg: 850, webp: 520, avif: 320 };
  const q = quality / 100;
  // Нелинейная зависимость качество → размер
  const factor = 0.1 + q * 0.9 * (1 + (q - 0.5) * 0.4);
  return Math.round(base[format] * factor);
}

function estimateLoadTime(sizeKb) {
  // Имитация 3G: ~1.5 Mbps = 187 KB/s
  return (sizeKb / 187 * 1000).toFixed(0);
}

const formatColors = {
  jpeg: "#f59e0b",
  webp: "#38bdf8",
  avif: "#a78bfa",
};

const formatDescriptions = {
  jpeg: "Старый добрый JPEG. Широкая поддержка, но большой размер.",
  webp: "Google WebP. На 25-35% меньше JPEG при том же качестве.",
  avif: "AVIF — будущее. На 50% меньше JPEG, но медленнее кодируется.",
};

export default function App() {
  const [format, setFormat] = useState("webp");
  const [quality, setQuality] = useState(80);
  const [loading, setLoading] = useState(false);
  const [loaded, setLoaded] = useState(false);

  const originalSize = 2400;
  const currentSize = estimateSize(format, quality);
  const loadTime = estimateLoadTime(currentSize);
  const saving = Math.round((1 - currentSize / originalSize) * 100);

  const simulate = () => {
    setLoaded(false);
    setLoading(true);
    setTimeout(() => {
      setLoading(false);
      setLoaded(true);
    }, parseInt(loadTime) + 200);
  };

  const Bar = ({ label, size, color, isOriginal }) => {
    const maxSize = 2400;
    const width = Math.round((size / maxSize) * 100);
    return (
      <div style={{ marginBottom: 12 }}>
        <div style={{ display: "flex", justifyContent: "space-between", fontSize: 12, marginBottom: 4 }}>
          <span style={{ color: isOriginal ? MUTED : TEXT }}>{label}</span>
          <span style={{ color: isOriginal ? MUTED : color, fontWeight: 600 }}>
            {size} KB {!isOriginal && \`(−\${Math.round((1 - size / originalSize) * 100)}%)\`}
          </span>
        </div>
        <div style={{ height: 10, background: "#334155", borderRadius: 5, overflow: "hidden" }}>
          <div style={{
            width: \`\${width}%\`, height: "100%", borderRadius: 5,
            background: isOriginal ? "#475569" : color,
            transition: "width 0.4s ease",
          }} />
        </div>
      </div>
    );
  };

  return (
    <div style={{ background: BG, minHeight: "100vh", padding: "24px", fontFamily: "system-ui, sans-serif", color: TEXT }}>
      <div style={{ maxWidth: 740, margin: "0 auto" }}>
        <h2 style={{ color: ASTRO, marginBottom: 4, fontSize: 22 }}>🖼️ Image Optimization Simulator</h2>
        <p style={{ color: MUTED, marginBottom: 24, fontSize: 14 }}>
          Сравнение форматов и настроек оптимизации изображений в Astro
        </p>

        <div style={{ display: "grid", gridTemplateColumns: "1fr 1fr", gap: 20 }}>
          {/* Controls */}
          <div style={{ background: CARD, borderRadius: 12, padding: 20 }}>
            <div style={{ fontWeight: 600, marginBottom: 16, color: ASTRO }}>⚙️ Параметры &lt;Image /&gt;</div>

            <div style={{ marginBottom: 20 }}>
              <div style={{ fontSize: 13, color: MUTED, marginBottom: 8 }}>format=</div>
              <div style={{ display: "flex", gap: 8 }}>
                {["jpeg", "webp", "avif"].map(f => (
                  <button key={f} onClick={() => setFormat(f)} style={{
                    flex: 1, padding: "8px 4px", borderRadius: 8, border: "none", cursor: "pointer",
                    background: format === f ? formatColors[f] + "33" : "#334155",
                    color: format === f ? formatColors[f] : MUTED,
                    fontWeight: format === f ? 700 : 400,
                    fontSize: 13, border: format === f ? \`1px solid \${formatColors[f]}\` : "1px solid transparent",
                    transition: "all 0.2s",
                  }}>
                    {f.toUpperCase()}
                  </button>
                ))}
              </div>
              <div style={{ marginTop: 8, fontSize: 12, color: MUTED, padding: "6px 10px", background: "#ffffff08", borderRadius: 6 }}>
                {formatDescriptions[format]}
              </div>
            </div>

            <div style={{ marginBottom: 20 }}>
              <div style={{ display: "flex", justifyContent: "space-between", fontSize: 13, marginBottom: 8 }}>
                <span style={{ color: MUTED }}>quality=</span>
                <span style={{ color: formatColors[format], fontWeight: 700 }}>{quality}</span>
              </div>
              <input
                type="range" min={10} max={100} value={quality}
                onChange={e => setQuality(Number(e.target.value))}
                style={{ width: "100%", accentColor: formatColors[format] }}
              />
              <div style={{ display: "flex", justifyContent: "space-between", fontSize: 11, color: MUTED, marginTop: 4 }}>
                <span>10 (минимум)</span>
                <span>100 (оригинал)</span>
              </div>
            </div>

            {/* Generated code */}
            <div style={{ background: "#0f172a", borderRadius: 8, padding: 12, fontFamily: "monospace", fontSize: 12 }}>
              <div style={{ color: MUTED }}>&lt;<span style={{ color: ASTRO }}>Image</span></div>
              <div style={{ paddingLeft: 12 }}>
                <div><span style={{ color: BLUE }}>src</span>=<span style={{ color: GREEN }}>\`{{photo}}\`</span></div>
                <div><span style={{ color: BLUE }}>alt</span>=<span style={{ color: GREEN }}>"Фото"</span></div>
                <div><span style={{ color: BLUE }}>format</span>=<span style={{ color: GREEN }}>"{format}"</span></div>
                <div><span style={{ color: BLUE }}>quality</span>=<span style={{ color: GREEN }}>{quality}</span></div>
              </div>
              <div style={{ color: MUTED }}>/&gt;</div>
            </div>
          </div>

          {/* Results */}
          <div style={{ background: CARD, borderRadius: 12, padding: 20 }}>
            <div style={{ fontWeight: 600, marginBottom: 16, color: ASTRO }}>📊 Результат</div>

            {/* Image preview placeholder */}
            <div style={{
              height: 120, borderRadius: 8, marginBottom: 16, position: "relative", overflow: "hidden",
              background: "linear-gradient(135deg, #1e3a5f, #1e293b)",
              display: "flex", alignItems: "center", justifyContent: "center",
            }}>
              {loading ? (
                <div style={{ textAlign: "center" }}>
                  <div style={{ fontSize: 24, animation: "pulse 1s infinite" }}>⏳</div>
                  <div style={{ fontSize: 12, color: MUTED, marginTop: 4 }}>Загружаем... {loadTime}ms</div>
                </div>
              ) : loaded ? (
                <div style={{ textAlign: "center" }}>
                  <div style={{ fontSize: 36 }}>🏔️</div>
                  <div style={{ fontSize: 12, color: GREEN, marginTop: 4 }}>✓ Загружено за {loadTime}ms</div>
                </div>
              ) : (
                <div style={{ textAlign: "center" }}>
                  <div style={{ fontSize: 36 }}>🏔️</div>
                  <div style={{ fontSize: 12, color: MUTED, marginTop: 4 }}>photo.jpg (оригинал)</div>
                </div>
              )}
              {/* Quality indicator */}
              <div style={{
                position: "absolute", top: 8, right: 8,
                padding: "2px 8px", borderRadius: 4, fontSize: 11,
                background: formatColors[format] + "33",
                color: formatColors[format], border: \`1px solid \${formatColors[format]}66\`,
              }}>
                {format.toUpperCase()} q{quality}
              </div>
            </div>

            <button onClick={simulate} style={{
              width: "100%", padding: "9px", background: ASTRO, border: "none",
              borderRadius: 8, color: "#fff", cursor: "pointer", fontWeight: 600,
              marginBottom: 16, fontSize: 14,
            }}>
              ▶ Симулировать загрузку
            </button>

            {/* File size comparison */}
            <Bar label="Оригинал (PNG)" size={originalSize} color="#475569" isOriginal />
            <Bar label={\`\${format.toUpperCase()} q\${quality}\`} size={currentSize} color={formatColors[format]} />

            <div style={{
              marginTop: 12, padding: "10px 14px", borderRadius: 8,
              background: saving > 0 ? "#22c55e22" : "#f59e0b22",
              border: \`1px solid \${saving > 0 ? "#22c55e" : "#f59e0b"}44\`,

              <div style={{ fontSize: 13, color: saving > 0 ? GREEN : "#f59e0b", fontWeight: 700 }}>
                {saving > 0 ? \`🎉 Экономия \${saving}% (\${originalSize - currentSize} KB)\` : "⚠️ Нет экономии"}
              </div>
              <div style={{ fontSize: 12, color: MUTED, marginTop: 2 }}>
                Время загрузки (3G): ~{loadTime}ms
              </div>
            </div>
          </div>
        </div>

        {/* Loading timeline */}
        <div style={{ background: CARD, borderRadius: 12, padding: 20, marginTop: 20 }}>
          <div style={{ fontWeight: 600, marginBottom: 16, color: ASTRO }}>📡 Сравнение на разных форматах (текущее качество q{quality})</div>
          {["jpeg", "webp", "avif"].map(f => {
            const sz = estimateSize(f, quality);
            const ms = estimateLoadTime(sz);
            const maxMs = parseInt(estimateLoadTime(estimateSize("jpeg", quality)));
            const barW = Math.round((parseInt(ms) / maxMs) * 100);
            return (

                <div style={{ display: "flex", justifyContent: "space-between", fontSize: 12, marginBottom: 4 }}>
                  <span style={{ color: f === format ? formatColors[f] : MUTED, fontWeight: f === format ? 700 : 400 }}>
                    {f === format ? "▶ " : ""}{f.toUpperCase()} — {sz} KB

                <div style={{ height: 8, background: "#334155", borderRadius: 4, overflow: "hidden" }}>

                    background: formatColors[f],
                    opacity: f === format ? 1 : 0.4,
                    transition: "width 0.4s ease",
                  }} />

              </div>
            );
          })}
        </div>
      </div>
    </div>
  );
}
