## ✨ Особенности

- ✅ **Интерактивные плейграунды** — живой код прямо в браузере (Sandpack) в каждом уроке
- ✅ **Мультифреймворковость** — сравнения React vs Vue vs Svelte vs Solid и т.д.
- ✅ **Реальные проекты** — финальный урок каждого курса — полноценное приложение
- ✅ **От простого к сложному** — каждый курс от Hello World до продакшен-паттернов
- ✅ **Русский язык** — всё объяснено понятно и подробно

## 🗺️ Рекомендуемые пути

**Для начинающих:**
HTML → CSS → JavaScript → TypeScript → React → Git

**Full-Stack разработчик:**
React → Next.js или Remix → tRPC → Prisma → Docker

**Frontend-специалист:**
React → Vue → Svelte → Solid.js → Astro → TanStack

**DevOps/Backend:**
Docker → tRPC → Prisma → Git
