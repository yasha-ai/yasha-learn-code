# События

jQuery предоставляет единый API для работы с событиями во всех браузерах.

## Привязка событий

.on(), .off(), .one()

```javascript
// Пример кода для Привязка событий
$('.example').text('Hello from Привязка событий');
```

## Короткие методы

.click(), .hover(), .focus(), .blur()

```javascript
// Пример кода для Короткие методы
$('.example').text('Hello from Короткие методы');
```

## Клавиатурные события

.keydown(), .keyup(), .keypress()

```javascript
// Пример кода для Клавиатурные события
$('.example').text('Hello from Клавиатурные события');
```

## События формы

.submit(), .change(), .select()

```javascript
// Пример кода для События формы
$('.example').text('Hello from События формы');
```

## Объект события

event.preventDefault(), event.stopPropagation()

```javascript
// Пример кода для Объект события
$('.example').text('Hello from Объект события');
```
