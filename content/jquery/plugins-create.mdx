# Создание собственных плагинов

Расширение jQuery своими методами.

## Базовая структура

$.fn.myPlugin

```javascript
// Пример кода для Базовая структура
$('.example').text('Hello from Базовая структура');
```

## Параметры плагина

Options объект

```javascript
// Пример кода для Параметры плагина
$('.example').text('Hello from Параметры плагина');
```

## Chainability

Возврат this

```javascript
// Пример кода для Chainability
$('.example').text('Hello from Chainability');
```

## Приватные методы

Closure паттерн

```javascript
// Пример кода для Приватные методы
$('.example').text('Hello from Приватные методы');
```
