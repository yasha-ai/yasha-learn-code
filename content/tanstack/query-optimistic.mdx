# Оптимистичные Обновления

## Что такое оптимистичные обновления?

Оптимистичное обновление — техника UX, при которой интерфейс **мгновенно** отражает изменение,
не дожидаясь ответа сервера. Если сервер вернёт ошибку — UI откатывается к предыдущему состоянию.

Это создаёт ощущение мгновенного отклика, особенно заметное при лайках, завершении задач, изменении имени.

## Паттерн реализации

Оптимистичные обновления используют три коллбэка `useMutation`:

1. **`onMutate`** — выполняется до запроса. Обновляем кэш, сохраняем предыдущее состояние
2. **`onError`** — выполняется при ошибке. Откатываем кэш к сохранённому состоянию
3. **`onSettled`** — выполняется всегда. Инвалидируем кэш для синхронизации с сервером

```typescript
const queryClient = useQueryClient()

const toggleLike = useMutation({
  mutationFn: (postId: number) => likePost(postId),

  onMutate: async (postId) => {
    // 1. Отменяем текущие запросы (избегаем перезаписи нашего оптимистичного обновления)
    await queryClient.cancelQueries({ queryKey: ['posts'] })

    // 2. Сохраняем предыдущее состояние для отката
    const previousPosts = queryClient.getQueryData<Post[]>(['posts'])

    // 3. Оптимистично обновляем кэш
    queryClient.setQueryData<Post[]>(['posts'], (old) =>
      old?.map(post =>
        post.id === postId
          ? { ...post, liked: !post.liked, likes: post.liked ? post.likes - 1 : post.likes + 1 }
          : post
      )
    )

    // 4. Возвращаем контекст для onError
    return { previousPosts }
  },

  onError: (err, postId, context) => {
    // Откат при ошибке
    if (context?.previousPosts) {
      queryClient.setQueryData(['posts'], context.previousPosts)
    }
  },

  onSettled: () => {
    // Синхронизация с сервером после завершения
    queryClient.invalidateQueries({ queryKey: ['posts'] })
  },
})
```

## Ключевые детали

### Почему cancelQueries?

Без отмены запросов возможна гонка: идёт фоновый запрос `['posts']`, мы оптимистично обновляем кэш,
но когда фоновый запрос завершается — он перезапишет наши изменения.

### Context в onError

`onMutate` может вернуть любое значение — оно передаётся как `context` в `onError` и `onSettled`.
Используйте это для сохранения предыдущего состояния.

### Типизация

```typescript
useMutation<
  ResponseType,     // тип ответа сервера
  ErrorType,        // тип ошибки
  VariablesType,    // тип аргументов mutate()
  ContextType       // тип возвращаемого onMutate
>({ ... })
```

<Playground template="react" files={{ 'App.tsx': `import { useState } from 'react'

type Todo = { id: number; text: string; done: boolean }

let nextId = 5

const INITIAL_TODOS: Todo[] = [
  { id: 1, text: 'Изучить оптимистичные обновления', done: false },
  { id: 2, text: 'Написать тесты', done: false },
  { id: 3, text: 'Прочитать документацию', done: true },
  { id: 4, text: 'Провести демо', done: false },
]

// Симуляция сервера с задержкой и возможностью ошибки
async function serverToggle(id: number, shouldFail: boolean): Promise<void> {
  await new Promise(res => setTimeout(res, 700))
  if (shouldFail) throw new Error('Сервер недоступен')
}

async function serverAdd(text: string): Promise<Todo> {
  await new Promise(res => setTimeout(res, 600))
  return { id: nextId++, text, done: false }
}

async function serverDelete(id: number): Promise<void> {
  await new Promise(res => setTimeout(res, 500))
}

type LogEntry = { msg: string; type: 'mutate' | 'success' | 'error' | 'settle' | 'rollback' }

export default function App() {
  const [todos, setTodos] = useState<Todo[]>(INITIAL_TODOS)
  const [simulateError, setSimulateError] = useState(false)
  const [pendingIds, setPendingIds] = useState<Set<number>>(new Set())
  const [newText, setNewText] = useState('')
  const [log, setLog] = useState<LogEntry[]>([])

  const addLog = (msg: string, type: LogEntry['type']) =>
    setLog(l => [{ msg: \`\${new Date().toLocaleTimeString()} \${msg}\`, type }, ...l].slice(0, 12))

  const toggleTodo = async (id: number) => {
    const previousTodos = [...todos]

    // onMutate: оптимистичное обновление
    setTodos(prev => prev.map(t => t.id === id ? { ...t, done: !t.done } : t))
    setPendingIds(s => new Set(s).add(id))
    addLog(\`onMutate: todo #\${id} оптимистично обновлён\`, 'mutate')

    try {
      await serverToggle(id, simulateError)
      addLog(\`onSuccess: сервер подтвердил изменение #\${id}\`, 'success')
    } catch (err) {
      // onError: откат
      addLog(\`onError: ошибка! Откат к предыдущему состоянию\`, 'error')
      setTodos(previousTodos)
      addLog(\`Откат выполнен — состояние восстановлено\`, 'rollback')
    } finally {
      // onSettled
      setPendingIds(s => { const next = new Set(s); next.delete(id); return next })
      addLog(\`onSettled: синхронизация с сервером\`, 'settle')
    }
  }

  const addTodo = async () => {
    if (!newText.trim()) return
    const tempId = -Date.now()
    const optimisticTodo: Todo = { id: tempId, text: newText, done: false }

    // Оптимистично добавляем
    setTodos(prev => [...prev, optimisticTodo])
    setPendingIds(s => new Set(s).add(tempId))
    setNewText('')
    addLog(\`onMutate: добавлена временная задача (id: temp)\`, 'mutate')

    try {
      const realTodo = await serverAdd(newText)
      setTodos(prev => prev.map(t => t.id === tempId ? realTodo : t))
      addLog(\`onSuccess: получен реальный id: \${realTodo.id}\`, 'success')
    } catch (err) {
      setTodos(prev => prev.filter(t => t.id !== tempId))
      addLog(\`onError: задача не создана, удаляем из UI\`, 'error')
    } finally {
      setPendingIds(s => { const next = new Set(s); next.delete(tempId); return next })
      addLog(\`onSettled: завершено\`, 'settle')
    }
  }

  const deleteTodo = async (id: number) => {
    const previousTodos = [...todos]
    setTodos(prev => prev.filter(t => t.id !== id))
    addLog(\`onMutate: задача #\${id} оптимистично удалена\`, 'mutate')
    try {
      await serverDelete(id)
      addLog(\`onSuccess: удалено на сервере\`, 'success')
    } catch {
      setTodos(previousTodos)
      addLog(\`onError: откат — задача восстановлена\`, 'rollback')
    }
  }

  const logColors: Record<LogEntry['type'], string> = {
    mutate: '#f97316',
    success: '#22c55e',
    error: '#ef4444',
    settle: '#64748b',
    rollback: '#a855f7',
  }

  return (
    <div style={{ background: '#0f172a', minHeight: '100vh', padding: '24px', fontFamily: 'system-ui, sans-serif', color: '#e2e8f0' }}>
      <h1 style={{ color: '#ef4444', fontSize: '22px', marginBottom: '4px' }}>Оптимистичные обновления</h1>
      <p style={{ color: '#64748b', fontSize: '13px', marginBottom: '20px' }}>onMutate → обновляем UI → onError → откат</p>

      <div style={{ display: 'grid', gridTemplateColumns: '1fr 1fr', gap: '16px' }}>
        <div>
          <div style={{ background: '#1e293b', borderRadius: '12px', padding: '16px', marginBottom: '12px' }}>
            <div style={{ display: 'flex', gap: '8px', marginBottom: '8px' }}>
              <input
                value={newText}
                onChange={e => setNewText(e.target.value)}
                onKeyDown={e => e.key === 'Enter' && addTodo()}
                placeholder="Новая задача..."
                style={{ flex: 1, background: '#0f172a', border: '1px solid #334155', borderRadius: '8px', padding: '8px 12px', color: '#e2e8f0', fontSize: '13px' }}
              />
              <button onClick={addTodo} disabled={!newText.trim()} style={{ background: '#ef4444', border: 'none', borderRadius: '8px', padding: '8px 14px', color: 'white', cursor: 'pointer', fontSize: '13px' }}>
                ➕
              </button>
            </div>
            <label style={{ display: 'flex', alignItems: 'center', gap: '8px', fontSize: '12px', color: '#64748b', cursor: 'pointer' }}>
              <input type="checkbox" checked={simulateError} onChange={e => setSimulateError(e.target.checked)} />
              Ошибка при toggle (протестируй откат!)
            </label>
          </div>

          <div style={{ background: '#1e293b', borderRadius: '12px', padding: '16px' }}>
            <h3 style={{ color: '#e2e8f0', fontSize: '14px', marginBottom: '12px' }}>📋 Задачи</h3>
            {todos.map(todo => (
              <div key={todo.id} style={{
                display: 'flex',
                alignItems: 'center',
                gap: '10px',
                background: '#0f172a',
                borderRadius: '8px',
                padding: '10px 12px',
                marginBottom: '6px',
                opacity: pendingIds.has(todo.id) ? 0.6 : 1,
                transition: 'opacity 0.2s',
                border: pendingIds.has(todo.id) ? '1px dashed #f97316' : '1px solid transparent',
              }}>
                <button
                  onClick={() => toggleTodo(todo.id)}
                  disabled={pendingIds.has(todo.id)}
                  style={{
                    width: '20px', height: '20px', borderRadius: '4px',
                    background: todo.done ? '#22c55e' : 'transparent',
                    border: \`2px solid \${todo.done ? '#22c55e' : '#475569'}\`,
                    cursor: 'pointer', color: 'white', fontSize: '11px',
                    display: 'flex', alignItems: 'center', justifyContent: 'center', flexShrink: 0,
                  }}
                >
                  {pendingIds.has(todo.id) ? '⏳' : todo.done ? '✓' : ''}
                </button>
                <span style={{ flex: 1, fontSize: '13px', textDecoration: todo.done ? 'line-through' : 'none', color: todo.done ? '#475569' : '#e2e8f0' }}>
                  {todo.text}
                  {pendingIds.has(todo.id) && <span style={{ color: '#f97316', fontSize: '11px', marginLeft: '6px' }}>⏳ ожидание</span>}
                </span>
                <button onClick={() => deleteTodo(todo.id)} style={{ background: 'none', border: 'none', color: '#475569', cursor: 'pointer', fontSize: '13px' }}>
                  🗑️
                </button>
              </div>
            ))}
          </div>
        </div>

        <div style={{ background: '#1e293b', borderRadius: '12px', padding: '16px' }}>
          <h3 style={{ color: '#e2e8f0', fontSize: '14px', marginBottom: '12px' }}>📜 Лог мутаций</h3>

            {log.length === 0 ? (
              <p style={{ color: '#475569', textAlign: 'center', padding: '20px 0' }}>Выполни действие...</p>

              log.map((entry, i) => (

                  {entry.msg}
                </div>

            )}
          </div>

          <div style={{ marginTop: '16px', borderTop: '1px solid #334155', paddingTop: '12px' }}>

                <div style={{ width: '10px', height: '10px', borderRadius: '2px', background: c }} />
                <span style={{ fontSize: '11px', color: '#64748b' }}>{k}</span>
              </div>
            ))}
          </div>
        </div>
      </div>
    </div>
  )
}
` }} />
